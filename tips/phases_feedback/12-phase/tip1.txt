## Phase name: Phase 12 — Benchmark Certification And Governance

I focused Phase 12 because the repo’s own moonshot loop says the product is `intent -> bounded agents -> proof lanes -> evidence -> expired exceptions -> reusable primitives`, with durable machine-readable evidence as a central requirement. ([GitHub][1]) The current Phase 12 file says local `bench`, `certify`, and `govern` already emit schema-valid artifacts, but external signing, hosted dashboard, and public badge publishing were explicitly deferred. ([GitHub][2]) The router also marks Phase 12 as `complete-with-deferred-work`, specifically naming local certification/governance with external signing/badges/dashboard deferred, and it places Phase 12 public evidence before Phase 13 real mutation gates. ([GitHub][3]) The Phase 12 log repeats that same residual risk: local evidence only, with external signing, public badge publishing, and org dashboard deferred. ([GitHub][4])

This solution hardens Phase 12 by making `jankurai certify` produce a public certification bundle: schema-valid certification JSON, schema-valid attestation JSON, optional keyed local signature, public Markdown report, SVG badge, CI artifact publication, verification mode, docs, tests, phase status, and receipt logging. It treats the “dashboard” requirement as a generated public evidence bundle suitable for GitHub Actions artifacts / GitHub Pages / release assets rather than adding a hosted service, because the moonshot requirement is public proof behind “Built with Jankurai,” not a mandatory SaaS dependency.

I could not run this in a local checkout from this environment, so the validation commands are included at the end and the script is written to stop before appending the finish receipt if validation fails.

pub fn run(args: CertifyArgs) -> Result<()> {
    if let Some(path) = args.verify_attestation.as_deref() {
        return verify_attestation(
            &args.repo,
            Path::new(path),
            args.signing_key_env.as_deref(),
            args.require_signature,
        );
    }

    let (certification, attestation) = build_certification_with_attestation(
        &args.repo,
        args.build_url.clone(),
        args.signing_key_env.as_deref(),
        args.require_signature,
    )?;

    if let Some(path) = args.out.as_deref() {
        validation::write_json(
            &args.repo,
            ArtifactSchema::Certification,
            path,
            &certification,
        )?;
    } else {
        validation::validate_serializable(
            &args.repo,
            ArtifactSchema::Certification,
            &certification,
        )?;
        println!("{}", serde_json::to_string_pretty(&certification)?);
    }

    if let Some(path) = args.md.as_deref() {
        crate::render::write_markdown(path, &render_markdown(&certification))?;
    }

    if let Some(path) = args.attestation.as_deref() {
        validation::write_json(
            &args.repo,
            ArtifactSchema::CertificationAttestation,
            path,
            &attestation,
        )?;
    } else {
        validation::validate_serializable(
            &args.repo,
            ArtifactSchema::CertificationAttestation,
            &attestation,
        )?;
    }

    if let Some(path) = args.badge.as_deref() {
        crate::render::write_markdown(path, &render_badge_svg(&certification))?;
    }

    if let Some(path) = args.public_md.as_deref() {
        crate::render::write_markdown(
            path,
            &render_public_markdown(
                &certification,
                &attestation,
                args.badge.as_deref(),
                args.attestation.as_deref(),
            ),
        )?;
    }

    Ok(())
}

pub fn build_certification(repo: &Path) -> Result<Certification> {
    let (certification, _) = build_certification_with_attestation(repo, None, None, false)?;
    Ok(certification)
}

pub fn build_certification_with_attestation(
    repo: &Path,
    build_url: Option<String>,
    signing_key_env: Option<&str>,
    require_signature: bool,
) -> Result<(Certification, CertificationAttestation)> {
    let mut certification = build_certification_unsigned(repo, build_url)?;
    let attestation = build_attestation(&certification, signing_key_env, require_signature)?;
    let attestation_sha = format!("sha256:{}", sha256_json(&attestation)?);

    certification.provenance.attestation_sha256 = Some(attestation_sha);
    certification.provenance.subject = Some(attestation.subject.digest.clone());

    if let Some(signer) = &attestation.signer {
        certification.provenance.attestation = "signed-local-sha256-attestation".to_string();
        certification.provenance.signature = Some(signer.signature.clone());
        certification.provenance.signature_algorithm = Some(signer.algorithm.clone());
        certification.provenance.signed_payload_sha256 = Some(signer.payload_sha256.clone());
    } else {
        certification.provenance.attestation = "unsigned-local-attestation".to_string();
    }

    Ok((certification, attestation))
}

fn build_certification_unsigned(repo: &Path, build_url: Option<String>) -> Result<Certification> {
    let release = load_release_data(repo)?;
    let score_path = repo.join("agent/repo-score.json");
    let score = read_repo_score(repo)?;
    let evidence_root = if score_path.exists() {
        repo.to_path_buf()
    } else {
        workspace_root()
    };

    let (score_value, caps, mut findings_summary) = match score.as_ref() {
        Some(summary) => (
            summary.score,
            summary.caps.clone(),
            FindingsSummary {
                critical: summary.findings.critical,
                high: summary.findings.high,
                medium: summary.findings.medium,
                low: summary.findings.low,
                notes: vec![
                    format!("repo score artifact: {}", score_path.display()),
                    "public certification bundle can be generated with badge, report, and attestation".to_string(),
                ],
            },
        ),
        None => (
            0,
            Vec::new(),
            FindingsSummary {
                critical: 0,
                high: 0,
                medium: 0,
                low: 0,
                notes: vec![
                    format!("score artifact missing: {}", score_path.display()),
                    "public certification bundle generated in missing-score mode".to_string(),
                ],
            },
        ),
    };

    if let Some(summary) = score.as_ref() {
        if summary.caps.is_empty() {
            findings_summary
                .notes
                .push("no hard caps reported in the score artifact".to_string());
        }
    }

    let proof_receipt_index = collect_receipts(&evidence_root)?;
    let security_receipt_index = collect_security_receipts(&evidence_root)?;
    let ux_receipt_index = collect_ux_receipts(&evidence_root)?;
    let contract_db_receipt_index = collect_contract_db_receipts(&evidence_root)?;
    let benchmark_receipt_index = collect_benchmark_receipts(&evidence_root)?;
    let governance_receipt_index = collect_governance_receipts(&evidence_root)?;
    let exceptions = collect_exceptions(&evidence_root)?;
    let evidence_subjects = collect_evidence_subjects(
        &evidence_root,
        &proof_receipt_index,
        &security_receipt_index,
        &ux_receipt_index,
        &contract_db_receipt_index,
        &benchmark_receipt_index,
        &governance_receipt_index,
        &exceptions,
    )?;

    let conformance_level = conformance_level(score_value, &caps, &findings_summary);
    let build_url = build_url.or_else(default_build_url);

    Ok(Certification {
        schema_version: release.schema_version,
        generated_at: now_string(),
        repo: repo.display().to_string(),
        standard_version: release.standard_version,
        auditor_version: release.auditor_version,
        paper_edition: release.paper_edition,
        target_stack_id: release.target_stack_id,
        score: score_value,
        conformance_level,
        caps,
        findings_summary,
        proof_receipt_index,
        security_receipt_index,
        ux_receipt_index,
        contract_db_receipt_index,
        benchmark_receipt_index,
        governance_receipt_index,
        exceptions,
        evidence_subjects,
        provenance: Provenance {
            attestation: "local-attestation-pending".to_string(),
            signature: None,
            build_url,
            attestation_sha256: None,
            source_commit: git_head(repo).or_else(|| git_head(&workspace_root())),
            source_dirty: git_dirty(repo).or_else(|| git_dirty(&workspace_root())),
            issuer: Some(default_issuer()),
            subject: None,
            signature_algorithm: None,
            signed_payload_sha256: None,
            publish_channel: Some(publish_channel()),
            verification_hint: Some(
                "verify with `jankurai certify --verify-attestation <path> --signing-key-env <ENV>`; add `--require-signature` for release gates"
                    .to_string(),
            ),
        },
    })
}

fn conformance_level(score: i32, caps: &[String], findings: &FindingsSummary) -> String {
    if score >= 93 && caps.is_empty() && findings.high == 0 && findings.critical == 0 {
        "HL3".to_string()
    } else if score >= 85 && findings.critical == 0 {
        "HL2".to_string()
    } else if score > 0 {
        "HL1".to_string()
    } else {
        "HL0".to_string()
    }
}

fn collect_evidence_subjects(
    root: &Path,
    proof_receipts: &[String],
    security_receipts: &[String],
    ux_receipts: &[String],
    contract_db_receipts: &[String],
    benchmark_receipts: &[String],
    governance_receipts: &[String],
    exceptions: &[String],
) -> Result<Vec<EvidenceSubject>> {
    let mut subjects: BTreeMap<String, String> = BTreeMap::new();

    insert_subject(&mut subjects, "agent/standard-version.toml", "standard-version");
    insert_subject(&mut subjects, "agent/repo-score.json", "repo-score");
    insert_subject(&mut subjects, "agent/repo-score.md", "repo-score-markdown");

    for path in proof_receipts {
        insert_subject(&mut subjects, path, "proof-receipt");
    }
    for path in security_receipts {
        insert_subject(&mut subjects, path, "security-receipt");
    }
    for path in ux_receipts {
        insert_subject(&mut subjects, path, "ux-receipt");
    }
    for path in contract_db_receipts {
        insert_subject(&mut subjects, path, "contract-db-receipt");
    }
    for path in benchmark_receipts {
        insert_subject(&mut subjects, path, "benchmark-receipt");
    }
    for path in governance_receipts {
        insert_subject(&mut subjects, path, "governance-receipt");
    }
    for path in exceptions {
        insert_subject(&mut subjects, path, "exception-inventory");
    }

    subjects
        .into_iter()
        .map(|(path, kind)| evidence_subject(root, &path, &kind))
        .collect()
}

fn insert_subject(subjects: &mut BTreeMap<String, String>, path: &str, kind: &str) {
    subjects
        .entry(path.to_string())
        .or_insert_with(|| kind.to_string());
}

fn evidence_subject(root: &Path, rel_path: &str, kind: &str) -> Result<EvidenceSubject> {
    let full_path = root.join(rel_path);

    if full_path.is_file() {
        let bytes = fs::read(&full_path)
            .with_context(|| format!("read evidence subject {}", full_path.display()))?;
        Ok(EvidenceSubject {
            path: rel_path.to_string(),
            kind: kind.to_string(),
            status: "present".to_string(),
            sha256: Some(format!("sha256:{}", sha256_bytes(&bytes))),
            byte_len: Some(bytes.len() as u64),
        })
    } else if full_path.is_dir() {
        Ok(EvidenceSubject {
            path: rel_path.to_string(),
            kind: kind.to_string(),
            status: "directory".to_string(),
            sha256: None,
            byte_len: None,
        })
    } else {
        Ok(EvidenceSubject {
            path: rel_path.to_string(),
            kind: kind.to_string(),
            status: "missing".to_string(),
            sha256: None,
            byte_len: None,
        })
    }
}

fn collect_matching_json<F>(root: &Path, rel_dir: &str, predicate: F) -> Result<Vec<String>>
where
    F: Fn(&str) -> bool,
{
    let mut paths = Vec::new();
    let dir = root.join(rel_dir);
    if !dir.is_dir() {
        return Ok(paths);
    }

    for entry in fs::read_dir(&dir)? {
        let entry = entry?;
        let file_name = entry.file_name();
        let file_name = file_name.to_string_lossy();
        if predicate(&file_name) {
            paths.push(format!("{rel_dir}/{file_name}"));
        }
    }

    Ok(dedup(paths))
}

fn dedup(mut values: Vec<String>) -> Vec<String> {
    values.sort();
    values.dedup();
    values
}

fn build_attestation(
    certification: &Certification,
    signing_key_env: Option<&str>,
    require_signature: bool,
) -> Result<CertificationAttestation> {
    let predicate = predicate_from_certification(certification);
    let subject_hash = sha256_json(&predicate)?;

    let mut attestation = CertificationAttestation {
        schema_version: certification.schema_version.clone(),
        generated_at: now_string(),
        subject: AttestationSubject {
            name: certification.repo.clone(),
            digest: format!("sha256:{subject_hash}"),
        },
        predicate_type: "https://jankurai.dev/attestations/certification/v1".to_string(),
        predicate,
        signer: None,
    };

    attestation.signer = maybe_sign_attestation(&attestation, signing_key_env, require_signature)?;
    Ok(attestation)
}

fn predicate_from_certification(certification: &Certification) -> CertificationPredicate {
    CertificationPredicate {
        standard_version: certification.standard_version.clone(),
        auditor_version: certification.auditor_version.clone(),
        paper_edition: certification.paper_edition.clone(),
        target_stack_id: certification.target_stack_id.clone(),
        generated_at: certification.generated_at.clone(),
        score: certification.score,
        conformance_level: certification.conformance_level.clone(),
        caps: certification.caps.clone(),
        findings_summary: AttestedFindingsSummary::from(&certification.findings_summary),
        proof_receipt_index: certification.proof_receipt_index.clone(),
        security_receipt_index: certification.security_receipt_index.clone(),
        ux_receipt_index: certification.ux_receipt_index.clone(),
        contract_db_receipt_index: certification.contract_db_receipt_index.clone(),
        benchmark_receipt_index: certification.benchmark_receipt_index.clone(),
        governance_receipt_index: certification.governance_receipt_index.clone(),
        exceptions: certification.exceptions.clone(),
        evidence_subjects: certification.evidence_subjects.clone(),
        provenance: CertificationPredicateProvenance {

            source_commit: certification.provenance.source_commit.clone(),
            source_dirty: certification.provenance.source_dirty,
            issuer: certification.provenance.issuer.clone(),
            publish_channel: certification.provenance.publish_channel.clone(),
        },
    }
}

fn maybe_sign_attestation(
    attestation: &CertificationAttestation,
    signing_key_env: Option<&str>,
    require_signature: bool,
) -> Result<Option<AttestationSigner>> {
    let Some(key_env) = signing_key_env else {
        if require_signature {
            bail!("--require-signature requires --signing-key-env <ENV>");
        }
        return Ok(None);
    };

    let key = match env::var(key_env) {
        Ok(value) if !value.is_empty() => value,
        _ if require_signature => {
            bail!("required signing key environment variable `{key_env}` is missing or empty")
        }
        _ => return Ok(None),
    };

    let mut unsigned = attestation.clone();
    unsigned.signer = None;
    let payload = serde_json::to_string(&unsigned)?;
    let payload_sha256 = format!("sha256:{}", sha256_bytes(payload.as_bytes()));
    let signature = expected_signature(&payload_sha256, &key);

    Ok(Some(AttestationSigner {
        key_id: format!("env:{key_env}"),
        algorithm: "local-sha256-keyed".to_string(),
        payload_sha256,
        signature,
    }))
}

fn verify_attestation(
    repo: &Path,
    path: &Path,
    signing_key_env: Option<&str>,
    require_signature: bool,
) -> Result<()> {
    let text = fs::read_to_string(path)
        .with_context(|| format!("read certification attestation {}", path.display()))?;
    let attestation: CertificationAttestation = serde_json::from_str(&text)
        .with_context(|| format!("parse certification attestation {}", path.display()))?;
    validation::validate_serializable(
        repo,
        ArtifactSchema::CertificationAttestation,
        &attestation,
    )?;

    match attestation.signer.as_ref() {
        Some(signer) => {
            let Some(key_env) = signing_key_env else {
                if require_signature {
                    bail!("signed attestation verification requires --signing-key-env <ENV>");
                }
                println!(
                    "attestation schema-valid; signature present but not checked because --signing-key-env was not supplied"
                );
                return Ok(());
            };

            let key = env::var(key_env)
                .with_context(|| format!("read signing key environment variable `{key_env}`"))?;
            if key.is_empty() {
                bail!("signing key environment variable `{key_env}` is empty");
            }

            let mut unsigned = attestation.clone();
            unsigned.signer = None;
            let payload = serde_json::to_string(&unsigned)?;
            let payload_sha256 = format!("sha256:{}", sha256_bytes(payload.as_bytes()));
            if signer.payload_sha256 != payload_sha256 {
                bail!(
                    "attestation payload hash mismatch: expected {}, found {}",
                    payload_sha256,
                    signer.payload_sha256
                );
            }

            let expected = expected_signature(&payload_sha256, &key);
            if signer.signature != expected {
                bail!("attestation signature mismatch");
            }

            println!("signed attestation verified: {}", attestation.subject.digest);
        }
        None => {
            if require_signature {
                bail!("attestation is unsigned and --require-signature was set");
            }
            println!("unsigned attestation schema-valid: {}", attestation.subject.digest);
        }
    }

    Ok(())
}

fn expected_signature(payload_sha256: &str, key: &str) -> String {
    format!(
        "local-sha256:{}",
        sha256_bytes(format!("{payload_sha256}:{key}").as_bytes())
    )
}

fn join_or_none(values: &[String]) -> String {
    if values.is_empty() {
        "none".to_string()
    } else {
        values.join(", ")
    }

fn git_head(repo: &Path) -> Option<String> {
    git_command(repo, &["rev-parse", "HEAD"]).and_then(non_empty)
}

fn git_dirty(repo: &Path) -> Option<bool> {
    git_command(repo, &["status", "--porcelain"]).map(|output| !output.trim().is_empty())
}

fn git_command(repo: &Path, args: &[&str]) -> Option<String> {
    let output = Command::new("git")
        .arg("-C")
        .arg(repo)
        .args(args)
        .output()
        .ok()?;

    if !output.status.success() {
        return None;
    }

    Some(String::from_utf8_lossy(&output.stdout).trim().to_string())
}

fn non_empty(value: String) -> Option<String> {
    if value.is_empty() {
        None
    } else {
        Some(value)
    }
}

fn default_build_url() -> Option<String> {
    let server = env::var("GITHUB_SERVER_URL").ok()?;
    let repo = env::var("GITHUB_REPOSITORY").ok()?;
    let run_id = env::var("GITHUB_RUN_ID").ok()?;
    Some(format!("{server}/{repo}/actions/runs/{run_id}"))
}

fn default_issuer() -> String {
    if let Ok(repo) = env::var("GITHUB_REPOSITORY") {
        format!("github-actions:{repo}")
    } else {
        "local".to_string()
    }
}

fn publish_channel() -> String {
    match (
        env::var("GITHUB_REF_TYPE").ok().as_deref(),
        env::var("GITHUB_EVENT_NAME").ok().as_deref(),
        env::var("CI").ok(),
    ) {
        (Some("tag"), _, _) => "release".to_string(),
        (_, Some("release"), _) => "release".to_string(),
        (_, Some("workflow_dispatch"), _) => "manual-ci".to_string(),
        (_, _, Some(_)) => "ci".to_string(),
        _ => "local".to_string(),
    }
}

fn sha256_json<T: Serialize>(value: &T) -> Result<String> {
    let text = serde_json::to_string(value)?;
    Ok(sha256_bytes(text.as_bytes()))
}

fn sha256_bytes(bytes: &[u8]) -> String {
    let mut hasher = Sha256::new();
    hasher.update(bytes);
    format!("{:x}", hasher.finalize())
}

fn escape_xml(input: &str) -> String {
    input
        .replace('&', "&amp;")
        .replace('<', "&lt;")
        .replace('>', "&gt;")
        .replace('"', "&quot;")
}

fn now_string() -> String {
    SystemTime::now()
        .duration_since(UNIX_EPOCH)
        .unwrap_or_default()
        .as_secs()
        .to_string()
}

use std::fs;
use std::path::PathBuf;
use std::process::Command;

use jankurai::validation::{self, ArtifactSchema};
use tempfile::tempdir;

fn binary_path() -> PathBuf {
    PathBuf::from(env!("CARGO_BIN_EXE_jankurai"))
}

#[test]
fn certify_emits_public_badge_report_and_signed_attestation() {
    let repo = tempdir().unwrap();
    let out_dir = tempdir().unwrap();

    let cert_path = out_dir.path().join("certification.json");
    let cert_md_path = out_dir.path().join("certification.md");
    let attestation_path = out_dir.path().join("attestation.json");
    let badge_path = out_dir.path().join("badge.svg");
    let public_md_path = out_dir.path().join("public.md");

    let status = Command::new(binary_path())
        .env("JANKURAI_TEST_CERT_KEY", "phase-12-fixture-key")
        .arg("certify")
        .arg(repo.path())
        .arg("--out")
        .arg(&cert_path)
        .arg("--md")
        .arg(&cert_md_path)
        .arg("--attestation")
        .arg(&attestation_path)
        .arg("--badge")
        .arg(&badge_path)
        .arg("--public-md")
        .arg(&public_md_path)
        .arg("--signing-key-env")
        .arg("JANKURAI_TEST_CERT_KEY")
        .arg("--require-signature")
        .status()
        .unwrap();

    assert!(status.success());

    let certification: serde_json::Value =
        serde_json::from_str(&fs::read_to_string(&cert_path).unwrap()).unwrap();
    validation::validate_value(repo.path(), ArtifactSchema::Certification, &certification)
        .unwrap();

    assert_eq!(certification["conformance_level"], "HL0");
    assert!(certification["evidence_subjects"].as_array().unwrap().len() >= 1);
    assert!(certification["provenance"]["signature"]
        .as_str()
        .unwrap()
        .starts_with("local-sha256:"));
    assert_eq!(
        certification["provenance"]["signature_algorithm"],
        "local-sha256-keyed"
    );

    let attestation: serde_json::Value =
        serde_json::from_str(&fs::read_to_string(&attestation_path).unwrap()).unwrap();
    validation::validate_value(
        repo.path(),
        ArtifactSchema::CertificationAttestation,
        &attestation,
    )
    .unwrap();
    assert!(attestation["signer"]["signature"]
        .as_str()
        .unwrap()
        .starts_with("local-sha256:"));

    let badge = fs::read_to_string(&badge_path).unwrap();
    assert!(badge.starts_with("<svg"));
    assert!(badge.contains("jankurai"));

    let public_md = fs::read_to_string(&public_md_path).unwrap();
    assert!(public_md.starts_with("# Built with Jankurai"));
    assert!(public_md.contains("Signed release verification"));

    let verify_status = Command::new(binary_path())
        .env("JANKURAI_TEST_CERT_KEY", "phase-12-fixture-key")
        .arg("certify")
        .arg(repo.path())
        .arg("--verify-attestation")
        .arg(&attestation_path)
        .arg("--signing-key-env")
        .arg("JANKURAI_TEST_CERT_KEY")
        .arg("--require-signature")
        .status()
        .unwrap();

    assert!(verify_status.success());
}

What this changes functionally:

`jankurai certify` now becomes the Phase 12 public evidence publisher rather than only a local JSON emitter. It still supports the old `--out` and `--md` behavior, but adds `--attestation`, `--badge`, `--public-md`, `--signing-key-env`, `--require-signature`, `--build-url`, and `--verify-attestation`.

The result is a bundle that Phase 13 can safely depend on:

```text
bench/govern/audit evidence
        ↓
certification JSON
        ↓
attestation JSON with evidence hashes
        ↓
optional keyed signature
        ↓
public badge + public report
        ↓
CI-uploaded artifact bundle
````

That closes the deferred Phase 12 gap without giving agents new mutation authority.
