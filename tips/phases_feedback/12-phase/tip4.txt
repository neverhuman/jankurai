**Phase name: Phase 12 — Benchmark Certification And Governance**

I would treat this as a Phase 12 hardening/completion slice, not a Phase 13 mutation slice. The moonshot loop is `intent -> bounded agents -> proof lanes -> evidence -> expired exceptions -> reusable primitives`; Phase 12’s job is to make the “evidence” part public, deterministic, and governance-bound rather than merely local. ([GitHub][1])

The current Phase 12 doc already has local `bench`, `certify`, and `govern` artifacts, and explicitly says external signing, hosted dashboard, and public badge publishing were not added. ([GitHub][2]) The Phase 00 router likewise marks Phase 12 as `complete-with-deferred-work` because “local certification/governance exist” while “external signing/badges/dashboard” are deferred, and it names Phase 12 public evidence as the next high-leverage area. ([GitHub][3]) The release plan also points in this direction: badges, public benchmark evidence, governance policy, and stable compliance evidence are part of the broader release/certification surface. ([GitHub][4])

I could not run the repo locally in this sandbox because the execution container could not resolve `github.com`; this patch is based on the raw GitHub sources I could inspect through browsing. Run the validation commands at the end in the repo root.

Below is one copy-pasteable implementation script. It adds a **`jankurai publish`** command that consumes the existing Phase 12 artifacts, validates them, creates a schema-backed public evidence bundle, emits badge JSON/SVG, captures build URL/git provenance when available, computes SHA-256 artifact digests, updates CI to publish those artifacts, and marks Phase 12 hardened.

pub fn run(args: PublishArgs) -> Result<()> {
    let bundle = build_public_evidence_bundle(
        &args.repo,
        &args.certification,
        &args.benchmark,
        &args.governance,
    )?;

    if let Some(path) = args.out.as_deref() {
        validation::write_json(
            &args.repo,
            ArtifactSchema::PublicEvidenceBundle,
            path,
            &bundle,
        )?;
    } else {
        validation::validate_serializable(
            &args.repo,
            ArtifactSchema::PublicEvidenceBundle,
            &bundle,
        )?;
        println!("{}", serde_json::to_string_pretty(&bundle)?);
    }

    if let Some(path) = args.md.as_deref() {
        crate::render::write_markdown(path, &render_markdown(&bundle))?;
    }

    if let Some(path) = args.badge_json.as_deref() {
        validation::write_json(
            &args.repo,
            ArtifactSchema::CertificationBadge,
            path,
            &bundle.badge,
        )?;
    }

    if let Some(path) = args.badge_svg.as_deref() {
        write_text(path, &render_badge_svg(&bundle.badge))?;
    }

    Ok(())
}

pub fn build_public_evidence_bundle(
    repo: &Path,
    certification_path: &str,
    benchmark_path: &str,
    governance_path: &str,
) -> Result<PublicEvidenceBundle> {
    let release = load_release_data(repo)?;

    let certification_value =
        read_validated_json(repo, certification_path, ArtifactSchema::Certification)?;
    let benchmark_value = read_validated_json(repo, benchmark_path, ArtifactSchema::BenchmarkReport)?;
    let governance_value =
        read_validated_json(repo, governance_path, ArtifactSchema::GovernancePolicy)?;

    let certification: CertificationInput =
        serde_json::from_value(certification_value).context("parse certification artifact")?;
    let benchmark: BenchmarkInput =
        serde_json::from_value(benchmark_value).context("parse benchmark report artifact")?;
    let governance: GovernanceInput =
        serde_json::from_value(governance_value).context("parse governance policy artifact")?;

    if certification.standard_version != release.standard_version {
        bail!(
            "certification standard_version `{}` does not match manifest `{}`",
            certification.standard_version,
            release.standard_version
        );
    }
    if governance.standard_version != release.standard_version {
        bail!(
            "governance standard_version `{}` does not match manifest `{}`",
            governance.standard_version,
            release.standard_version
        );
    }

    let governance_summary = GovernanceSummary {
        standard_version: governance.standard_version.clone(),
        minimum_score: governance.minimum_score,
        fail_on: governance.fail_on.clone(),
        advisory_on: governance.advisory_on.clone(),
        update_channel: governance.update_channel.clone(),
        rfc_path: governance.rfc_path.clone(),
    };

    let (publishable, public_status, blocking_reasons) =
        evaluate_publishability(&certification, &benchmark, &governance);

    let mut artifacts = Vec::new();
    add_artifact(
        &mut artifacts,
        repo,
        certification_path,
        "certification",
        Some(ArtifactSchema::Certification.rel_path()),
        true,
    );
    add_artifact(
        &mut artifacts,
        repo,
        benchmark_path,
        "benchmark-report",
        Some(ArtifactSchema::BenchmarkReport.rel_path()),
        true,
    );
    add_artifact(
        &mut artifacts,
        repo,
        governance_path,
        "governance-policy",
        Some(ArtifactSchema::GovernancePolicy.rel_path()),
        true,
    );
    add_artifact(
        &mut artifacts,
        repo,
        "agent/standard-version.toml",
        "version-manifest",
        Some(ArtifactSchema::StandardVersion.rel_path()),
        true,
    );

    for path in &certification.proof_receipt_index {
        add_artifact(
            &mut artifacts,
            repo,
            path,
            "proof-receipt",
            Some(ArtifactSchema::ProofReceipt.rel_path()),
            false,
        );
    }
    for path in &certification.security_receipt_index {
        add_artifact(
            &mut artifacts,
            repo,
            path,
            "security-receipt",
            Some(ArtifactSchema::SecurityEvidence.rel_path()),
            false,
        );
    }
    for path in &certification.ux_receipt_index {
        add_artifact(
            &mut artifacts,
            repo,
            path,
            "ux-receipt",
            Some(ArtifactSchema::UxQaReport.rel_path()),
            false,
        );
    }
    for path in &certification.contract_db_receipt_index {
        add_artifact(&mut artifacts, repo, path, "contract-db-receipt", None, false);
    }
    for path in &certification.exceptions {
        add_artifact(&mut artifacts, repo, path, "exception-inventory", None, false);
    }

    artifacts.sort_by(|a, b| (&a.path, &a.role).cmp(&(&b.path, &b.role)));
    artifacts.dedup_by(|a, b| a.path == b.path && a.role == b.role);

    let badge = CertificationBadge {
        schema_version: release.schema_version.clone(),
        label: "jankurai".to_string(),
        message: format!(
            "{} score {}",
            certification.conformance_level, certification.score
        ),
        color: badge_color(publishable, &public_status).to_string(),
        score: certification.score,
        conformance_level: certification.conformance_level.clone(),
        standard_version: certification.standard_version.clone(),
        artifact: Some("target/jankurai/public/p12-public-evidence.json".to_string()),
    };

    let subject_material = json!({
        "certification_path": certification_path,
        "benchmark_path": benchmark_path,
        "governance_path": governance_path,
        "score": certification.score,
        "conformance_level": certification.conformance_level,
        "standard_version": certification.standard_version,
        "benchmark_summary": benchmark.summary,
        "governance": governance_summary,
        "artifacts": artifacts,
    });
    let subject_digest = digest_json(&subject_material)?;

    let mut limitations = benchmark.limitations.clone();
    limitations.push(
        "Attestation is local digest evidence; external Sigstore/cosign signing can sign the subject_digest without changing this bundle."
            .to_string(),
    );
    limitations.push(
        "Hosted dashboards are intentionally not required for Phase 12 completion; CI artifacts and badge outputs are the publishable surface."
            .to_string(),
    );

    let bundle = PublicEvidenceBundle {
        schema_version: release.schema_version,
        generated_at: now_string(),
        repo: certification.repo,
        standard_version: release.standard_version,
        auditor_version: certification.auditor_version,
        paper_edition: certification.paper_edition,
        target_stack_id: certification.target_stack_id,
        public_status,
        publishable,
        score: certification.score,
        conformance_level: certification.conformance_level,
        findings_summary: certification.findings_summary,
        benchmark_summary: benchmark.summary,
        governance: governance_summary,
        badge,
        artifacts,
        attestation: PublicAttestation {
            kind: "local-sha256-attestation".to_string(),
            predicate_type: "https://jankurai.dev/attestations/phase12-public-evidence/v1"
                .to_string(),
            signature: format!(
                "local-sha256-attestation:{}",
                subject_digest.trim_start_matches("sha256:")
            ),
            subject_digest,
            signing_key_hint:
                "local digest only; sign this subject_digest with external identity if required"
                    .to_string(),
            command: "jankurai publish".to_string(),
            git_head: git_head(repo),
            build_url: build_url(),
        },
        validation_commands: vec![
            "cargo run -p jankurai -- bench . --out target/jankurai/p12-benchmark-report.json --md target/jankurai/p12-benchmark-report.md".to_string(),
            "cargo run -p jankurai -- certify . --out target/jankurai/p12-certification.json --md target/jankurai/p12-certification.md".to_string(),
            "cargo run -p jankurai -- govern . --out target/jankurai/p12-governance-policy.json --md target/jankurai/p12-governance-policy.md".to_string(),
            "cargo run -p jankurai -- publish . --certification target/jankurai/p12-certification.json --benchmark target/jankurai/p12-benchmark-report.json --governance target/jankurai/p12-governance-policy.json --out target/jankurai/public/p12-public-evidence.json --md target/jankurai/public/p12-public-evidence.md --badge-json target/jankurai/public/jankurai-badge.json --badge-svg target/jankurai/public/jankurai-badge.svg".to_string(),
        ],
        blocking_reasons,
        limitations,
    };

    validation::validate_serializable(repo, ArtifactSchema::PublicEvidenceBundle, &bundle)?;
    validation::validate_serializable(repo, ArtifactSchema::CertificationBadge, &bundle.badge)?;
    Ok(bundle)
}

fn evaluate_publishability(
    certification: &CertificationInput,
    benchmark: &BenchmarkInput,
    governance: &GovernanceInput,
) -> (bool, String, Vec<String>) {
    let mut reasons = Vec::new();

    if certification.score < governance.minimum_score {
        reasons.push(format!(
            "score {} is below governance minimum {}",
            certification.score, governance.minimum_score
        ));
    }

    if certification.findings_summary.critical > 0 {
        reasons.push(format!(
            "{} critical finding(s) present",
            certification.findings_summary.critical
        ));
    }

    if certification.findings_summary.high > 0 {
        reasons.push(format!(
            "{} high finding(s) present",
            certification.findings_summary.high
        ));
    }

    if !certification.caps.is_empty() {
        reasons.push(format!("hard caps applied: {}", certification.caps.join(", ")));
    }

    if benchmark.summary.failed > 0 {
        reasons.push(format!(
            "{} benchmark task(s) failed",
            benchmark.summary.failed
        ));
    }

    if benchmark.summary.inconclusive > 0 {
        reasons.push(format!(
            "{} benchmark task(s) inconclusive",
            benchmark.summary.inconclusive
        ));
    }

    let publishable = reasons.is_empty();
    let status = if publishable {
        "publishable"
    } else if certification.score > 0 {
        "advisory"
    } else {
        "blocked"
    };

    (publishable, status.to_string(), reasons)
}

fn badge_color(publishable: bool, status: &str) -> &'static str {
    if publishable {
        "brightgreen"
    } else if status == "advisory" {
        "yellow"
    } else {
        "red"
    }
}

fn add_artifact(
    artifacts: &mut Vec<PublishedArtifact>,
    repo: &Path,
    path: &str,
    role: &str,
    schema: Option<&str>,
    public: bool,
) {
    let normalized = normalize_path(repo, path);
    if artifacts
        .iter()
        .any(|artifact| artifact.path == normalized && artifact.role == role)
    {
        return;
    }

    let abs = resolve_path(repo, path);
    let present = abs.exists();
    let sha256 = if present && abs.is_file() {
        sha256_file(&abs).ok()
    } else {
        None
    };

    artifacts.push(PublishedArtifact {
        path: normalized,
        role: role.to_string(),
        present,
        sha256,
        schema: schema.map(str::to_string),
        public,
    });
}

fn read_validated_json(repo: &Path, path: &str, schema: ArtifactSchema) -> Result<Value> {
    let abs = resolve_path(repo, path);
    let text = fs::read_to_string(&abs).with_context(|| format!("read {}", abs.display()))?;
    let value: Value =
        serde_json::from_str(&text).with_context(|| format!("parse {}", abs.display()))?;
    validation::validate_value(repo, schema, &value).with_context(|| {
        format!(
            "validate {} against {}",
            normalize_path(repo, path),
            schema.rel_path()
        )
    })?;
    Ok(value)
}

fn resolve_path(repo: &Path, path: &str) -> PathBuf {
    let candidate = PathBuf::from(path);
    if candidate.is_absolute() {
        candidate
    } else {
        repo.join(candidate)
    }
}

fn normalize_path(repo: &Path, path: &str) -> String {
    let candidate = PathBuf::from(path);
    let rel = if candidate.is_absolute() {
        candidate
            .strip_prefix(repo)
            .map(Path::to_path_buf)
            .unwrap_or(candidate)
    } else {
        candidate
    };
    rel.to_string_lossy().replace('\\', "/")
}

fn sha256_file(path: &Path) -> Result<String> {
    let bytes = fs::read(path).with_context(|| format!("read {}", path.display()))?;
    Ok(format!("sha256:{:x}", Sha256::digest(bytes)))
}

fn digest_json(value: &Value) -> Result<String> {
    let bytes = serde_json::to_vec(value)?;
    Ok(format!("sha256:{:x}", Sha256::digest(bytes)))
}

fn git_head(repo: &Path) -> Option<String> {
    Command::new("git")
        .args(["rev-parse", "HEAD"])
        .current_dir(repo)
        .output()
        .ok()
        .filter(|output| output.status.success())
        .and_then(|output| String::from_utf8(output.stdout).ok())
        .map(|value| value.trim().to_string())
        .filter(|value| !value.is_empty())
}

fn build_url() -> Option<String> {
    match (
        env::var("GITHUB_SERVER_URL"),
        env::var("GITHUB_REPOSITORY"),
        env::var("GITHUB_RUN_ID"),
    ) {
        (Ok(server), Ok(repository), Ok(run_id)) => {
            Some(format!("{server}/{repository}/actions/runs/{run_id}"))
        }
        _ => env::var("BUILD_URL").ok(),
    }
}

fn write_text(path: &str, content: &str) -> Result<()> {
    if path != "-" {
        if let Some(parent) = Path::new(path).parent() {
            if !parent.as_os_str().is_empty() {
                fs::create_dir_all(parent)?;
            }
        }
        fs::write(path, content)?;
    } else {
        print!("{content}");
    }
    Ok(())
}

fn render_markdown(bundle: &PublicEvidenceBundle) -> String {
    use std::fmt::Write;

    let mut out = String::new();
    let _ = writeln!(out, "# jankurai Public Evidence");
    let _ = writeln!(out);
    let _ = writeln!(out, "- public status: `{}`", bundle.public_status);
    let _ = writeln!(out, "- publishable: `{}`", bundle.publishable);
    let _ = writeln!(out, "- repo: `{}`", bundle.repo);
    let _ = writeln!(out, "- standard version: `{}`", bundle.standard_version);
    let _ = writeln!(out, "- auditor version: `{}`", bundle.auditor_version);
    let _ = writeln!(out, "- paper edition: `{}`", bundle.paper_edition);
    let _ = writeln!(out, "- target stack ID: `{}`", bundle.target_stack_id);
    let _ = writeln!(out, "- score: `{}`", bundle.score);
    let _ = writeln!(out, "- conformance: `{}`", bundle.conformance_level);
    let _ = writeln!(
        out,
        "- benchmark: passed=`{}` failed=`{}` inconclusive=`{}`",
        bundle.benchmark_summary.passed,
        bundle.benchmark_summary.failed,
        bundle.benchmark_summary.inconclusive
    );
    let _ = writeln!(
        out,
        "- findings: critical=`{}` high=`{}` medium=`{}` low=`{}`",
        bundle.findings_summary.critical,
        bundle.findings_summary.high,
        bundle.findings_summary.medium,
        bundle.findings_summary.low
    );
    let _ = writeln!(
        out,
        "- governance: min_score=`{}` update_channel=`{}` fail_on=`{}`",
        bundle.governance.minimum_score,
        bundle.governance.update_channel,
        bundle.governance.fail_on.join(", ")
    );

    if !bundle.blocking_reasons.is_empty() {
        let _ = writeln!(out);
        let _ = writeln!(out, "## Blocking Reasons");
        let _ = writeln!(out);
        for reason in &bundle.blocking_reasons {
            let _ = writeln!(out, "- {}", reason);
        }
    }

    let _ = writeln!(out);
    let _ = writeln!(out, "## Badge");
    let _ = writeln!(out);
    let _ = writeln!(out, "- label: `{}`", bundle.badge.label);
    let _ = writeln!(out, "- message: `{}`", bundle.badge.message);
    let _ = writeln!(out, "- color: `{}`", bundle.badge.color);

    let _ = writeln!(out);
    let _ = writeln!(out, "## Attestation");
    let _ = writeln!(out);
    let _ = writeln!(out, "- kind: `{}`", bundle.attestation.kind);
    let _ = writeln!(
        out,
        "- predicate type: `{}`",
        bundle.attestation.predicate_type
    );
    let _ = writeln!(
        out,
        "- subject digest: `{}`",
        bundle.attestation.subject_digest
    );
    let _ = writeln!(out, "- signature: `{}`", bundle.attestation.signature);
    if let Some(git_head) = &bundle.attestation.git_head {
        let _ = writeln!(out, "- git head: `{}`", git_head);
    }
    if let Some(build_url) = &bundle.attestation.build_url {
        let _ = writeln!(out, "- build URL: `{}`", build_url);
    }

    let _ = writeln!(out);
    let _ = writeln!(out, "## Artifacts");
    let _ = writeln!(out);
    let _ = writeln!(out, "| Role | Path | Present | Public | SHA-256 | Schema |");
    let _ = writeln!(out, "| --- | --- | --- | --- | --- | --- |");
    for artifact in &bundle.artifacts {
        let _ = writeln!(
            out,
            "| `{}` | `{}` | `{}` | `{}` | `{}` | `{}` |",
            artifact.role,
            artifact.path,
            artifact.present,
            artifact.public,
            artifact.sha256.as_deref().unwrap_or("none"),
            artifact.schema.as_deref().unwrap_or("none")
        );
    }

    let _ = writeln!(out);
    let _ = writeln!(out, "## Reproduction");
    let _ = writeln!(out);
    for command in &bundle.validation_commands {
        let _ = writeln!(out, "- `{}`", command);
    }

    if !bundle.limitations.is_empty() {
        let _ = writeln!(out);
        let _ = writeln!(out, "## Limitations");
        let _ = writeln!(out);
        for limitation in &bundle.limitations {
            let _ = writeln!(out, "- {}", limitation);
        }
    }

    out
}

fn render_badge_svg(badge: &CertificationBadge) -> String {
    let label = escape_xml(&badge.label);
    let message = escape_xml(&badge.message);
    let color = badge_hex_color(&badge.color);
    let label_width = ((label.chars().count() as i32 * 7) + 22).max(64);
    let message_width = ((message.chars().count() as i32 * 7) + 22).max(86);
    let width = label_width + message_width;
    let label_center = label_width / 2;
    let message_center = label_width + (message_width / 2);

    format!(
        r##"<svg xmlns="http://www.w3.org/2000/svg" width="{width}" height="20" role="img" aria-label="{label}: {message}">
  <title>{label}: {message}</title>
  <linearGradient id="s" x2="0" y2="100%">
    <stop offset="0" stop-color="#fff" stop-opacity=".7"/>
    <stop offset=".1" stop-color="#aaa" stop-opacity=".1"/>
    <stop offset=".9" stop-color="#000" stop-opacity=".3"/>
    <stop offset="1" stop-color="#000" stop-opacity=".5"/>
  </linearGradient>
  <clipPath id="r">
    <rect width="{width}" height="20" rx="3" fill="#fff"/>
  </clipPath>
  <g clip-path="url(#r)">
    <rect width="{label_width}" height="20" fill="#555"/>
    <rect x="{label_width}" width="{message_width}" height="20" fill="{color}"/>
    <rect width="{width}" height="20" fill="url(#s)"/>
  </g>
  <g fill="#fff" text-anchor="middle" font-family="Verdana,Geneva,DejaVu Sans,sans-serif" font-size="11">
    <text x="{label_center}" y="15" fill="#010101" fill-opacity=".3">{label}</text>
    <text x="{label_center}" y="14">{label}</text>
    <text x="{message_center}" y="15" fill="#010101" fill-opacity=".3">{message}</text>
    <text x="{message_center}" y="14">{message}</text>
  </g>
</svg>
"##
    )
}

fn badge_hex_color(color: &str) -> &'static str {
    match color {
        "brightgreen" => "#4c1",
        "green" => "#97ca00",
        "yellowgreen" => "#a4a61d",
        "yellow" => "#dfb317",
        "orange" => "#fe7d37",
        "red" => "#e05d44",
        "blue" => "#007ec6",
        _ => "#9f9f9f",
    }
}

fn escape_xml(value: &str) -> String {
    value
        .replace('&', "&amp;")
        .replace('<', "&lt;")
        .replace('>', "&gt;")
        .replace('"', "&quot;")
}

        .to_string()

    pub github: bool,
    pub mode: String,
    pub min_score: i32,
}

pub fn install(args: CiInstallArgs) -> Result<()> {
    if !args.github {
        bail!("only `jankurai ci install --github` is currently supported");
    }

    let path = args.repo.join(".github/workflows/jankurai.yml");
    if let Some(parent) = path.parent() {
        fs::create_dir_all(parent).with_context(|| format!("create {}", parent.display()))?;

        return Ok(());

    Ok(())
}

# Phase 12 Public Evidence

Phase 12 turns local benchmark, certification, and governance artifacts into a publishable evidence surface.

## Command Flow

Run these commands from the repository root:

```bash
cargo run -p jankurai -- bench . \
  --out target/jankurai/p12-benchmark-report.json \
  --md target/jankurai/p12-benchmark-report.md

cargo run -p jankurai -- certify . \

  --md target/jankurai/p12-certification.md

cargo run -p jankurai -- govern . \
  --out target/jankurai/p12-governance-policy.json \
  --md target/jankurai/p12-governance-policy.md

cargo run -p jankurai -- publish . \
  --certification target/jankurai/p12-certification.json \
  --benchmark target/jankurai/p12-benchmark-report.json \
  --governance target/jankurai/p12-governance-policy.json \
  --out target/jankurai/public/p12-public-evidence.json \
  --md target/jankurai/public/p12-public-evidence.md \
  --badge-json target/jankurai/public/jankurai-badge.json \
  --badge-svg target/jankurai/public/jankurai-badge.svg
````

## Produced Artifacts

| Artifact                                          | Contract                                         |
| ------------------------------------------------- | ------------------------------------------------ |
| `target/jankurai/p12-benchmark-report.json`       | `schemas/benchmark-report.schema.json`           |
| `target/jankurai/p12-certification.json`          | `schemas/certification.schema.json`              |
| `target/jankurai/p12-governance-policy.json`      | `schemas/governance-policy.schema.json`          |
| `target/jankurai/public/p12-public-evidence.json` | `schemas/public-evidence-bundle.schema.json`     |
| `target/jankurai/public/jankurai-badge.json`      | `schemas/certification-badge.schema.json`        |
| `target/jankurai/public/jankurai-badge.svg`       | self-contained badge derived from the badge JSON |

## Publishability Rules

`jankurai publish` marks a bundle as `publishable` only when:

* the certification score is at least the governance minimum score;
* no critical or high findings are present;
* no hard caps are applied;
* the benchmark report has zero failed and zero inconclusive tasks;
* the certification and governance artifacts agree with `agent/standard-version.toml`.

If any gate fails, the bundle remains useful but is marked `advisory` or `blocked` and includes `blocking_reasons`.

## Attestation Model

The public evidence bundle includes a local SHA-256 attestation over the certification, benchmark summary, governance policy, and artifact digest index. CI captures `GITHUB_SERVER_URL`, `GITHUB_REPOSITORY`, and `GITHUB_RUN_ID` when present. This gives public evidence a stable subject digest without introducing private key storage into the repository.

External identity-backed signing can be added later by signing `attestation.subject_digest`; the bundle schema does not need to change.

## Dashboard Policy

A hosted dashboard is optional distribution, not a Phase 12 dependency. The hardened Phase 12 surface is:

1. schema-valid JSON;
2. human Markdown;
3. badge JSON;
4. badge SVG;
5. CI artifact publishing;
6. local digest attestation.

# Phase 12: Benchmark Certification And Governance

Status: hardened
Owner: standard
Last reviewed: 2026-05-03
Parallel MCP candidate: yes

## Objective

Prove the Jankurai thesis publicly and make conformance meaningful. Phase 12 builds benchmark corpora, certification artifacts, badges, release attestations, rule governance, and organization-level reporting. The exit state is that "Built with Jankurai" has evidence behind it.

## Hardened Exit State

Phase 12 is hardened when local evidence is converted into a reproducible public evidence surface:

* `jankurai bench` emits a schema-valid `BenchmarkReport`.
* `jankurai certify` emits a schema-valid `Certification`.
* `jankurai govern` emits a schema-valid `GovernancePolicy`.
* `jankurai publish` consumes those three artifacts, validates them, computes artifact digests, evaluates publishability against governance policy, and emits:

  * `target/jankurai/public/p12-public-evidence.json`
  * `target/jankurai/public/p12-public-evidence.md`
  * `target/jankurai/public/jankurai-badge.json`
  * `target/jankurai/public/jankurai-badge.svg`
* CI uploads all Phase 12 evidence artifacts.
* The public bundle includes a local SHA-256 attestation and CI build URL when available.
* Hosted dashboards remain optional distribution, not a completion dependency.

Completed and retained:

* benchmark summary

* local attestation subject digest
* build URL and git HEAD when available
* reproduction commands
* blocking reasons when evidence is not publishable

## Publishability Rules

`jankurai publish` marks evidence `publishable` only when:

* certification score is at least the governance minimum score
* critical and high finding counts are zero
* no hard caps are present
* benchmark failures are zero
* benchmark inconclusive counts are zero
* certification and governance standard versions match `agent/standard-version.toml`

Otherwise the evidence remains schema-valid but is marked `advisory` or `blocked`.
