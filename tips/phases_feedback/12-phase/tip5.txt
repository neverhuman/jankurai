## Phase name

**Phase 12: Benchmark Certification And Governance**

I’m treating Phase 12 as the correct place for this work because the repo’s moonshot is explicitly about turning intent into bounded proof, durable evidence, expired exceptions, and reusable primitives; Phase 12 is where that evidence becomes externally meaningful. The repo README frames Jankurai as a control plane for ownership, proof, generated zones, and repair evidence, and repeats the operating loop `intent -> bounded agents -> proof lanes -> evidence -> expired exceptions -> reusable primitives`. ([GitHub][1]) The moonshot doc says the repo should make ownership, proof, generated truth, durable evidence, and expiring exceptions machine-readable. ([GitHub][2])

Phase 12’s current file says local `bench`, `certify`, and `govern` artifacts exist, but that “external signing service, hosted dashboard, or public badge publishing” was not added. ([GitHub][3]) The phase router also marks Phase 12 as `complete-with-deferred-work` because “external signing/badges/dashboard” remained deferred. ([GitHub][4]) The release plan specifically calls for compliance badges, benchmark publishing, raw evidence, and governance as adoption surfaces. ([GitHub][5])

The solution below finishes Phase 12 without requiring a hosted service. It adds:

1. **Public badge JSON** compatible with static badge endpoints.
2. **Local release attestation JSON** with SHA-256 material digests.
3. **Optional HMAC-SHA256 signature** using `JANKURAI_ATTESTATION_KEY`.
4. **Release report Markdown** for public artifact publishing.
5. **Certification governance gate** with score, conformance, findings, caps, and evidence checks.
6. **Governance policy hardening** that names the certification gate and required evidence.
7. **CI artifact publishing** for Phase 12 public evidence.
8. **Justfile lane** for repeatable Phase 12 proof.
9. **Schema coverage** for every new machine-readable artifact.
10. **Phase 12 status update** from deferred local evidence to hardened public evidence.

I could not run the repo locally because the execution container could not resolve GitHub for cloning, so treat this as a complete copy-paste implementation that should be applied and then validated with the commands at the end.

pub fn run(args: CertifyArgs) -> Result<()> {
    let mut certification = build_certification(&args.repo)?;
    if let Some(build_url) = args.build_url.clone() {
        certification.provenance.build_url = Some(build_url);
    }

    certification.governance_decision = evaluate_certification(
        &certification,
        args.minimum_score,
        &args.require_level,
    );

    let certification_json_paths =
        collect_output_paths(args.out.as_ref(), args.publish_path("certification.json"));
    let certification_md_paths =
        collect_output_paths(args.md.as_ref(), args.publish_path("certification.md"));
    let badge_json_paths =
        collect_output_paths(args.badge_out.as_ref(), args.publish_path("badge.json"));
    let badge_md_paths =
        collect_output_paths(args.badge_md.as_ref(), args.publish_path("BADGE.md"));
    let attestation_json_paths = collect_output_paths(
        args.attestation_out.as_ref(),
        args.publish_path("attestation.json"),
    );
    let report_md_paths =
        collect_output_paths(args.report_md.as_ref(), args.publish_path("report.md"));

    if let Some(attestation_path) = attestation_json_paths.first() {
        certification.provenance.attestation = format!("local-release-attestation:{attestation_path}");
    }

    if certification_json_paths.is_empty() {
        validation::validate_serializable(
            &args.repo,
            ArtifactSchema::Certification,
            &certification,
        )?;
        println!("{}", serde_json::to_string_pretty(&certification)?);
    } else {
        for path in &certification_json_paths {
            validation::write_json(
                &args.repo,
                ArtifactSchema::Certification,
                path,
                &certification,
            )?;
        }
    }

    for path in &certification_md_paths {
        crate::render::write_markdown(path, &render_markdown(&certification))?;
    }

    let badge = build_badge(&certification);
    for path in &badge_json_paths {
        validation::write_json(&args.repo, ArtifactSchema::CertificationBadge, path, &badge)?;
    }
    for path in &badge_md_paths {
        crate::render::write_markdown(path, &render_badge_markdown(&badge, badge_json_paths.first()))?;
    }

    let attestation = build_release_attestation(
        &args.repo,
        &certification,
        certification_json_paths.first().map(String::as_str),
        badge_json_paths.first().map(String::as_str),
        &args.signing_key_env,
    )?;
    for path in &attestation_json_paths {
        validation::write_json(
            &args.repo,
            ArtifactSchema::ReleaseAttestation,
            path,
            &attestation,
        )?;
    }

    for path in &report_md_paths {
        crate::render::write_markdown(
            path,
            &render_public_report(
                &certification,
                &badge,
                &attestation,
                certification_json_paths.first(),
                badge_json_paths.first(),
                attestation_json_paths.first(),
            ),
        )?;
    }

    if args.gate && !certification.governance_decision.passed {
        bail!(
            "certification gate failed: {}",
            certification.governance_decision.reasons.join("; ")
        );
    }

    Ok(())
}

pub fn build_certification(repo: &Path) -> Result<Certification> {
    let release = load_release_data(repo)?;
    let score_path = repo.join("agent/repo-score.json");
    let score = read_repo_score(repo)?;
    let evidence_root = if score_path.exists() {
        repo.to_path_buf()
    } else {
        workspace_root()
    };

    let (score_value, caps, mut findings_summary) = match score.as_ref() {
        Some(summary) => (
            summary.score,
            summary.caps.clone(),
            FindingsSummary {
                critical: summary.findings.critical,
                high: summary.findings.high,
                medium: summary.findings.medium,
                low: summary.findings.low,
                notes: vec![
                    format!("repo score artifact: {}", score_path.display()),
                    "release evidence artifact generated locally; external hosting is optional".to_string(),
                ],
            },
        ),
        None => (
            0,
            Vec::new(),
            FindingsSummary {
                critical: 0,
                high: 0,
                medium: 0,
                low: 0,
                notes: vec![
                    format!("score artifact missing: {}", score_path.display()),
                    "release evidence artifact generated locally; external hosting is optional".to_string(),
                ],
            },
        ),
    };

    if let Some(summary) = score.as_ref() {
        if summary.caps.is_empty() {
            findings_summary
                .notes
                .push("no hard caps reported in the score artifact".to_string());
        }
    }

    let proof_receipt_index = collect_receipts(&evidence_root)?;
    let security_receipt_index = collect_security_receipts(&evidence_root)?;
    let ux_receipt_index = collect_ux_receipts(&evidence_root)?;
    let contract_db_receipt_index = collect_contract_db_receipts(&evidence_root)?;
    let exceptions = collect_exceptions(&evidence_root)?;
    let conformance_level = conformance_level(score_value, &caps, &findings_summary);
    let evidence_status = evidence_status(
        score.is_some(),
        &proof_receipt_index,
        &security_receipt_index,
        &ux_receipt_index,
        &contract_db_receipt_index,
        &exceptions,
    );

    let mut certification = Certification {
        schema_version: release.schema_version,
        generated_at: now_string(),
        repo: repo.display().to_string(),
        standard_version: release.standard_version,
        auditor_version: release.auditor_version,
        paper_edition: release.paper_edition,
        target_stack_id: release.target_stack_id,
        score: score_value,
        conformance_level,
        caps,
        findings_summary,
        proof_receipt_index,
        security_receipt_index,
        ux_receipt_index,
        contract_db_receipt_index,
        exceptions,
        evidence_status,
        governance_decision: CertificationDecision {
            status: "pending".to_string(),
            passed: false,
            minimum_score: 85,
            required_level: "HL2".to_string(),
            reasons: vec!["governance gate not evaluated yet".to_string()],
        },
        provenance: Provenance {
            attestation: "local-artifact-only".to_string(),
            signature: None,
            build_url: default_build_url(),
        },
    };

    certification.governance_decision = evaluate_certification(&certification, 85, "HL2");
    Ok(certification)
}

pub fn build_badge(certification: &Certification) -> CertificationBadge {
    let is_error = !certification.governance_decision.passed;
    let color = if is_error {
        "red"
    } else if certification.conformance_level == "HL3" {
        "brightgreen"
    } else if certification.conformance_level == "HL2" {
        "green"
    } else if certification.conformance_level == "HL1" {
        "yellow"
    } else {
        "red"
    };

    let cap_suffix = if certification.caps.is_empty() {
        "0 caps".to_string()
    } else {
        format!("{} caps", certification.caps.len())
    };

    CertificationBadge {
        schema_version: 1,
        label: "jankurai".to_string(),
        message: format!(
            "{} · score {} · {}",
            certification.conformance_level, certification.score, cap_suffix
        ),
        color: color.to_string(),
        is_error,
        named_logo: "github".to_string(),
    }
}

pub fn build_release_attestation(
    repo: &Path,
    certification: &Certification,
    certification_path: Option<&str>,
    badge_path: Option<&str>,
    signing_key_env: &str,
) -> Result<ReleaseAttestation> {
    let certification_digest = sha256_serialized_hex(certification)?;
    let subject = AttestationSubject {
        name: "jankurai-certification".to_string(),
        digest: format!("sha256:{certification_digest}"),
    };

    let predicate = AttestationPredicate {
        certification_path: certification_path.unwrap_or("stdout").to_string(),
        badge_path: badge_path.map(ToString::to_string),
        standard_version: certification.standard_version.clone(),
        auditor_version: certification.auditor_version.clone(),
        paper_edition: certification.paper_edition.clone(),
        target_stack_id: certification.target_stack_id.clone(),
        score: certification.score,
        conformance_level: certification.conformance_level.clone(),
        governance_decision: certification.governance_decision.clone(),
        build_url: certification.provenance.build_url.clone(),
    };

    let mut material_candidates = vec![
        certification_path.map(ToString::to_string),
        badge_path.map(ToString::to_string),
        Some("agent/repo-score.json".to_string()),
        Some("agent/repo-score.md".to_string()),
        Some("target/jankurai/p12-benchmark-report.json".to_string()),
        Some("target/jankurai/p12-benchmark-report.md".to_string()),
        Some("target/jankurai/p12-governance-policy.json".to_string()),
        Some("target/jankurai/p12-governance-policy.md".to_string()),
        Some("target/jankurai/p12-certification-proof.json".to_string()),
        Some("target/jankurai/p12-certification-proof.md".to_string()),
        Some("target/jankurai/security/evidence.json".to_string()),
        Some("target/jankurai/security-evidence.json".to_string()),
    ];

    material_candidates.extend(
        certification
            .proof_receipt_index
            .iter()
            .cloned()
            .map(Some),
    );
    material_candidates.extend(
        certification
            .security_receipt_index
            .iter()
            .cloned()
            .map(Some),
    );
    material_candidates.extend(
        certification
            .ux_receipt_index
            .iter()
            .cloned()
            .map(Some),
    );
    material_candidates.extend(
        certification
            .contract_db_receipt_index
            .iter()
            .cloned()
            .map(Some),
    );

    let materials = collect_materials(repo, material_candidates)?;

    let payload = json!({
        "subject": subject,
        "predicate": predicate,
        "materials": materials,
    });
    let payload_bytes = serde_json::to_vec(&payload)?;
    let signed_payload_sha256 = sha256_bytes_hex(&payload_bytes);

    let (signature, unsigned_reason) = match env::var(signing_key_env) {
        Ok(key) if !key.is_empty() => (
            Some(SignatureEnvelope {
                algorithm: "HMAC-SHA256".to_string(),
                key_id: format!("env:{signing_key_env}"),
                signed_payload_sha256,
                signature: hmac_sha256_hex(key.as_bytes(), &payload_bytes),
            }),
            None,
        ),
        _ => (
            None,
            Some(format!(
                "unsigned local attestation; set {signing_key_env} to emit an HMAC-SHA256 signature"
            )),
        ),
    };

    Ok(ReleaseAttestation {
        schema_version: certification.schema_version.clone(),
        generated_at: now_string(),
        statement_type: "https://jankurai.dev/attestations/release-evidence/v1".to_string(),
        predicate_type: "https://jankurai.dev/predicates/certification/v1".to_string(),
        subject,
        predicate,
        materials,
        signature,
        unsigned_reason,
    })
}

fn evaluate_certification(
    certification: &Certification,
    minimum_score: i32,
    required_level: &str,
) -> CertificationDecision {
    let mut reasons = Vec::new();

    if certification.score < minimum_score {
        reasons.push(format!(
            "score {} is below minimum {}",
            certification.score, minimum_score
        ));
    }

    if conformance_rank(&certification.conformance_level) < conformance_rank(required_level) {
        reasons.push(format!(
            "conformance {} is below required {}",
            certification.conformance_level, required_level
        ));
    }

    if certification.findings_summary.critical > 0 || certification.findings_summary.high > 0 {
        reasons.push(format!(
            "critical/high findings remain: critical={} high={}",
            certification.findings_summary.critical, certification.findings_summary.high
        ));
    }

    if !certification.caps.is_empty() {
        reasons.push(format!(
            "hard caps remain: {}",
            certification.caps.join(", ")
        ));
    }

    if !certification.evidence_status.score_artifact_present {
        reasons.push("score artifact is missing".to_string());
    }
    if !certification.evidence_status.proof_receipts_present {
        reasons.push("proof evidence is missing".to_string());
    }
    if !certification.evidence_status.security_receipts_present {
        reasons.push("security evidence is missing".to_string());
    }
    if !certification.evidence_status.contract_db_receipts_present {
        reasons.push("contract/db evidence is missing".to_string());
    }

    let passed = reasons.is_empty();
    CertificationDecision {
        status: if passed { "pass" } else { "fail" }.to_string(),
        passed,
        minimum_score,
        required_level: required_level.to_string(),
        reasons,
    }
}

fn evidence_status(
    score_artifact_present: bool,
    proof_receipts: &[String],
    security_receipts: &[String],
    ux_receipts: &[String],
    contract_db_receipts: &[String],
    exceptions: &[String],
) -> EvidenceStatus {
    let mut missing = Vec::new();

    if !score_artifact_present {
        missing.push("agent/repo-score.json".to_string());
    }
    if proof_receipts.is_empty() {
        missing.push("proof receipts or lane evidence".to_string());
    }
    if security_receipts.is_empty() {
        missing.push("target/jankurai/security/evidence.json".to_string());
    }
    if contract_db_receipts.is_empty() {
        missing.push("contract/db receipt index".to_string());
    }

    EvidenceStatus {
        score_artifact_present,
        proof_receipts_present: !proof_receipts.is_empty(),
        security_receipts_present: !security_receipts.is_empty(),
        ux_receipts_present: !ux_receipts.is_empty(),
        contract_db_receipts_present: !contract_db_receipts.is_empty(),
        exceptions_present: !exceptions.is_empty(),
        missing_evidence: missing,
    }
}

fn conformance_level(score: i32, caps: &[String], findings: &FindingsSummary) -> String {
    if score >= 93 && caps.is_empty() && findings.high == 0 && findings.critical == 0 {
        "HL3".to_string()
    } else if score >= 85 && findings.critical == 0 {
        "HL2".to_string()
    } else if score > 0 {
        "HL1".to_string()
    } else {
        "HL0".to_string()
    }
}

fn conformance_rank(level: &str) -> i32 {
    match level.to_ascii_uppercase().as_str() {
        "HL3" => 3,
        "HL2" => 2,
        "HL1" => 1,
        _ => 0,
    }
}

fn collect_receipts(root: &Path) -> Result<Vec<String>> {
    let mut paths = Vec::new();

    for candidate in [
        "target/jankurai/p10-cell-registry-lane.json",
        "target/jankurai/p10-audit-log-proof.json",
        "target/jankurai/p12-certification-proof.json",
        "target/jankurai/evidence-index.json",
    ] {
        if root.join(candidate).exists() {
            paths.push(candidate.to_string());
        }
    }

    paths.extend(collect_matching_json(root, "target/jankurai", |name| {
        name.contains("proof") && name.ends_with(".json")
    })?);
    paths.extend(collect_matching_json_recursive(
        root,
        "target/jankurai/proof-receipts",
        |name| name.ends_with(".json"),
    )?);

    Ok(dedup(paths))
}

fn collect_matching_json<F>(root: &Path, rel_dir: &str, predicate: F) -> Result<Vec<String>>
where
    F: Fn(&str) -> bool,
{
    let mut paths = Vec::new();
    let dir = root.join(rel_dir);
    if !dir.exists() {
        return Ok(paths);
    }

    for entry in fs::read_dir(&dir)? {
        let entry = entry?;
        let file_name = entry.file_name();
        let file_name = file_name.to_string_lossy();
        if predicate(&file_name) {
            paths.push(format!("{rel_dir}/{file_name}"));
        }
    }

    Ok(dedup(paths))
}

fn collect_matching_json_recursive<F>(root: &Path, rel_dir: &str, predicate: F) -> Result<Vec<String>>
where
    F: Fn(&str) -> bool + Copy,
{
    let mut paths = Vec::new();
    let dir = root.join(rel_dir);
    if !dir.exists() {
        return Ok(paths);
    }

    for entry in fs::read_dir(&dir)? {
        let entry = entry?;
        let path = entry.path();
        let file_name = entry.file_name().to_string_lossy().to_string();
        if path.is_dir() {
            let nested = format!("{rel_dir}/{file_name}");
            paths.extend(collect_matching_json_recursive(root, &nested, predicate)?);
        } else if predicate(&file_name) {
            paths.push(format!("{rel_dir}/{file_name}"));
        }
    }

    Ok(dedup(paths))
}

fn collect_materials(
    repo: &Path,
    candidates: Vec<Option<String>>,
) -> Result<Vec<AttestationMaterial>> {
    let mut materials = Vec::new();

    for candidate in dedup(candidates.into_iter().flatten().collect()) {
        let candidate_path = Path::new(&candidate);
        let full_path = if candidate_path.is_absolute() {
            candidate_path.to_path_buf()
        } else {
            repo.join(candidate_path)
        };

        if !full_path.exists() || full_path.is_dir() {
            continue;
        }

        let bytes = fs::read(&full_path)
            .with_context(|| format!("read attestation material {}", full_path.display()))?;

        materials.push(AttestationMaterial {
            path: candidate,
            kind: material_kind(&full_path),
            sha256: sha256_bytes_hex(&bytes),
        });
    }

    materials.sort_by(|left, right| left.path.cmp(&right.path));
    Ok(materials)
}

fn material_kind(path: &Path) -> String {
    let text = path.to_string_lossy();
    if text.contains("repo-score") {
        "repo-score".to_string()
    } else if text.contains("benchmark") {
        "benchmark".to_string()
    } else if text.contains("governance") {
        "governance-policy".to_string()
    } else if text.contains("security") {
        "security-evidence".to_string()
    } else if text.contains("proof") || text.contains("evidence-index") {
        "proof-evidence".to_string()
    } else if text.contains("openapi") || text.contains("/db/") {
        "contract-db-evidence".to_string()
    } else if text.contains("badge") {
        "badge".to_string()
    } else if text.contains("certification") {
        "certification".to_string()
    } else {
        "material".to_string()
    }
}

fn sha256_serialized_hex<T: Serialize>(value: &T) -> Result<String> {
    let bytes = serde_json::to_vec(value)?;
    Ok(sha256_bytes_hex(&bytes))
}

fn sha256_bytes_hex(bytes: &[u8]) -> String {
    let digest = Sha256::digest(bytes);
    to_hex(&digest)
}

fn hmac_sha256_hex(key: &[u8], data: &[u8]) -> String {
    const BLOCK_SIZE: usize = 64;

    let mut key_block = if key.len() > BLOCK_SIZE {
        Sha256::digest(key).to_vec()
    } else {
        key.to_vec()
    };
    key_block.resize(BLOCK_SIZE, 0);

    let mut outer_pad = [0x5c_u8; BLOCK_SIZE];
    let mut inner_pad = [0x36_u8; BLOCK_SIZE];

    for (idx, byte) in key_block.iter().enumerate() {
        outer_pad[idx] ^= byte;
        inner_pad[idx] ^= byte;
    }

    let mut inner = Sha256::new();
    inner.update(inner_pad);
    inner.update(data);
    let inner_digest = inner.finalize();

    let mut outer = Sha256::new();
    outer.update(outer_pad);
    outer.update(inner_digest);
    to_hex(&outer.finalize())
}

fn to_hex(bytes: &[u8]) -> String {
    const HEX: &[u8; 16] = b"0123456789abcdef";
    let mut out = String::with_capacity(bytes.len() * 2);
    for byte in bytes {
        out.push(HEX[(byte >> 4) as usize] as char);
        out.push(HEX[(byte & 0x0f) as usize] as char);
    }
    out
}

write("crates/jankurai/tests/phase_12_public_evidence.rs", r'''
use jankurai::commands::{bench, certify, govern};
use jankurai::validation::{self, ArtifactSchema};
use std::path::PathBuf;

fn repo_root() -> PathBuf {
    PathBuf::from(env!("CARGO_MANIFEST_DIR")).join("../..")
}

#[test]
fn phase12_public_evidence_artifacts_are_schema_valid() {
    let repo = repo_root();

    let suite = bench::build_benchmark_suite(&repo).expect("benchmark suite");
    validation::validate_serializable(&repo, ArtifactSchema::BenchmarkSuite, &suite)
        .expect("benchmark suite schema");

    let benchmark_report =
        bench::build_benchmark_report(&repo, &suite).expect("benchmark report");
    validation::validate_serializable(&repo, ArtifactSchema::BenchmarkReport, &benchmark_report)
        .expect("benchmark report schema");

    let policy = govern::build_governance_policy(&repo).expect("governance policy");
    validation::validate_serializable(&repo, ArtifactSchema::GovernancePolicy, &policy)
        .expect("governance policy schema");

    let certification = certify::build_certification(&repo).expect("certification");
    validation::validate_serializable(&repo, ArtifactSchema::Certification, &certification)
        .expect("certification schema");

    let badge = certify::build_badge(&certification);
    validation::validate_serializable(&repo, ArtifactSchema::CertificationBadge, &badge)
        .expect("badge schema");

    let attestation = certify::build_release_attestation(
        &repo,
        &certification,
        Some("target/jankurai/phase12-public/certification.json"),
        Some("target/jankurai/phase12-public/badge.json"),
        "JANKURAI_ATTESTATION_KEY",
    )
    .expect("release attestation");

    validation::validate_serializable(&repo, ArtifactSchema::ReleaseAttestation, &attestation)
        .expect("release attestation schema");

    assert!(policy.certification_gate.minimum_score >= 85);
    assert!(!badge.message.is_empty());
    assert_eq!(attestation.subject.name, "jankurai-certification");
}
''')
