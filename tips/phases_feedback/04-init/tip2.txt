## Phase name

**Phase 04: Init Profiles And Golden Repos**

I’d treat this phase as **not fully closed yet**, even though the phase file currently says “complete.” The phase objective is to turn `jankurai init` into a profile-driven repo generator, with a robust generator contract, one excellent default profile, and golden repo fixtures that prove generated repos are audit-ready; the same phase file still lists follow-on gaps around additional profiles, deeper merge policy, and optional repo-local profile JSON loading. ([GitHub][1])

This is the right phase to finish because it directly advances the moonshot loop: `intent -> bounded agents -> proof lanes -> evidence -> expired exceptions -> reusable primitives`. The moonshot says the repo should tell an agent what it owns, how to prove a change, where generated truth lives, and when exceptions expire. ([GitHub][2]) The standard also says HL3+ repos need root `AGENTS.md`, `agent/JANKURAI_STANDARD.md`, owner/test maps, generated-zone manifest, standard version, score artifacts, one-command validation, and CI audit. ([GitHub][3])

The current repo already frames Jankurai as the control plane for agent-native repositories, with ownership, proof, generated zones, and repair evidence made machine-readable. ([GitHub][4]) The patch below finishes Phase 04 by making `init` behave like a real creation layer: all public profiles are supported, profile manifests are strict, local profile files are supported, plan/apply semantics match, generated adapters can be safely refreshed, user-owned files are not overwritten, golden tests cover idempotency and merge behavior, and docs are updated to match the actual product surface.

````bash
pub fn adapter_plan(repo: &Path, ide: &str) -> Vec<AdapterAction> {
    selected_adapter_paths(ide)
        .into_iter()
        .map(|path| AdapterAction {
            path: path.into(),
            action: if repo.join(path).exists() {
                "keep-existing".into()
            } else {
                "create".into()
            },
        })
        .collect()
}

pub fn write_adapters(repo: &Path, ide: &str, force_generated: bool) -> Result<Vec<AdapterAction>> {
    let selected = selected_adapter_paths(ide);
    let mut actions = Vec::new();

    for template in crate::init::templates::TEMPLATES
        .iter()
        .filter(|template| selected.contains(&template.path))
    {
        let path = repo.join(template.path);
        if let Some(parent) = path.parent() {
            fs::create_dir_all(parent).with_context(|| format!("create {}", parent.display()))?;
        }

        if path.exists() {
            let text = fs::read_to_string(&path).unwrap_or_default();
            if force_generated && text.contains(GENERATED_MARKER) {
                fs::write(&path, template.body).with_context(|| format!("write {}", path.display()))?;
                actions.push(AdapterAction {
                    path: template.path.into(),
                    action: "overwrote-generated".into(),
                });
            } else {
                actions.push(AdapterAction {
                    path: template.path.into(),
                    action: "keep-existing".into(),
                });
            }
            continue;
        }

        fs::write(&path, template.body).with_context(|| format!("write {}", path.display()))?;
        actions.push(AdapterAction {
            path: template.path.into(),
            action: "created".into(),
        });
    }

    Ok(actions)
}

pub fn verify_adapters(repo: &Path) -> Result<Vec<AdapterFailure>> {
    let mut failures = Vec::new();

    for path in ADAPTER_PATHS {
        let full = repo.join(path);
        if !full.exists() {
            continue;
        }

        let text = fs::read_to_string(&full).with_context(|| format!("read {}", full.display()))?;
        if !text.contains("AGENTS.md")
            || !text.contains("agent/JANKURAI_STANDARD.md")
            || !text.contains(master_plan_pointer())
            || !text.contains(planner_protocol_pointer())
            || !text.contains("tips/phases/00-phase-index.md")
            || !text.contains("tips/phases/logs/")
        {
            failures.push(AdapterFailure {
                path: (*path).into(),
                problem: "adapter lacks canonical AGENTS.md, standard, MASTER_PLAN, planner protocol, phase index, and phase log pointers".into(),
            });
        }

        let lower = text.to_ascii_lowercase();
        if lower.contains("ignore agents.md")
            || lower.contains("ignore `agents.md`")
            || lower.contains("do not read agents.md")
            || lower.contains("do not use agent/jankurai_standard.md")
            || lower.contains("do not use agent/master_plan.md")
        {
            failures.push(AdapterFailure {
                path: (*path).into(),
                problem: "adapter contains contradictory command/path policy".into(),
            });
        }

        if text.lines().count() > 80 {
            failures.push(AdapterFailure {
                path: (*path).into(),
                problem: "adapter is too long; adapters must not duplicate the full standard".into(),
            });
        }
    }

    Ok(failures)
}

fn selected_adapter_paths(ide: &str) -> Vec<&'static str> {
    let mut paths = Vec::new();

    for token in ide.split(',').map(str::trim) {
        match token {
            "all" => {
                paths.extend_from_slice(ADAPTER_PATHS);
                break;
            }
            "claude" => {
                paths.push("CLAUDE.md");
                paths.push(".claude/skills/jankurai/SKILL.md");
            }
            "cursor" => paths.push(".cursor/rules/jankurai.mdc"),
            "copilot" => {
                paths.push(".github/copilot-instructions.md");
                paths.push(".github/instructions/jankurai.instructions.md");
                paths.push(".github/instructions/jankurai-rust.instructions.md");
                paths.push(".github/instructions/jankurai-web.instructions.md");
                paths.push(".github/instructions/jankurai-python-ai.instructions.md");
            }
            "gemini" => paths.push("GEMINI.md"),
            "antigravity" => {
                paths.push(".agents/agents.md");
                paths.push(".agents/skills/jankurai/SKILL.md");
                paths.push(".agents/workflows/jankurai-audit.md");
            }
            "aider" => paths.push(".aider.conf.yml"),
            _ => {}
        }
    }

    paths.sort_unstable();
    paths.dedup();
    paths
}

        .with_context(|| format!("read init profile manifest {}", path.display()))?;

fn load_profile_from_str(repo: &Path, json_str: &str) -> Result<ProfileManifest> {
    let value: serde_json::Value = serde_json::from_str(json_str)?;
    validation::validate_value(repo, ArtifactSchema::InitProfile, &value)?;
    let file: ProfileManifestFile = serde_json::from_value(value)?;
    let manifest: ProfileManifest = file.into();
    manifest.validate_contract()?;
    Ok(manifest)
}

fn normalize_profile_id(profile: &str) -> &str {
    match profile {
        "rust-ts-postgres"
        | "rust-ts-vite-react-postgres"
        | "rust-ts-vite-react-postgres-bounded-python" => "rust-ts-postgres",
        "rust" | "rust-postgres" => "rust-api",
        "react-app" | "react-vite" => "react-web",
        "saas" | "fullstack" => "b2b-saas",
        "ai" | "ai-service" => "ai-product",
        "regulated" | "regulated-fintech" | "healthcare-lite" => "regulated-saas",
        "migration" | "legacy-migration" => "migration-target",
        other => other,
    }
}

fn ensure_non_empty(field: &str, value: &str) -> Result<()> {
    if value.trim().is_empty() {
        bail!("init profile field `{field}` may not be empty");
    }
    Ok(())
}

fn ensure_non_empty_list(field: &str, values: &[String]) -> Result<()> {
    if values.is_empty() {
        bail!("init profile field `{field}` must contain at least one item");
    }
    Ok(())
}

fn validate_string_list(field: &str, values: &[String], paths: bool) -> Result<()> {
    let mut seen = BTreeSet::new();
    for value in values {
        if value.trim().is_empty() {
            bail!("init profile field `{field}` contains an empty value");
        }
        if !seen.insert(value.as_str()) {
            bail!("init profile field `{field}` contains duplicate `{value}`");
        }
        if paths && !is_safe_relative_path(value) {
            bail!("init profile field `{field}` contains unsafe relative path `{value}`");
        }
    }
    Ok(())
}

fn is_safe_relative_path(path: &str) -> bool {
    if path.is_empty() || path.starts_with('/') || path.contains('\\') {
        return false;
    }

    let mut saw_normal = false;
    for component in Path::new(path).components() {
        match component {
            Component::Normal(_) => saw_normal = true,
            Component::CurDir | Component::ParentDir | Component::RootDir | Component::Prefix(_) => {
                return false;
            }
        }
    }

    saw_normal
}

        (
            super::profiles::resolve_profile(repo, profile)?,
            "bundled".to_string(),
            None,
        )
    };

    let actions = actions_for_manifest(repo, &profile_manifest, force_generated_adapters)?;

    let mut warnings = Vec::new();
    if !existing.is_empty() {
        warnings.push(format!("existing control files left intact: {}", existing.join(", ")));
    }
    if profile_file.is_some() {
        warnings.push("profile file mode still uses registered templates only; unknown generated paths fail at plan time".into());
    }

    Ok(InitPlan {
        profile: profile_manifest.id.clone(),
        profile_source,
        profile_file: profile_file_display,
        profile_manifest,
        ide: ide.into(),
        mode: mode.into(),
        ci: ci.into(),
        issue_backend: issue_backend.into(),
        ux_qa,
        force_generated_adapters,
        package_manager: super::package_manager::detect_package_manager(repo).to_string(),
        detected,
        actions,
        warnings,
    })
}

pub fn actions_for_manifest(
    repo: &Path,
    manifest: &super::profiles::ProfileManifest,
    force_generated_adapters: bool,
) -> Result<Vec<PlannedAction>> {
    let mut paths = manifest.generated_paths.clone();
    paths.sort();

    let mut actions = Vec::new();
    for path in &paths {
        if super::templates::template_for_path(path).is_none() {
            bail!(
                "profile `{}` declares `{path}` but no init template is registered (see init/templates.rs)",
                manifest.id
            );
        }
        actions.push(classify_path(repo, path, force_generated_adapters)?);
    }

    Ok(actions)
}

fn classify_path(repo: &Path, rel: &str, force_generated_adapters: bool) -> Result<PlannedAction> {
    let path = repo.join(rel);
    let generated_adapter = super::adapters::is_adapter_path(rel);

    if !path.exists() {
        return Ok(PlannedAction {
            path: rel.into(),
            action: "create".into(),
            reason: "path is absent".into(),
            refreshable: false,
            generated_adapter,
        });
    }

    let text = std::fs::read_to_string(&path).unwrap_or_default();

    if generated_adapter && text.contains(super::adapters::GENERATED_MARKER) {
        if force_generated_adapters {
            return Ok(PlannedAction {
                path: rel.into(),
                action: "refresh-generated-adapter".into(),
                reason: "existing adapter has the generated refresh marker and refresh was requested".into(),
                refreshable: true,
                generated_adapter,
            });
        }

        return Ok(PlannedAction {
            path: rel.into(),
            action: "keep-existing".into(),
            reason: "existing adapter has the generated refresh marker; rerun with --force-generated-adapters to refresh".into(),
            refreshable: true,
            generated_adapter,
        });
    }

    if should_mark_for_merge(rel) {
        if text.contains(super::merge::MERGE_MARKER_HEADER) {
            return Ok(PlannedAction {
                path: rel.into(),
                action: "keep-existing".into(),
                reason: "merge marker already present; not appending a duplicate marker".into(),
                refreshable: false,
                generated_adapter,
            });
        }

        if is_jankurai_controlled(&text) {
            return Ok(PlannedAction {
                path: rel.into(),
                action: "keep-existing".into(),
                reason: "existing file already points to the jankurai standard".into(),
                refreshable: false,
                generated_adapter,
            });
        }

        return Ok(PlannedAction {
            path: rel.into(),
            action: "merge-marker".into(),
            reason: "existing user-owned control file needs manual merge review".into(),
            refreshable: false,
            generated_adapter,
        });
    }

    Ok(PlannedAction {
        path: rel.into(),
        action: "keep-existing".into(),
        reason: "existing user-owned file is never overwritten by init".into(),
        refreshable: false,
        generated_adapter,
    })
}

pub fn should_mark_for_merge(path: &str) -> bool {
    matches!(path, "AGENTS.md" | "agent/JANKURAI_STANDARD.md")
}

pub fn is_jankurai_controlled(text: &str) -> bool {
    text.contains("agent/JANKURAI_STANDARD.md")
        || text.contains("jankurai Standard Agent Bootstrap")
        || text.contains("jankurai Agent-Native Repository Standard")
        || text.contains("Read `AGENTS.md` first")
        || (text.contains("Standard version:") && text.to_ascii_lowercase().contains("jankurai"))
}

    let mut out = String::new();
    use std::fmt::Write;

    let _ = writeln!(out, "Jankurai Init Plan");
    let _ = writeln!(out);
    let _ = writeln!(out, "- profile: `{}`", plan.profile);
    let _ = writeln!(out, "- profile source: `{}`", plan.profile_source);
    if let Some(profile_file) = &plan.profile_file {
        let _ = writeln!(out, "- profile file: `{}`", profile_file);
    }
    let _ = writeln!(out, "- profile id: `{}`", plan.profile_manifest.id);
    let _ = writeln!(
        out,
        "- profile display: `{}`",
        plan.profile_manifest.display_name
    );
    let _ = writeln!(
        out,
        "- target stack: `{}`",
        plan.profile_manifest.target_stack_id
    );
    let _ = writeln!(out, "- ide: `{}`", plan.ide);
    let _ = writeln!(out, "- mode: `{}`", plan.mode);
    let _ = writeln!(out, "- ci: `{}`", plan.ci);
    let _ = writeln!(out, "- issue backend: `{}`", plan.issue_backend);
    let _ = writeln!(out, "- ux qa: `{}`", plan.ux_qa);
    let _ = writeln!(
        out,
        "- force generated adapters: `{}`",
        plan.force_generated_adapters
    );
    let _ = writeln!(out, "- package manager: `{}`", plan.package_manager);

    if !plan.profile_manifest.required_lanes.is_empty() {
        let _ = writeln!(
            out,
            "- required lanes: `{}`",
            plan.profile_manifest.required_lanes.join(", ")
        );
    }

    if !plan.profile_manifest.optional_lanes.is_empty() {
        let _ = writeln!(
            out,
            "- optional lanes: `{}`",
            plan.profile_manifest.optional_lanes.join(", ")
        );
    }

    if !plan.detected.is_empty() {
        let _ = writeln!(out, "- detected: `{}`", plan.detected.join(", "));
    }

    if !plan.profile_manifest.validation_commands.is_empty() {
        let _ = writeln!(
            out,
            "- validation: `{}`",
            plan.profile_manifest.validation_commands.join(" | ")
        );
    }

    for warning in &plan.warnings {
        let _ = writeln!(out, "- warning: `{warning}`");
    }

    let _ = writeln!(out);
    let _ = writeln!(out, "Planned file actions:");
    for action in &plan.actions {
        let _ = writeln!(
            out,
            " {} {} — {}",
            action.action, action.path, action.reason
        );
    }

    out
}

fn apply_templates(
    repo: &Path,
    plan: &crate::init::plan::InitPlan,
) -> Result<Vec<InitAction>> {
    let mut actions = Vec::new();

    for planned in &plan.actions {
        match planned.action.as_str() {
            "create" => {
                write_template(repo, &planned.path)?;
            }
            "refresh-generated-adapter" => {
                write_template(repo, &planned.path)?;
            }
            "merge-marker" => {
                append_merge_marker(repo, &planned.path)?;
            }
            "keep-existing" => {}
            other => bail!("unknown init planned action `{other}` for `{}`", planned.path),
        }

        actions.push(InitAction {
            path: planned.path.clone(),
            action: planned.action.clone(),
            reason: planned.reason.clone(),
        });
    }

    Ok(actions)
}

fn write_template(repo: &Path, rel: &str) -> Result<()> {
    let template = crate::init::templates::template_for_path(rel)
        .with_context(|| format!("no template registered for profile path `{rel}`"))?;
    let path = repo.join(rel);

    if let Some(parent) = path.parent() {
        fs::create_dir_all(parent).with_context(|| format!("create {}", parent.display()))?;
    }

    fs::write(&path, template.body).with_context(|| format!("write {}", path.display()))?;
    Ok(())
}

fn append_merge_marker(repo: &Path, rel: &str) -> Result<()> {
    let path = repo.join(rel);
    let text = fs::read_to_string(&path).unwrap_or_default();

    if text.contains(crate::init::merge::MERGE_MARKER_HEADER) {
        return Ok(());
    }

    fs::write(&path, format!("{text}{}", crate::init::merge::merge_marker(rel)))
        .with_context(|| format!("append merge marker to {}", path.display()))?;
    Ok(())
}

fn print_diff(repo: &Path, plan: &crate::init::plan::InitPlan) -> Result<()> {
    for action in &plan.actions {
        match action.action.as_str() {
            "create" => {
                let template = crate::init::templates::template_for_path(&action.path)
                    .with_context(|| format!("no template registered for `{}`", action.path))?;
                println!("--- /dev/null");
                println!("+++ {}", action.path);
                for line in template.body.lines() {
                    println!("+{line}");
                }
            }
            "refresh-generated-adapter" => {
                let template = crate::init::templates::template_for_path(&action.path)
                    .with_context(|| format!("no template registered for `{}`", action.path))?;
                println!("--- {} (existing generated adapter)", action.path);
                println!("+++ {} (refreshed generated adapter)", action.path);
                for line in template.body.lines() {
                    println!("+{line}");
                }
            }
            "merge-marker" => {
                println!("--- {} (existing user-owned file)", action.path);
                println!("+++ {} (append jankurai merge marker)", action.path);
                for line in crate::init::merge::merge_marker(&action.path).lines() {
                    println!("+{line}");
                }
            }
            "keep-existing" => {
                let exists = repo.join(&action.path).exists();
                println!(
                    "--- {} {}",
                    action.path,
                    if exists { "exists; no overwrite" } else { "not changed" }
                );
                println!("reason: {}", action.reason);
            }
            other => bail!("unknown init planned action `{other}` for `{}`", action.path),
        }
    }

    Ok(())
}

#[derive(Debug, Serialize)]
struct InitAction {
    path: String,
    action: String,
    reason: String,
}

fn write_receipt(
    repo: &Path,
    action: &str,
    plan: &crate::init::plan::InitPlan,
    actions: &[InitAction],
) -> Result<()> {
    let dir = repo.join("target/jankurai/receipts");
    fs::create_dir_all(&dir)?;

    let now = SystemTime::now()
        .duration_since(UNIX_EPOCH)
        .unwrap_or_default()
        .as_secs();

    let path = dir.join(format!("{action}-{now}.json"));
    let payload = serde_json::json!({
        "action": action,
        "profile_id": plan.profile_manifest.id.as_str(),
        "profile_source": plan.profile_source.as_str(),
        "created_at": now,
        "actions": actions,
    });

    validation::validate_value(repo, ArtifactSchema::InitReceipt, &payload)?;
    fs::write(path, serde_json::to_string_pretty(&payload)?)?;
    Ok(())
}
````

`jankurai init` is now a profile-driven generator, not only a control-file installer.

## Recommended flow

Dry-run first:

```bash
jankurai init --profile rust-ts-postgres --dry-run --plan-json target/jankurai/init-plan.json
```

Review the plan:

```bash
cat target/jankurai/init-plan.json
```

```bash
jankurai init --profile rust-ts-postgres --yes
jankurai doctor --fail-on high
jankurai audit . --json agent/repo-score.json --md agent/repo-score.md
```

## Safe rerun behavior

`init` is intentionally conservative.

* Missing generated paths are created.
* Existing user-owned files are kept.
* Existing `AGENTS.md` and `agent/JANKURAI_STANDARD.md` get a merge marker when they do not already point to the Jankurai standard.
* Existing generated adapters refresh only when they contain the generated marker and you pass `--force-generated-adapters`.
* Unknown template paths fail during plan generation.

## Profile file mode

Use a repo-local or external profile file when an adopter needs a controlled custom scaffold:

```bash
jankurai init \
  --profile-file agent/init-profile.json \
  --dry-run \
  --plan-json target/jankurai/custom-init-plan.json
```

Profile-file mode still uses registered templates only. This prevents a profile from writing arbitrary undeclared content.

## Generated versus source files

Generated outputs must be declared in `agent/generated-zones.toml`.

Do not edit generated output by hand. Change the source contract, regenerate, and leave evidence under `target/jankurai/`.

## Which profile to choose

* `rust-api`: Rust API, contracts, DB policy, security lane, no web UX lane.
* `react-web`: TypeScript/React/Vite web slot, UX policy, contracts, no DB policy.
* `rust-ts-postgres`: default fullstack reference with Rust, web, Postgres, contracts, bounded AI/data slot, UX, and security.
* `b2b-saas`: fullstack plus auth, org, admin, and compliance-evidence docs.
* `ai-product`: bounded Python AI/data/eval policy with contracts and Rust boundary.
* `regulated-saas`: stricter privacy, retention, backup/restore, incident, data-classification, and release lanes.
* `migration-target`: legacy containment, strangler map, migration ops, contracts, and DB policy.
