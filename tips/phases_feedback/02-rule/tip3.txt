## Phase: `02-rule-engine-semantic-oracle.md`

The right pending work to finish here is **turning the current Phase 02 partial implementation into a first-class semantic boundary oracle**: a rule-registry-backed analyzer that emits actionable, owner/lane/evidence-rich findings instead of relying only on loose text scans.

Why this is the right slice: `moonshot.md` says the system must make agent-native engineering “boring” through explicit intent, bounded agents, proof lanes, durable evidence, expiring exceptions, and reusable primitives. It also rejects ambiguous ownership, hidden fallbacks, handwritten contract mirrors, generated hand edits, and proof that does not cover changed behavior. ([GitHub][1]) Phase 02’s own objective is to move from heuristic scanning to a versioned rule engine with semantic understanding, using an architecture of `inventory -> analyzers -> rule registry -> findings -> caps -> score -> reports`; its receipt says rule metadata exists but the analyzer split and AST/import pilot are still follow-on work. ([GitHub][2])

The repo already has the ingredients: `Finding` has fields for `rule_id`, `tlr`, `lane`, `docs_url`, `owner`, `line`, `matched_term`, and `reason`; `FindingBuilder` enriches findings from the rule registry; and `agent/boundaries.toml` already declares Rust, TypeScript, Python, queue, streaming, and DB boundary policy. ([GitHub][3]) The gap is that several of the most important moonshot boundary checks still live as scan-like helpers, especially DB wrong-layer and streaming-runtime checks, rather than as semantic analyzer output with registry validation and tests. ([GitHub][4])

Below is the **single patch I would land** for Phase 02.

# Phase 02 completion patch: Semantic Boundary Oracle

## What this finishes

This patch completes Phase 02 by adding:

1. A policy-aware semantic oracle under `audit/analyzers/semantic_oracle.rs`.
2. Extended boundary manifest parsing for `[rust]`, `[typescript]`, and `[python]`.
3. First-class findings for:

   * Rust domain impurity.
   * TypeScript/web wrong-layer imports and handwritten client/DTO mirrors.
   * Python durable product-truth leakage.
   * Direct DB usage outside DB/adapters/migrations.
   * Streaming-runtime drift with expiring exception validation.
4. Registry-backed rule metadata for the new semantic findings.
5. A `FindingBuilder::add_detailed` method so semantic findings preserve `matched_term`, `reason`, line number, lane, owner, docs URL, rerun command, and evidence kind.
6. Tests that fail when Phase 02 regressions happen:

   * Unknown `rule_id`.
   * Missing rule metadata.
   * Boundary analyzer emits non-actionable high/critical findings.
   * Semantic boundary hits do not include matched term/reason/line.

This keeps the repo aligned with the canonical audit loop described in the README: `intent -> bounded agents -> proof lanes -> evidence -> expired exceptions -> reusable primitives`, with the audit lane remaining `cargo run -p jankurai -- . --json agent/repo-score.json --md agent/repo-score.md`. ([GitHub][5])

#[derive(Debug, Clone)]
struct ManifestState {
    manifest: BoundaryManifest,
    present: bool,
    parse_error: Option<String>,

pub fn findings(ctx: &AuditContext) -> Vec<SemanticFinding> {
    let state = load_boundary_manifest(ctx);
    let mut out = Vec::new();

    if !state.present {
        out.push(SemanticFinding::new(
            "HLT-024-BOUNDARY-MANIFEST-GAP",
            BOUNDARIES_MANIFEST,
            None,
            Some("missing-boundaries-manifest".to_string()),
            "Boundary oracle manifest is missing.",
            "Phase 02 requires explicit boundary policy so semantic findings can map code to ownership, proof lanes, and exceptions.",
            "Create agent/boundaries.toml with [rust], [typescript], [python], [queues], and [db] sections. Use the current stack policy as the source of truth.",
            vec![
                "expected manifest: agent/boundaries.toml".to_string(),
                "semantic oracle fell back to conservative defaults".to_string(),
            ],
            0.99,
        ));
    }

    if let Some(error) = &state.parse_error {
        out.push(SemanticFinding::new(
            "HLT-024-BOUNDARY-MANIFEST-GAP",
            BOUNDARIES_MANIFEST,
            Some(1),
            Some("toml-parse-error".to_string()),
            "Boundary oracle manifest is not valid TOML.",
            "The semantic oracle cannot safely apply trusted boundary policy while the manifest fails to parse.",
            "Fix agent/boundaries.toml TOML syntax, then rerun `just score`.",
            vec![
                format!("parse error: {error}"),
                "trusted policy must win over untrusted local heuristics".to_string(),
            ],
            0.99,
        ));
    }

    out.extend(rust_domain_impurity(ctx, &state.manifest));
    out.extend(typescript_web_wrong_layer(ctx, &state.manifest));
    out.extend(python_product_truth_leakage(ctx, &state.manifest));
    out.extend(db_wrong_layer_access(ctx, &state.manifest));
    out.extend(streaming_runtime_drift(ctx, &state.manifest));

    out.sort_by(|a, b| {
        a.path
            .cmp(&b.path)
            .then(a.line.cmp(&b.line))
            .then(a.rule_id.cmp(b.rule_id))
            .then(a.matched_term.cmp(&b.matched_term))
    });
    out.dedup_by(|a, b| {
        a.rule_id == b.rule_id
            && a.path == b.path
            && a.line == b.line
            && a.matched_term == b.matched_term
    });

    out
}

fn load_boundary_manifest(ctx: &AuditContext) -> ManifestState {
    let path = ctx.root.join(BOUNDARIES_MANIFEST);
    match std::fs::read_to_string(&path) {
        Ok(text) => match manifest::parse(&text) {
            Ok(manifest) => ManifestState {
                manifest,
                present: true,
                parse_error: None,
            },
            Err(error) => ManifestState {
                manifest: BoundaryManifest::default(),
                present: true,
                parse_error: Some(error.to_string()),
            },
        },
        Err(_) => ManifestState {
            manifest: BoundaryManifest::default(),
            present: false,
            parse_error: None,
        },
    }
}

fn rust_domain_impurity(
    ctx: &AuditContext,
    manifest: &BoundaryManifest,
) -> Vec<SemanticFinding> {
    let policy = manifest.rust_policy();
    let domain_paths = effective_strings(&policy.domain_paths, DEFAULT_RUST_DOMAIN_PATHS);
    let forbidden_imports = effective_strings(
        &policy.forbidden_domain_imports,
        DEFAULT_RUST_FORBIDDEN_IMPORTS,
    );

    let mut out = Vec::new();

    for file in ctx.scope_files.iter().filter(|file| file.suffix == ".rs") {
        if skip_semantic_file(file) || !matches_any_policy_path(&file.rel_path, &domain_paths) {
            continue;
        }

        let import_edges = import_edges(file);
        if let Some(edge) = import_edges
            .iter()
            .find(|edge| marker_matches_any(&edge.target, &forbidden_imports))
        {
            let marker = matching_marker(&edge.target, &forbidden_imports)
                .unwrap_or_else(|| edge.target.clone());
            out.push(SemanticFinding::new(
                "HLT-022-RUST-DOMAIN-IMPURITY",
                file.rel_path.clone(),
                Some(edge.line),
                Some(marker.clone()),
                "Rust domain code imports side-effectful infrastructure.",
                format!(
                    "Path `{}` matches Rust domain policy and import `{}` violates the pure-domain boundary.",
                    file.rel_path, edge.target
                ),
                "Move filesystem, environment, network, time, DB, queue, logging, and randomness access into an adapter/application layer. Pass deterministic values into the domain as explicit inputs.",
                vec![
                    format!("policy paths: {}", domain_paths.join(", ")),
                    format!("import line {}: {}", edge.line, edge.raw.trim()),
                    format!("matched forbidden import marker: `{marker}`"),
                ],
                0.96,
            ));
            continue;
        }

        if let Some((line, marker, raw)) = first_marker_in_text(&file.text, &forbidden_imports) {
            out.push(SemanticFinding::new(
                "HLT-022-RUST-DOMAIN-IMPURITY",
                file.rel_path.clone(),
                Some(line),
                Some(marker.clone()),
                "Rust domain code contains side-effectful infrastructure access.",
                format!(
                    "Path `{}` matches Rust domain policy and contains forbidden marker `{}`.",
                    file.rel_path, marker
                ),
                "Move this side effect behind an explicit adapter or inject the value into the domain. Domain code should be deterministic and free of IO/runtime ownership.",
                vec![
                    format!("policy paths: {}", domain_paths.join(", ")),
                    format!("line {line}: {}", raw.trim()),
                    format!("matched forbidden marker: `{marker}`"),
                ],
                0.92,
            ));
        }
    }

    out
}

fn typescript_web_wrong_layer(
    ctx: &AuditContext,
    manifest: &BoundaryManifest,
) -> Vec<SemanticFinding> {
    let policy = manifest.typescript_policy();
    let web_paths = effective_strings(&policy.web_paths, DEFAULT_TS_WEB_PATHS);
    let forbidden_imports = effective_strings(
        &policy.forbidden_web_imports,
        DEFAULT_TS_FORBIDDEN_IMPORTS,
    );
    let generated_contract_paths = effective_strings(
        &policy.generated_contract_paths,
        &["contracts/generated", "packages/generated", "apps/web/src/generated"],
    );

    let generated_contracts_exist = ctx.scope_files.iter().any(|file| {
        generated_contract_paths
            .iter()
            .any(|path| path_prefix_match(&file.rel_path, path))
    }) || ctx.all_files.iter().any(|file| {
        generated_contract_paths
            .iter()
            .any(|path| path_prefix_match(&file.rel_path, path))
    });

    let mut out = Vec::new();

    for file in ctx.scope_files.iter().filter(|file| {
        matches!(
            file.suffix.as_str(),
            ".ts" | ".tsx" | ".js" | ".jsx" | ".mjs" | ".cjs"
        )
    }) {
        if skip_semantic_file(file) || !matches_any_policy_path(&file.rel_path, &web_paths) {
            continue;
        }

        let import_edges = import_edges(file);
        if let Some(edge) = import_edges
            .iter()
            .find(|edge| marker_matches_any(&edge.target, &forbidden_imports))
        {
            let marker = matching_marker(&edge.target, &forbidden_imports)
                .unwrap_or_else(|| edge.target.clone());
            out.push(SemanticFinding::new(
                "HLT-023-TS-WEB-WRONG-LAYER",
                file.rel_path.clone(),
                Some(edge.line),
                Some(marker.clone()),
                "TypeScript web code imports backend/storage/runtime infrastructure.",
                format!(
                    "Path `{}` matches web policy and import `{}` violates the frontend boundary.",
                    file.rel_path, edge.target
                ),
                "Move DB, queue, filesystem, and backend runtime access into server/adapters. Web code should consume generated contracts or typed API clients only.",
                vec![
                    format!("policy paths: {}", web_paths.join(", ")),
                    format!("import line {}: {}", edge.line, edge.raw.trim()),
                    format!("matched forbidden import marker: `{marker}`"),
                ],
                0.96,
            ));
            continue;
        }

        if let Some((line, marker, raw)) = first_marker_in_text(&file.text, &forbidden_imports) {
            out.push(SemanticFinding::new(
                "HLT-023-TS-WEB-WRONG-LAYER",
                file.rel_path.clone(),
                Some(line),
                Some(marker.clone()),
                "TypeScript web code contains backend/storage/runtime marker.",
                format!(
                    "Path `{}` matches web policy and contains forbidden marker `{}`.",
                    file.rel_path, marker
                ),
                "Replace raw infrastructure access with generated contracts, typed API clients, or a server-side adapter.",
                vec![
                    format!("policy paths: {}", web_paths.join(", ")),
                    format!("line {line}: {}", raw.trim()),
                    format!("matched forbidden marker: `{marker}`"),
                ],
                0.90,
            ));
            continue;
        }

        if generated_contracts_exist {
            if let Some((line, marker, raw)) = handwritten_contract_line(file) {
                out.push(SemanticFinding::new(
                    "HLT-007-GENERATED-CONTRACT-DRIFT",

fn python_product_truth_leakage(

    let queue_policy = manifest.queue_policy();

    let migration_paths = effective_strings(&db_policy.migration_paths, &["db/migrations"]);
    let constraint_paths = effective_strings(&db_policy.constraint_paths, &["db/constraints"]);
    let adapter_paths = effective_strings(
        &queue_policy.adapter_paths,
        &[
            "crates/adapters",
            "packages/adapters",
            "apps/api/src/adapters",
            "apps/server/src/adapters",
        ],
    );

    let markers = strings_from_static(DB_MARKERS);
    let mut out = Vec::new();

    for file in ctx.scope_files.iter().filter(|file| is_code_like(file)) {
        if skip_semantic_file(file)
            || is_db_allowed_path(
                &file.rel_path,
                &root_paths,
                &migration_paths,
                &constraint_paths,
                &adapter_paths,
            )
            || !is_db_risky_layer(&file.rel_path)
        {
            continue;
        }

        let import_edges = import_edges(file);
        if let Some(edge) = import_edges
            .iter()
            .find(|edge| marker_matches_any(&edge.target, &markers))
        {
            let marker = matching_marker(&edge.target, &markers)
                .unwrap_or_else(|| edge.target.clone());
            out.push(SemanticFinding::new(
                "HLT-006-DIRECT-DB-WRONG-LAYER",
                file.rel_path.clone(),
                Some(edge.line),
                Some(marker.clone()),
                "Code outside the DB/adapters/migrations boundary imports direct DB infrastructure.",
                format!(
                    "Path `{}` is not a DB/adapters/migrations path and imports `{}`.",
                    file.rel_path, edge.target
                ),
                "Move direct DB access into the DB/adapters boundary. Expose behavior through a typed domain/application interface and cover the change with the mapped proof lane.",
                vec![
                    format!("DB root paths: {}", root_paths.join(", ")),
                    format!("migration paths: {}", migration_paths.join(", ")),
                    format!("adapter paths: {}", adapter_paths.join(", ")),
                    format!("import line {}: {}", edge.line, edge.raw.trim()),
                    format!("matched DB marker: `{marker}`"),
                ],
                0.96,
            ));
            continue;
        }

        if let Some((line, marker, raw)) = first_marker_in_text(&file.text, &markers) {
            out.push(SemanticFinding::new(
                "HLT-006-DIRECT-DB-WRONG-LAYER",
                file.rel_path.clone(),
                Some(line),
                Some(marker.clone()),
                "Code outside the DB/adapters/migrations boundary contains direct DB access.",
                format!(
                    "Path `{}` is not a DB/adapters/migrations path and contains DB marker `{}`.",
                    file.rel_path, marker
                ),
                "Move raw SQL/ORM/client usage into the DB/adapters boundary. Keep domain/web/Python layers behind typed contracts and explicit proof lanes.",
                vec![
                    format!("DB root paths: {}", root_paths.join(", ")),
                    format!("migration paths: {}", migration_paths.join(", ")),
                    format!("adapter paths: {}", adapter_paths.join(", ")),
                    format!("line {line}: {}", raw.trim()),
                    format!("matched DB marker: `{marker}`"),
                ],
                0.90,

        }
    }

fn streaming_runtime_drift(
    ctx: &AuditContext,
    manifest: &BoundaryManifest,
) -> Vec<SemanticFinding> {
    let queue_policy = manifest.queue_policy();
    let adapter_paths = effective_strings(
        &queue_policy.adapter_paths,
        &[
            "crates/adapters/queue",
            "crates/adapters/streaming",
            "packages/adapters/queue",
            "apps/api/src/adapters/queue",
        ],
    );
    let client_markers = effective_strings(
        &queue_policy.client_markers,
        DEFAULT_STREAMING_CLIENT_MARKERS,
    );

    let mut out = Vec::new();

    for file in ctx.scope_files.iter().filter(|file| is_code_like(file)) {
        if skip_semantic_file(file) || matches_any_policy_path(&file.rel_path, &adapter_paths) {

        }

        let import_edges = import_edges(file);
        if let Some(edge) = import_edges
            .iter()
            .find(|edge| marker_matches_any(&edge.target, &client_markers))

                .unwrap_or_else(|| edge.target.clone());
            let runtime = runtime_for_marker(&marker);

            if exception_allows_runtime(manifest, runtime, &mut out) {

            out.push(SemanticFinding::new(

                file.rel_path.clone(),
                Some(edge.line),
                Some(marker.clone()),
                "Non-adapter code imports a streaming runtime/client.",
                format!(
                    "Path `{}` is outside queue adapter paths and imports streaming runtime `{}`.",
                    file.rel_path, edge.target
                ),
                "Move streaming client usage into the queue adapter boundary. If this is an approved brownfield runtime, record a complete owner/expiry/migration exception in agent/boundaries.toml.",
                vec![
                    format!("queue adapter paths: {}", adapter_paths.join(", ")),
                    format!("import line {}: {}", edge.line, edge.raw.trim()),
                    format!("matched streaming marker: `{marker}`"),
                ],
                0.96,
            ));
            continue;
        }

        if let Some((line, marker, raw)) = first_marker_in_text(&file.text, &client_markers) {
            let runtime = runtime_for_marker(&marker);

            if exception_allows_runtime(manifest, runtime, &mut out) {
                continue;
            }

            out.push(SemanticFinding::new(
                "HLT-019-STREAMING-RUNTIME-DRIFT",
                file.rel_path.clone(),
                Some(line),
                Some(marker.clone()),
                "Non-adapter code contains streaming runtime/client marker.",
                format!(
                    "Path `{}` is outside queue adapter paths and contains streaming runtime marker `{}`.",
                    file.rel_path, marker
                ),
                "Move streaming client usage into the queue adapter boundary. If this is an approved brownfield runtime, record a complete owner/expiry/migration exception in agent/boundaries.toml.",
                vec![
                    format!("queue adapter paths: {}", adapter_paths.join(", ")),
                    format!("line {line}: {}", raw.trim()),
                    format!("matched streaming marker: `{marker}`"),
                ],
                0.90,
            ));
        }
    }

    out

#[derive(Debug, Clone)]
struct ImportEdge {
    line: usize,
    target: String,
    raw: String,
}

fn import_edges(file: &FileInfo) -> Vec<ImportEdge> {
    let mut edges = Vec::new();

    for (idx, raw) in file.text.lines().enumerate() {
        let line_no = idx + 1;
        let line = raw.trim();

        if line.starts_with("//") || line.starts_with('#') {
            continue;
        }

        if file.suffix == ".rs" {
            if let Some(target) = parse_rust_use(line) {
                edges.push(ImportEdge {
                    line: line_no,
                    target,
                    raw: raw.to_string(),
                });
            }
            continue;
        }

        if matches!(
            file.suffix.as_str(),
            ".ts" | ".tsx" | ".js" | ".jsx" | ".mjs" | ".cjs"
        ) {
            if let Some(target) = parse_ts_import(line) {
                edges.push(ImportEdge {
                    line: line_no,
                    target,
                    raw: raw.to_string(),
                });
            }
            continue;
        }

        if file.suffix == ".py" {
            if let Some(target) = parse_python_import(line) {
                edges.push(ImportEdge {
                    line: line_no,
                    target,
                    raw: raw.to_string(),
                });
            }
        }
    }

    edges
}

fn parse_rust_use(line: &str) -> Option<String> {
    let line = line.strip_prefix("use ")?;
    let target = line.trim_end_matches(';').trim();
    if target.is_empty() {
        None
    } else {
        Some(target.to_string())
    }
}

fn parse_ts_import(line: &str) -> Option<String> {
    if line.starts_with("import ") {
        if let Some(target) = quoted_tail_after_from(line) {
            return Some(target);
        }

        if let Some(target) = first_quoted_string(line) {
            return Some(target);
        }
    }

    if line.contains("require(") {
        return first_quoted_string(line);
    }

    None
}

fn parse_python_import(line: &str) -> Option<String> {
    if let Some(rest) = line.strip_prefix("import ") {
        let target = rest
            .split(|ch: char| ch == ',' || ch.is_whitespace())
            .next()
            .unwrap_or_default()
            .trim();
        if !target.is_empty() {
            return Some(target.to_string());
        }
    }

    if let Some(rest) = line.strip_prefix("from ") {
        let target = rest.split_whitespace().next().unwrap_or_default().trim();
        if !target.is_empty() {
            return Some(target.to_string());
        }
    }

    None
}

fn quoted_tail_after_from(line: &str) -> Option<String> {
    let from_idx = line.find(" from ")?;
    first_quoted_string(&line[from_idx..])
}

fn first_quoted_string(line: &str) -> Option<String> {
    let bytes = line.as_bytes();
    let mut start = None;
    let mut quote = b'\0';

    for (idx, byte) in bytes.iter().enumerate() {
        if *byte == b'\'' || *byte == b'"' {
            start = Some(idx + 1);
            quote = *byte;
            break;
        }
    }

    let start = start?;
    let end = bytes[start..].iter().position(|byte| *byte == quote)? + start;
    Some(line[start..end].to_string())
}

fn exception_allows_runtime(
    manifest: &BoundaryManifest,
    runtime: &str,
    out: &mut Vec<SemanticFinding>,
) -> bool {
    let Some(exception) = manifest
        .streaming_exception
        .iter()
        .find(|exception| exception.runtime.eq_ignore_ascii_case(runtime))
    else {
        return false;
    };

    let missing = invalid_exception_reasons(exception);
    if missing.is_empty() {
        return true;
    }

    out.push(SemanticFinding::new(
        "HLT-019-STREAMING-RUNTIME-DRIFT",
        BOUNDARIES_MANIFEST,
        Some(1),
        Some(format!("{runtime}-exception")),
        "Streaming runtime exception is incomplete or expired.",
        format!(
            "Runtime `{runtime}` has an exception entry, but it is missing moonshot-required ownership, expiry, classification/reason, or migration data."
        ),
        "Update the streaming exception with owner, unexpired expires date, classification or reason, and migration_path. Otherwise remove the exception and move the runtime behind an adapter.",
        missing
            .into_iter()
            .map(|reason| format!("invalid exception: {reason}"))
            .collect(),
        0.99,
    ));

    true
}

fn invalid_exception_reasons(exception: &StreamingException) -> Vec<String> {
    let mut out = Vec::new();

    if exception.owner.trim().is_empty() {
        out.push("owner is empty".to_string());
    }

    if exception.migration_path.trim().is_empty() {
        out.push("migration_path is empty".to_string());
    }

    let classification = exception
        .classification
        .as_deref()
        .unwrap_or_default()
        .trim();
    let reason = exception.reason.as_deref().unwrap_or_default().trim();
    if classification.is_empty() && reason.is_empty() {
        out.push("classification/reason is empty".to_string());
    }

    if exception.expires.trim().is_empty() {
        out.push("expires is empty".to_string());
    } else if !expiry_is_today_or_future(&exception.expires) {
        out.push(format!(
            "expires `{}` is invalid or already expired",
            exception.expires
        ));
    }

    out
}

fn expiry_is_today_or_future(value: &str) -> bool {
    let Some((year, month, day)) = parse_ymd(value) else {
        return false;
    };

    let Some(today_days) = current_days_since_unix_epoch() else {
        return true;
    };

    let expiry_days = days_from_civil(year, month, day);
    expiry_days >= today_days
}

fn parse_ymd(value: &str) -> Option<(i32, u32, u32)> {
    let mut parts = value.split('-');
    let year = parts.next()?.parse::<i32>().ok()?;
    let month = parts.next()?.parse::<u32>().ok()?;
    let day = parts.next()?.parse::<u32>().ok()?;
    if parts.next().is_some() || month == 0 || month > 12 || day == 0 || day > 31 {
        return None;
    }
    Some((year, month, day))
}

fn current_days_since_unix_epoch() -> Option<i64> {
    let duration = SystemTime::now().duration_since(UNIX_EPOCH).ok()?;
    Some((duration.as_secs() / 86_400) as i64)
}

fn days_from_civil(year: i32, month: u32, day: u32) -> i64 {
    let year = year - i32::from(month <= 2);
    let era = if year >= 0 { year } else { year - 399 } / 400;
    let yoe = year - era * 400;
    let month = month as i32;
    let day = day as i32;
    let mp = month + if month > 2 { -3 } else { 9 };
    let doy = (153 * mp + 2) / 5 + day - 1;
    let doe = yoe * 365 + yoe / 4 - yoe / 100 + doy;
    (era * 146_097 + doe - 719_468) as i64
}

fn runtime_for_marker(marker: &str) -> &str {
    let marker = marker.to_ascii_lowercase();

    if marker.contains("kafka") || marker.contains("rdkafka") {
        "kafka"
    } else if marker.contains("nats") {
        "nats"
    } else if marker.contains("redis") || marker.contains("xread") || marker.contains("xadd") {
        "redis"
    } else if marker.contains("fluvio") {
        "fluvio"
    } else if marker.contains("iggy") {
        "iggy"
    } else if marker.contains("tansu") {
        "tansu"
    } else {
        "streaming"
    }
}

fn handwritten_contract_line(file: &FileInfo) -> Option<(usize, &'static str, String)> {
    for (idx, raw) in file.text.lines().enumerate() {
        let line = raw.trim();
        let lower = line.to_ascii_lowercase();

        if lower.starts_with("//") || lower.starts_with("/*") {
            continue;
        }

        if (lower.contains("interface ") || lower.contains("type "))
            && (lower.contains("dto") || lower.contains("response") || lower.contains("request"))
        {
            return Some((idx + 1, "handwritten-dto-contract", raw.to_string()));
        }

        if lower.contains("fetch(") && (lower.contains("/api/") || lower.contains("application/json"))
        {
            return Some((idx + 1, "handwritten-fetch-client", raw.to_string()));
        }

        if lower.contains("axios.") || lower.contains("axios(") {
            return Some((idx + 1, "handwritten-axios-client", raw.to_string()));
        }
    }

    None
}

fn first_marker_in_text(text: &str, markers: &[String]) -> Option<(usize, String, String)> {
    for (idx, raw) in text.lines().enumerate() {
        let line = raw.trim();
        if line.is_empty() || line.starts_with("//") || line.starts_with('#') {
            continue;
        }

        for marker in markers {
            if marker_matches(line, marker) {
                return Some((idx + 1, marker.clone(), raw.to_string()));
            }
        }
    }

    None
}

fn marker_matches_any(text: &str, markers: &[String]) -> bool {
    markers.iter().any(|marker| marker_matches(text, marker))
}

fn matching_marker(text: &str, markers: &[String]) -> Option<String> {
    markers
        .iter()
        .find(|marker| marker_matches(text, marker))
        .cloned()
}

fn marker_matches(text: &str, marker: &str) -> bool {
    let text = text.to_ascii_lowercase();
    let marker = marker.to_ascii_lowercase();

    if marker.trim().is_empty() {
        return false;
    }

    if marker
        .chars()
        .all(|ch| ch.is_ascii_alphanumeric() || ch == '_' || ch == '-')
        && marker.len() <= 5
    {
        return text
            .split(|ch: char| !(ch.is_ascii_alphanumeric() || ch == '_' || ch == '-'))
            .any(|token| token == marker);
    }

    text.contains(&marker)
}

fn effective_strings(values: &[String], defaults: &[&str]) -> Vec<String> {
    if values.is_empty() {
        strings_from_static(defaults)
    } else {
        values
            .iter()
            .map(|value| value.trim().to_string())
            .filter(|value| !value.is_empty())
            .collect()
    }
}

fn strings_from_static(values: &[&str]) -> Vec<String> {
    values.iter().map(|value| (*value).to_string()).collect()
}

fn matches_any_policy_path(path: &str, patterns: &[String]) -> bool {
    patterns
        .iter()
        .any(|pattern| path_policy_match(path, pattern))
}

fn path_policy_match(path: &str, pattern: &str) -> bool {
    let path = normalize_path(path);
    let pattern = normalize_path(pattern);

    if pattern.is_empty() {
        return false;
    }

    if !pattern.contains('*') {
        return path_prefix_match(&path, &pattern);
    }

    wildcard_prefix_match(&path, &pattern)
}

fn path_prefix_match(path: &str, prefix: &str) -> bool {
    let path = normalize_path(path);
    let prefix = normalize_path(prefix);
    path == prefix || path.starts_with(&format!("{prefix}/"))
}

fn wildcard_prefix_match(path: &str, pattern: &str) -> bool {
    let path_parts: Vec<&str> = path.split('/').collect();
    let pattern_parts: Vec<&str> = pattern.split('/').collect();

    if pattern_parts.len() > path_parts.len() {
        return false;
    }

    for (idx, pattern_part) in pattern_parts.iter().enumerate() {
        if *pattern_part == "*" {
            continue;
        }

        if path_parts.get(idx).copied() != Some(*pattern_part) {
            return false;
        }
    }

    true
}

fn normalize_path(path: &str) -> String {
    path.replace('\\', "/").trim_matches('/').to_string()
}

pub fn validate_registry() -> Result<(), String> {
    let mut seen = std::collections::HashSet::new();

    for rule in all() {
        if !seen.insert(rule.id) {
            return Err(format!("duplicate rule id `{}`", rule.id));
        }

        if !rule.id.starts_with("HLT-") {
            return Err(format!("rule `{}` does not use HLT-* id format", rule.id));
        }

        if rule.name.trim().is_empty() {
            return Err(format!("rule `{}` is missing name", rule.id));
        }

        if rule.category.trim().is_empty() {
            return Err(format!("rule `{}` is missing category", rule.id));
        }

        if rule.tlr.trim().is_empty() {
            return Err(format!("rule `{}` is missing TLR", rule.id));
        }

        if rule.severity.trim().is_empty() {
            return Err(format!("rule `{}` is missing severity", rule.id));
        }

        if rule.lane.trim().is_empty() {
            return Err(format!("rule `{}` is missing proof lane", rule.id));
        }

        if rule.owner_hint.trim().is_empty() {
            return Err(format!("rule `{}` is missing owner hint", rule.id));
        }

        if rule.docs_url.trim().is_empty() {
            return Err(format!("rule `{}` is missing docs URL", rule.id));
        }

        if rule.evidence_kind.trim().is_empty() {
            return Err(format!("rule `{}` is missing evidence kind", rule.id));
        }
    }

    Ok(())
}

* Audit output includes semantic boundary findings with `rule_id`, `line`, `matched_term`, `reason`, `owner`, `lane`, `docs_url`, `rerun_command`, `evidence_kind`, and `agent_fix`.
* Unknown semantic rule IDs fail fast.
* Missing/invalid `agent/boundaries.toml` becomes an actionable finding instead of a silent fallback.
* Streaming runtime exceptions must be owner-bound, expiring, classified/reasoned, and tied to a migration path.

That gets **Phase 02: Rule Engine + Semantic Oracle** to 100% in the way `moonshot.md` wants: agents get explicit boundaries, trusted policy, actionable evidence, validation lanes, owner mapping, and reusable rule primitives rather than ad hoc scans.
