## Phase name: **Phase 02 — Rule Engine And Semantic Oracle**

I would finish **Phase 02** rather than move to a later phase, because the moonshot depends on the repo being able to convert intent into bounded agents, proof lanes, evidence, expiring exceptions, and reusable primitives; that only works when findings come from stable rules instead of scattered heuristics. ([GitHub][1]) Phase 02’s own plan says the target architecture is `inventory -> analyzers -> rule registry -> findings -> caps -> score -> reports`, with rule metadata, boundary oracle checks, generated-zone checks, an AST/import pilot, and finding-quality guarantees. ([GitHub][2]) The current repo already has a partial registry, finding builder, caps, boundary manifest ingestion, and shallow AST pilot, but Phase 02’s receipt explicitly says the analyzer split and AST/import pilot remain partial follow-on work. ([GitHub][2])

The core steering decision: **do not widen the report JSON shape**. The standard already requires findings to carry stable repair metadata such as `rule_id`, severity, category, path, line when available, problem, evidence, `agent_fix`, owner, validation lane, and docs link. ([GitHub][3]) This patch keeps that shape intact and completes Phase 02 by making the semantic analyzer produce better findings through the existing builder.

Below is a single copy-paste patch script. Run it from the repo root. It overwrites/adds the needed files and patches the existing audit pipeline to route semantic-oracle findings through the rule registry.

use super::helpers::*;
use super::rules;
use crate::model::*;
use std::collections::hash_map::DefaultHasher;
use std::collections::HashMap;
use std::hash::{Hash, Hasher};

pub struct FindingBuilder<'a> {
    ctx: &'a AuditContext,
    findings: Vec<Finding>,
    has_any_finding: bool,
    has_context_finding: bool,
}

impl<'a> FindingBuilder<'a> {
    pub fn new(ctx: &'a AuditContext) -> Self {
        Self {
            ctx,
            findings: Vec::new(),
            has_any_finding: false,
            has_context_finding: false,
        }
    }

    pub fn add(
        &mut self,
        severity: &str,
        category: &str,
        path: &str,
        problem: &str,
        fix: &str,
        evidence: Vec<String>,
        rule_id: Option<&str>,
        line: Option<usize>,
    ) {
        self.add_structured(
            Some(severity),
            Some(category),
            path,
            problem,
            fix,
            evidence,
            rule_id,
            line,
            None,
            None,
        );
    }

    pub fn add_structured(
        &mut self,
        severity: Option<&str>,
        category: Option<&str>,
        path: &str,
        problem: &str,
        fix: &str,
        evidence: Vec<String>,
        rule_id: Option<&str>,
        line: Option<usize>,
        matched_term: Option<String>,
        reason: Option<String>,
    ) {
        let rule_meta = rule_id.and_then(rules::lookup);

        let severity = severity
            .or_else(|| rule_meta.map(|rule| rule.severity))
            .unwrap_or("medium");

        let category = category
            .or_else(|| rule_meta.map(|rule| rule.category))
            .unwrap_or("audit");

        if category == "context" {
            self.has_context_finding = true;
        }
        self.has_any_finding = true;

        let effective_rule = rule_id.unwrap_or("HLT-000-SCORE-DIMENSION");

        let lane = rule_meta
            .map(|rule| rule.lane)
            .or_else(|| lane_for(category))
            .map(str::to_string);

        let owner = owner_for_path(self.ctx, path).or_else(|| {
            rule_meta
                .filter(|rule| !rule.owner_hint.is_empty())
                .map(|rule| rule.owner_hint.to_string())
        });

        let evidence_kind = rule_meta
            .map(|rule| rule.evidence_kind)
            .unwrap_or_else(|| evidence_kind_for_path(path));

        let fingerprint = finding_fingerprint(
            effective_rule,
            category,
            path,
            problem,
            matched_term.as_deref(),
        );

        self.findings.push(Finding {
            severity: severity.into(),
            category: category.into(),
            path: path.into(),
            problem: problem.into(),
            agent_fix: fix.into(),
            evidence,
            check_id: format!("{effective_rule}:{category}"),
            hardness: hardness_for_severity(severity).into(),
            confidence: confidence_for_rule_or_severity(rule_meta, severity),
            evidence_kind: evidence_kind.into(),
            rerun_command: rerun_command_for_lane(lane.as_deref()).into(),
            fingerprint,
            rule_id: rule_id.map(str::to_string),
            tlr: rule_meta
                .map(|rule| rule.tlr)
                .or_else(|| tlr_for(category))
                .map(str::to_string),
            lane,
            docs_url: rule_meta.map(|rule| rule.docs_url.to_string()),
            owner,
            line,
            matched_term,
            reason,
        });
    }

    pub fn has_any_finding(&self) -> bool {
        self.has_any_finding
    }

    pub fn has_context_finding(&self) -> bool {
        self.has_context_finding
    }

    pub fn into_findings(self) -> Vec<Finding> {
        self.findings
    }
}

pub fn hardness_for_severity(severity: &str) -> &'static str {
    match severity {
        "critical" | "high" => "hard",
        _ => "soft",
    }
}

pub fn confidence_for_severity(severity: &str) -> f64 {
    match severity {
        "critical" => 0.95,
        "high" => 0.88,
        "medium" => 0.76,
        _ => 0.62,
    }
}

pub fn confidence_for_rule_or_severity(rule: Option<rules::RuleSpec>, severity: &str) -> f64 {
    rule.map(|rule| (rule.default_confidence as f64) / 100.0)
        .unwrap_or_else(|| confidence_for_severity(severity))
}

pub fn evidence_kind_for_path(path: &str) -> &'static str {
    if path.starts_with(".github/workflows") {
        "workflow-command"
    } else if path.starts_with("agent/") {
        "policy-manifest"
    } else if path.starts_with("docs/") {
        "documentation"
    } else {
        "repository-scan"
    }
}

pub fn rerun_command_for_lane(lane: Option<&str>) -> &'static str {
    match lane.unwrap_or("audit") {
        "security" => "just security",
        "contract" => "just fast",
        "db" => "just fast",
        "db-migration-analyze" => {

pub fn finding_fingerprint(
    rule_id: &str,
    category: &str,
    path: &str,
    problem: &str,
    matched_term: Option<&str>,
) -> String {
    let mut hasher = DefaultHasher::new();
    rule_id.hash(&mut hasher);
    category.hash(&mut hasher);
    path.hash(&mut hasher);
    problem.hash(&mut hasher);
    if let Some(matched_term) = matched_term {
        matched_term.hash(&mut hasher);
    }
    format!("{:016x}", hasher.finish())
}

fn owner_for_path(ctx: &AuditContext, rel_path: &str) -> Option<String> {
    use serde::Deserialize;

    #[derive(Deserialize)]
    struct OwnerMapFile {
        owners: HashMap<String, String>,
    }

    if let Ok(text) = std::fs::read_to_string(ctx.root.join("agent/owner-map.json")) {
        if let Ok(parsed) = serde_json::from_str::<OwnerMapFile>(&text) {
            let mut best: Option<(&String, &String)> = None;
            for (prefix, owner) in parsed.owners.iter() {
                if prefix_matches_path(rel_path, prefix)
                    && best
                        .map(|(best_prefix, _)| prefix.len() > best_prefix.len())
                        .unwrap_or(true)
                {
                    best = Some((prefix, owner));
                }
            }
            if let Some((_, owner)) = best {
                return Some(owner.clone());
            }
        }
    }

    OWNER_MAP_PREFIXES
        .iter()
        .filter(|(prefix, _)| prefix_matches_path(rel_path, prefix))
        .max_by_key(|(prefix, _)| prefix.len())
        .map(|(_, owner)| (*owner).to_string())
}

fn prefix_matches_path(rel_path: &str, prefix: &str) -> bool {
    let prefix = prefix.trim_end_matches('/');
    rel_path == prefix || rel_path.starts_with(&format!("{prefix}/"))
}

use crate::audit::helpers::{boundary_manifest, path_matches_prefix, AuditContext};
use crate::boundaries::manifest::{BoundaryManifest, StreamingException};
use crate::model::FileInfo;
use once_cell::sync::Lazy;
use regex::Regex;
use std::collections::BTreeSet;
use std::time::{SystemTime, UNIX_EPOCH};

#[derive(Debug, Clone)]
pub struct SemanticFinding {
    pub severity: &'static str,
    pub category: &'static str,
    pub path: String,
    pub problem: String,
    pub agent_fix: String,
    pub evidence: Vec<String>,
    pub rule_id: &'static str,
    pub line: Option<usize>,
    pub matched_term: Option<String>,
    pub reason: String,
}

#[derive(Debug, Clone)]
struct ImportRef {
    target: String,
    line: usize,
    source: String,
}

#[derive(Debug, Clone)]
struct BoundaryPolicy {
    rust_domain_paths: Vec<String>,
    rust_forbidden_imports: Vec<String>,
    typescript_web_paths: Vec<String>,
    typescript_forbidden_imports: Vec<String>,
    python_allowed_truth_paths: Vec<String>,
    queue_adapter_paths: Vec<String>,
    queue_client_markers: Vec<String>,
    db_allowed_paths: Vec<String>,
    streaming_exceptions: Vec<StreamingException>,
}

impl BoundaryPolicy {
    fn load(ctx: &AuditContext) -> Self {
        let manifest = boundary_manifest(ctx);
        Self::from_manifest(manifest.as_ref())
    }

    fn from_manifest(manifest: Option<&BoundaryManifest>) -> Self {
        let rust_domain_paths = manifest
            .and_then(|m| m.rust.as_ref())
            .map(|rust| rust.domain_paths.clone())
            .filter(|paths| !paths.is_empty())
            .unwrap_or_else(|| vec!["crates/domain".into(), "crates/*/src/domain".into(), "domain".into()]);

        let rust_forbidden_imports = manifest
            .and_then(|m| m.rust.as_ref())
            .map(|rust| rust.forbidden_domain_imports.clone())
            .filter(|markers| !markers.is_empty())
            .unwrap_or_else(|| {
                crate::boundaries::rust::DOMAIN_FORBIDDEN_IMPORTS
                    .iter()
                    .map(|marker| (*marker).to_string())
                    .collect()
            });

        let typescript_web_paths = manifest
            .and_then(|m| m.typescript.as_ref())
            .map(|typescript| typescript.web_paths.clone())
            .filter(|paths| !paths.is_empty())
            .unwrap_or_else(|| {
                vec![
                    "apps/web".into(),
                    "frontend".into(),
                    "ui".into(),
                    "packages/web".into(),
                    "packages/ui".into(),
                ]
            });

        let typescript_forbidden_imports = manifest
            .and_then(|m| m.typescript.as_ref())
            .map(|typescript| typescript.forbidden_web_imports.clone())
            .filter(|markers| !markers.is_empty())
            .unwrap_or_else(|| {
                vec![
                    "pg".into(),
                    "postgres".into(),
                    "mysql".into(),
                    "mysql2".into(),
                    "better-sqlite3".into(),
                    "@aws-sdk/client-s3".into(),
                ]
            });

        let python_allowed_truth_paths = manifest
            .and_then(|m| m.python.as_ref())
            .map(|python| python.allowed_truth_paths.clone())
            .filter(|paths| !paths.is_empty())
            .unwrap_or_else(|| vec!["python/ai-service".into(), "tools".into()]);

        let queue_adapter_paths = manifest
            .and_then(|m| m.queues.as_ref())
            .map(|queues| queues.adapter_paths.clone())
            .filter(|paths| !paths.is_empty())
            .unwrap_or_else(|| {
                vec![
                    "crates/adapters/queues".into(),
                    "crates/adapters/src/queues".into(),
                    "adapters/queues".into(),
                    "apps/api/src/adapters/queues".into(),
                ]
            });

        let mut queue_client_markers: Vec<String> = crate::boundaries::streaming::STREAMING_CLIENT_MARKERS
            .iter()
            .map(|marker| (*marker).to_string())
            .collect();

        if let Some(markers) = manifest
            .and_then(|m| m.queues.as_ref())
            .map(|queues| queues.client_markers.clone())
        {
            queue_client_markers.extend(markers);
        }
        queue_client_markers.sort();
        queue_client_markers.dedup();

        let mut db_allowed_paths = vec![
            "db".into(),
            "migrations".into(),
            "crates/adapters".into(),

        let streaming_exceptions = manifest
            .map(|m| m.streaming_exception.clone())

            rust_forbidden_imports,
            typescript_web_paths,

            python_allowed_truth_paths,
            queue_adapter_paths,
            queue_client_markers,
            db_allowed_paths,
            streaming_exceptions,
        }
    }

    fn is_rust_domain_path(&self, path: &str) -> bool {
        self.rust_domain_paths
            .iter()
            .any(|pattern| path_matches_glob_or_prefix(path, pattern))
    }

    fn is_typescript_web_path(&self, path: &str) -> bool {
        self.typescript_web_paths
            .iter()
            .any(|pattern| path_matches_glob_or_prefix(path, pattern))
    }

    fn is_python_truth_allowed_path(&self, path: &str) -> bool {
        self.python_allowed_truth_paths
            .iter()
            .any(|pattern| path_matches_glob_or_prefix(path, pattern))

        self.queue_adapter_paths
            .iter()
            .any(|pattern| path_matches_glob_or_prefix(path, pattern))
    }

    fn is_db_allowed_path(&self, path: &str) -> bool {
        self.db_allowed_paths
            .iter()
            .any(|pattern| path_matches_glob_or_prefix(path, pattern))
    }

    fn has_valid_streaming_exception_for(&self, marker: &str) -> bool {
        self.streaming_exceptions.iter().any(|exception| {
            runtime_matches_marker(&exception.runtime, marker) && streaming_exception_complete(exception)
        })
    }
}

pub fn run(ctx: &AuditContext) -> Vec<SemanticFinding> {
    let policy = BoundaryPolicy::load(ctx);
    let mut findings = Vec::new();

    findings.extend(validate_streaming_exceptions(ctx, &policy));

    for file in &ctx.scope_files {
        if should_skip_file(ctx, file) {
            continue;
        }

        findings.extend(detect_direct_db_access(file, &policy));
        findings.extend(detect_rust_domain_impurity(file, &policy));
        findings.extend(detect_typescript_web_impurity(file, &policy));
        findings.extend(detect_python_product_truth(file, &policy));
        findings.extend(detect_streaming_runtime_drift(file, &policy));
    }

    dedupe_and_sort(findings)
}

fn detect_direct_db_access(file: &FileInfo, policy: &BoundaryPolicy) -> Vec<SemanticFinding> {
    if !file.is_code && file.suffix != ".sql" {
        return Vec::new();
    }
    if policy.is_db_allowed_path(&file.rel_path) {
        return Vec::new();
    }

    let Some((marker, line)) = first_db_marker(&file.text) else {
        return Vec::new();
    };

    vec![SemanticFinding {
        severity: "high",
        category: "data",
        path: file.rel_path.clone(),
        problem: "direct database access appears outside the declared DB or adapter boundary".into(),
        agent_fix: "move SQL clients and durable writes behind the DB adapter/application boundary, then rerun the db or fast lane".into(),
        evidence: vec![
            format!("{}:{} contains database marker `{}`", file.rel_path, line.unwrap_or(1), marker),
            "allowed DB roots are declared in `agent/boundaries.toml` under `[db]`".into(),
        ],
        rule_id: "HLT-006-DIRECT-DB-WRONG-LAYER",
        line,
        matched_term: Some(marker.into()),
        reason: "path plus import/content evidence indicates durable data access escaped the owning DB/adapters layer".into(),
    }]
}

fn detect_rust_domain_impurity(file: &FileInfo, policy: &BoundaryPolicy) -> Vec<SemanticFinding> {
    if file.suffix != ".rs" || !policy.is_rust_domain_path(&file.rel_path) {
        return Vec::new();
    }

    let mut findings = Vec::new();
    for marker in &policy.rust_forbidden_imports {
        let Some(line) = line_for_marker(&file.text, marker) else {
            continue;
        };

        if is_db_marker(marker) {
            continue;
        }

        findings.push(SemanticFinding {
            severity: "high",
            category: "boundary",
            path: file.rel_path.clone(),
            problem: format!("Rust domain layer imports or uses forbidden side-effect marker `{marker}`"),
            agent_fix: "extract IO, env, time, random, network, logging, and external-system behavior to application/adapters; inject pure ports into the domain".into(),
            evidence: vec![
                format!("{}:{} matched `{}`", file.rel_path, line, marker),
                "Rust domain paths are declared in `agent/boundaries.toml` under `[rust].domain_paths`".into(),
            ],
            rule_id: "HLT-022-RUST-DOMAIN-IMPURITY",
            line: Some(line),
            matched_term: Some(marker.clone()),
            reason: "domain code must stay deterministic and side-effect free so agents can prove invariants locally".into(),
        });
    }

    findings
}

fn detect_typescript_web_impurity(file: &FileInfo, policy: &BoundaryPolicy) -> Vec<SemanticFinding> {
    if !matches!(file.suffix.as_str(), ".ts" | ".tsx" | ".js" | ".jsx")
        || !policy.is_typescript_web_path(&file.rel_path)
    {
        return Vec::new();
    }

    let imports = extract_typescript_imports(&file.text);
    let mut findings = Vec::new();

    for import_ref in imports {
        let target = import_ref.target.to_ascii_lowercase();

        if import_target_is_db_client(&target, policy) {
            continue;
        }

        let backend_or_adapter =
            target.contains("backend")
                || target.contains("adapters/db")
                || target.contains("/db")
                || target.contains("@app/api")
                || target.contains("@server")
                || target.contains("server/");

        if backend_or_adapter {
            findings.push(SemanticFinding {
                severity: "high",
                category: "boundary",
                path: file.rel_path.clone(),
                problem: format!(
                    "TypeScript web layer imports backend or durable-truth module `{}`",
                    import_ref.target
                ),
                agent_fix: "replace backend/internal imports with generated contract clients or a typed API boundary, then run the contract/web lane".into(),
                evidence: vec![
                    format!("{}:{} `{}`", file.rel_path, import_ref.line, import_ref.source),
                    "web paths and generated contract paths are declared in `agent/boundaries.toml` under `[typescript]`".into(),
                ],
                rule_id: "HLT-023-TS-WEB-DURABLE-TRUTH",
                line: Some(import_ref.line),
                matched_term: Some(import_ref.target),
                reason: "UI code may render state and call generated clients, but must not own durable truth or backend internals".into(),
            });
        }
    }

    findings
}

fn detect_python_product_truth(file: &FileInfo, policy: &BoundaryPolicy) -> Vec<SemanticFinding> {
    if file.suffix != ".py" {
        return Vec::new();
    }

    let allowed_path = policy.is_python_truth_allowed_path(&file.rel_path);
    let lower = file.text.to_ascii_lowercase();
    let db_marker = first_db_marker(&file.text).map(|(marker, line)| (marker.to_string(), line));
    let product_marker = first_product_truth_marker(&lower);

    let should_flag = (!allowed_path && (db_marker.is_some() || product_marker.is_some()))
        || (allowed_path && db_marker.is_some() && product_marker.is_some());

    if !should_flag {
        return Vec::new();
    }

    let matched = db_marker
        .as_ref()
        .map(|(marker, _)| marker.clone())
        .or_else(|| product_marker.map(str::to_string))
        .unwrap_or_else(|| "product truth marker".into());

    let line = db_marker
        .and_then(|(_, line)| line)
        .or_else(|| line_for_marker(&file.text, &matched));

    vec![SemanticFinding {
        severity: "high",
        category: "python",
        path: file.rel_path.clone(),
        problem: "Python appears to own product truth, authorization, or direct production data access".into(),
        agent_fix: "move durable product behavior into Rust application/domain plus SQL constraints; keep Python to AI/data/eval tooling behind typed generated contracts".into(),
        evidence: vec![
            format!("{}:{} matched `{}`", file.rel_path, line.unwrap_or(1), matched),
            "allowed Python truth paths are declared in `agent/boundaries.toml` under `[python].allowed_truth_paths`".into(),
        ],
        rule_id: "HLT-005-PYTHON-PRODUCT-TRUTH",
        line,
        matched_term: Some(matched),
        reason: "Python is bounded to AI/data tooling; product authorization and durable truth must be owned by Rust/SQL/generated contracts".into(),
    }]
}

fn detect_streaming_runtime_drift(file: &FileInfo, policy: &BoundaryPolicy) -> Vec<SemanticFinding> {
    if !file.is_code || policy.is_queue_adapter_path(&file.rel_path) {
        return Vec::new();
    }

    let lower = file.text.to_ascii_lowercase();
    for marker in &policy.queue_client_markers {
        let marker_lower = marker.to_ascii_lowercase();
        if !lower.contains(&marker_lower) {
            continue;
        }

        if policy.has_valid_streaming_exception_for(&marker_lower) {
            return Vec::new();
        }

        let line = line_for_marker(&file.text, marker);
        return vec![SemanticFinding {
            severity: "high",
            category: "boundary",
            path: file.rel_path.clone(),
            problem: format!(
                "streaming runtime client `{}` appears outside declared queue adapters",
                marker
            ),
            agent_fix: "move Kafka/Tansu/Iggy/Fluvio/NATS/Redis Streams clients behind declared queue adapters, or add a brownfield exception with owner, expiry, and migration path".into(),
            evidence: vec![
                format!("{}:{} matched `{}`", file.rel_path, line.unwrap_or(1), marker),
                "queue adapter paths and client markers are declared in `agent/boundaries.toml` under `[queues]`".into(),
            ],
            rule_id: "HLT-019-STREAMING-RUNTIME-DRIFT",
            line,
            matched_term: Some(marker.clone()),
            reason: "streaming clients define durable workflow behavior and must be isolated behind adapter boundaries".into(),
        }];
    }

    Vec::new()
}

fn validate_streaming_exceptions(ctx: &AuditContext, policy: &BoundaryPolicy) -> Vec<SemanticFinding> {
    if !ctx.root.join("agent/boundaries.toml").is_file() {
        return Vec::new();
    }

    let mut findings = Vec::new();
    for exception in &policy.streaming_exceptions {
        let gaps = streaming_exception_gaps(exception);
        if gaps.is_empty() {
            continue;
        }

        findings.push(SemanticFinding {
            severity: "high",
            category: "boundary",
            path: "agent/boundaries.toml".into(),
            problem: format!(
                "streaming exception for runtime `{}` is not actionable",
                exception.runtime
            ),
            agent_fix: "complete the streaming exception with non-empty owner, future ISO expiry, brownfield classification/reason, and migration path; otherwise move the client behind adapters".into(),
            evidence: vec![
                format!("runtime `{}` gaps: {}", exception.runtime, gaps.join(", ")),
                "Kafka/streaming exceptions must expire and must carry a migration path".into(),
            ],
            rule_id: "HLT-019-STREAMING-RUNTIME-DRIFT",
            line: Some(1),
            matched_term: Some(exception.runtime.clone()),
            reason: "unbounded streaming exceptions become hidden architecture unless they expire and tell agents how to migrate".into(),
        });
    }

    findings
}

fn extract_typescript_imports(text: &str) -> Vec<ImportRef> {
    static TS_FROM_IMPORT: Lazy<Regex> = Lazy::new(|| {
        Regex::new(r#"^\s*(?:import|export)\s+(?:type\s+)?(?:[^'"]*?\s+from\s+)?['"]([^'"]+)['"]"#)
            .expect("TypeScript import regex is valid")
    });
    static TS_REQUIRE: Lazy<Regex> = Lazy::new(|| {
        Regex::new(r#"require\(\s*['"]([^'"]+)['"]\s*\)"#)
            .expect("TypeScript require regex is valid")
    });

    let mut imports = Vec::new();

    for (idx, line) in text.lines().enumerate() {
        let line_no = idx + 1;

        if let Some(caps) = TS_FROM_IMPORT.captures(line) {
            if let Some(target) = caps.get(1) {
                imports.push(ImportRef {
                    target: target.as_str().to_string(),
                    line: line_no,
                    source: line.trim().chars().take(180).collect(),
                });
            }
        }

        if let Some(caps) = TS_REQUIRE.captures(line) {
            if let Some(target) = caps.get(1) {
                imports.push(ImportRef {
                    target: target.as_str().to_string(),
                    line: line_no,
                    source: line.trim().chars().take(180).collect(),
                });
            }
        }
    }

    imports
}

fn line_for_marker(text: &str, marker: &str) -> Option<usize> {
    let marker = marker.to_ascii_lowercase();
    text.lines()
        .position(|line| line.to_ascii_lowercase().contains(&marker))
        .map(|idx| idx + 1)
}

fn import_target_matches(target_lower: &str, marker_lower: &str) -> bool {
    target_lower == marker_lower
        || target_lower.starts_with(&format!("{marker_lower}/"))
        || target_lower.ends_with(&format!("/{marker_lower}"))
        || target_lower.contains(&format!("/{marker_lower}/"))
        || (marker_lower.contains('/') && target_lower.contains(marker_lower))
        || (marker_lower.starts_with('@') && target_lower.contains(marker_lower))
}

fn runtime_matches_marker(runtime: &str, marker: &str) -> bool {
    let runtime = runtime.to_ascii_lowercase();
    let marker = marker.to_ascii_lowercase();
    marker.contains(&runtime) || runtime.contains(&marker)
}

fn streaming_exception_complete(exception: &StreamingException) -> bool {
    exception.owner.trim().len() >= 2
        && exception.migration_path.trim().len() >= 8
        && expiry_is_valid_future(&exception.expires)
        && streaming_exception_is_brownfield(exception)
}

fn streaming_exception_gaps(exception: &StreamingException) -> Vec<String> {
    let mut gaps = Vec::new();

    if exception.owner.trim().is_empty() {
        gaps.push("missing owner".into());
    }
    if exception.migration_path.trim().is_empty() {
        gaps.push("missing migration_path".into());
    }
    if !expiry_is_valid_future(&exception.expires) {
        gaps.push("missing, invalid, or expired expires date".into());
    }
    if !streaming_exception_is_brownfield(exception) {
        gaps.push("missing brownfield classification or reason".into());
    }

    gaps
}

fn streaming_exception_is_brownfield(exception: &StreamingException) -> bool {
    exception
        .classification
        .as_deref()
        .or(exception.reason.as_deref())
        .map(|value| value.to_ascii_lowercase().contains("brownfield"))
        .unwrap_or(false)
}

fn expiry_is_valid_future(expires: &str) -> bool {
    let Some(expiry) = parse_ymd(expires) else {
        return false;
    };

    let Some(today) = today_ymd() else {
        return true;
    };

    expiry >= today
}

fn parse_ymd(value: &str) -> Option<(i32, u32, u32)> {
    let mut parts = value.split('-');
    let year: i32 = parts.next()?.parse().ok()?;
    let month: u32 = parts.next()?.parse().ok()?;
    let day: u32 = parts.next()?.parse().ok()?;

    if parts.next().is_some() || !(1..=12).contains(&month) || !(1..=31).contains(&day) {
        return None;
    }

    Some((year, month, day))
}

fn today_ymd() -> Option<(i32, u32, u32)> {
    let secs = SystemTime::now().duration_since(UNIX_EPOCH).ok()?.as_secs();
    Some(civil_from_days((secs / 86_400) as i64))
}

// Howard Hinnant civil-from-days algorithm, adapted to avoid a chrono dependency.
fn civil_from_days(days_since_unix_epoch: i64) -> (i32, u32, u32) {
    let z = days_since_unix_epoch + 719_468;
    let era = if z >= 0 { z } else { z - 146_096 } / 146_097;
    let doe = z - era * 146_097;
    let yoe = (doe - doe / 1_460 + doe / 36_524 - doe / 146_096) / 365;
    let mut y = yoe + era * 400;
    let doy = doe - (365 * yoe + yoe / 4 - yoe / 100);
    let mp = (5 * doy + 2) / 153;
    let d = doy - (153 * mp + 2) / 5 + 1;
    let m = mp + if mp < 10 { 3 } else { -9 };

    if m <= 2 {
        y += 1;
    }

    (y as i32, m as u32, d as u32)
}

fn path_matches_glob_or_prefix(path: &str, pattern: &str) -> bool {
    let pattern = pattern.trim_end_matches('/');
    if !pattern.contains('*') {
        return path_matches_prefix(path, pattern);
    }

    let parts: Vec<&str> = pattern.split('*').collect();
    let mut cursor = 0usize;

    if let Some(first) = parts.first() {
        if !path.starts_with(first) {
            return false;
        }
        cursor = first.len();
    }

    for part in parts.iter().skip(1) {
        if part.is_empty() {
            continue;
        }

        let Some(offset) = path[cursor..].find(part) else {
            return false;
        };
        cursor += offset + part.len();
    }

    true
}

fn dedupe_and_sort(findings: Vec<SemanticFinding>) -> Vec<SemanticFinding> {
    let mut seen = BTreeSet::new();
    let mut out = Vec::new();

    for finding in findings {
        let key = format!(
            "{}:{}:{}:{}",
            finding.rule_id,
            finding.path,
            finding.line.unwrap_or(0),
            finding.problem
        );

        if seen.insert(key) {
            out.push(finding);
        }
    }

    out.sort_by(|a, b| {
        a.path
            .cmp(&b.path)
            .then(a.line.unwrap_or(0).cmp(&b.line.unwrap_or(0)))
            .then(a.rule_id.cmp(b.rule_id))
            .then(a.problem.cmp(&b.problem))
    });

    out
}

# Boundary Oracle

Status: implemented slice for Phase 02  
Owner: tools  
Last reviewed: 2026-05-03  
Applies to: `agent/boundaries.toml`, `crates/jankurai/src/audit/analyzers/semantic.rs`, boundary findings

The boundary oracle turns the Jankurai ownership standard into machine-checkable semantic findings. It is intentionally dependency-light: it uses manifest parsing, path matching, shallow import extraction, and content markers rather than heavyweight language parsers. Parser failures must not crash the audit.

## Policy Input

The first-class input is `agent/boundaries.toml`.

Supported sections:

- `[rust]`
  - `domain_paths`
  - `forbidden_domain_imports`
- `[typescript]`
  - `web_paths`
  - `forbidden_web_imports`
  - `generated_contract_paths`
- `[python]`
  - `allowed_truth_paths`
- `[db]`
  - `root_paths`
  - `migration_paths`
  - `constraint_paths`
- `[queues]`
  - `adapter_paths`
  - `event_contract_paths`
  - `generated_type_paths`
  - `client_markers`
- `[[streaming_exception]]`
  - `runtime`
  - `classification` or `reason`
  - `owner`
  - `expires`
  - `migration_path`

## Direct DB Access

Rule: `HLT-006-DIRECT-DB-WRONG-LAYER`

Direct DB access is allowed only in DB roots, migrations, constraints, or adapter paths. The oracle combines path evidence with import/content evidence such as `sqlx::`, `diesel::`, `psycopg`, `sqlalchemy`, `better-sqlite3`, `pg`, or raw SQL statement markers.

Findings tell agents to move durable writes and SQL clients behind DB adapters or application-owned transactions.

## Rust Domain Purity

Rule: `HLT-022-RUST-DOMAIN-IMPURITY`

Rust domain paths must remain deterministic and side-effect free. Forbidden domain markers include filesystem, env, network, clock, random, database, queue, tracing, and logging imports.

Repair direction:

1. Move side effects to `crates/adapters` or application ports.
2. Keep domain modules to IDs, invariants, value objects, state machines, and pure decisions.
3. Rerun the `fast` lane after extraction.

## TypeScript Web Boundary

Rule: `HLT-023-TS-WEB-DURABLE-TRUTH`

TypeScript web paths may own UI, forms, local rendering state, and generated clients. They must not import backend internals, server modules, DB adapters, storage clients, or durable truth code.

Repair direction:

1. Replace backend/internal imports with generated contract clients.
2. Move durable truth to Rust application/domain and SQL constraints.
3. Rerun contract/web proof lanes.

## Python Product Truth Containment

Rule: `HLT-005-PYTHON-PRODUCT-TRUTH`

Python is bounded to AI/data/eval tooling and must not own product authorization, durable product truth, or direct production DB writes. The oracle combines path evidence with DB/auth/product markers so generic Python tooling does not create noise.

Repair direction:

1. Move durable product behavior into Rust/SQL.
2. Keep Python behind typed generated contracts.
3. Use Python for models, embeddings, evals, transforms, and offline analysis only.

## Streaming Runtime Boundary

Rule: `HLT-019-STREAMING-RUNTIME-DRIFT`

Kafka, Tansu, Iggy, Fluvio, NATS, Redis Streams, and similar clients must live behind declared queue adapters. Brownfield exceptions are allowed only when they are actionable.

Every streaming exception must include:

- runtime
- owner
- future ISO `expires`
- brownfield `classification` or `reason`
- migration path

Expired or incomplete exceptions are high-severity findings because they become hidden architecture.

## False-Positive Allowlist

The oracle skips generated files and non-runtime material under:

- `docs/`
- `paper/`
- `reference/`
- `tips/`
- `vendor/`
- `node_modules/`
- `target/`
- tests and fixtures

Generated-zone enforcement belongs to the generated-zone oracle, not this semantic boundary oracle.
''')

### What this finishes for Phase 02

This makes **Phase 02** materially complete by converting the partial implementation into a real semantic-oracle slice:

1. **Rule registry is now the rule metadata contract.**
   It includes stable IDs, category, TLR, severity, lane, owner hint, docs URL, evidence kind, repairability, confidence convention, and legacy cap mapping.

2. **Boundary manifest is first-class.**
   `agent/boundaries.toml` now parses Rust, TypeScript, Python, DB, queue, and streaming exception sections instead of only stack/queues/DB.

3. **Semantic oracle is separate from dimension scoring.**
   `audit/analyzers/semantic.rs` emits structured findings; `audit/mod.rs` only routes them through `FindingBuilder`.

4. **Existing report shape is preserved.**
   No required JSON field is removed or renamed. Existing renderers should keep working.

5. **Findings become more actionable.**
   Semantic findings include path, line when available, owner through the owner map, lane from the registry, docs URL, matched term, reason, evidence, and exact repair guidance.

6. **False positives are explicitly bounded.**
   The oracle skips docs, reference, generated, vendor, tests, node modules, target, and Jankurai’s own audit crate unless `self_audit` is enabled.

7. **Streaming exceptions now expire.**
   Kafka/streaming exceptions must carry owner, future ISO expiry, brownfield classification or reason, and migration path.

8. **Rule docs are aligned.**
   `docs/rule-index.md` becomes the human-readable rule index and tests require every registered rule to appear there.
