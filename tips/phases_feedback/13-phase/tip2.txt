Phase name: **Phase 13: Autonomous Repair And Optimization — hardened completion**

I read the moonshot vision and treated Phase 13 as the final, most dangerous payoff stage: mutation is only acceptable when the repo can prove ownership, scope, proof lane, evidence, rollback, and human-review boundaries. The current repo already has dry-run repair plans, fixture-only patching, dry-run PR draft evidence, optimization reports, and exception expiry; the named deferred gap is real repo patch execution plus real branch/commit/PR creation. That is exactly what this solution finishes. The repo’s README describes Jankurai as a control plane for ownership, proof, generated zones, and repair evidence, and `docs/moonshot.md` frames the loop as `intent -> bounded agents -> proof lanes -> evidence -> expired exceptions -> reusable primitives`. ([GitHub][1]) Phase 13’s own file says the current command surface does **not** write real projects, create branches, commit, open PRs, or auto-merge; this patch closes the first four while still refusing auto-merge. 

I could not execute the patch in this sandbox because the repo could not be cloned here, but I did inspect the live GitHub source files and designed this against the current `main` layout. The PR creation path uses `gh pr create --draft --base --head --title --body`; GitHub CLI documents that `gh pr create` prints the PR URL on success and supports those flags, including `--draft`. pub fn run_repo_apply(args: RepairArgs, plan: RepairPlan, max_risk: RepairRisk) -> Result<()> {

    let packets = packet_map(&plan);
    let gate = evaluate_plan_gates(&catalog, &packets, &plan, max_risk);

    if !gate.blocked_packets.is_empty() || plan.packets.is_empty() {

            generated_at: now_string(),
            status: "blocked".to_string(),
            execution_mode: execution_mode(&args).to_string(),
            dry_run: false,
            auto_pr_requested: args.create_pr || args.auto_pr,
            auto_pr_status: if args.create_pr || args.auto_pr {
                "blocked".to_string()
            } else {
                "not-requested".to_string()
            },
            max_risk: max_risk.as_str().to_string(),
            planned_packets: plan.packets.len(),
            risk_summary: gate.risk_summary,
            blocked_packets: gate.blocked_packets,
            applied_edits: Vec::new(),
            skipped_edits: gate.skipped_edits,
            files_written: Vec::new(),
            proof_evidence_index: None,
            auto_pr_draft: None,
            git_mutation: Some(GitMutationSummary {
                mutation_allowed: false,
                branch_name: String::new(),
                base_branch: String::new(),
                base_sha: String::new(),
                head_sha: None,
                commit_sha: None,
                remote: args.remote.clone(),
                created_branch: false,
                committed: false,
                pushed: false,
                rollback_performed: false,
                rollback_command: None,
                working_tree_clean_before: tracked_worktree_is_clean(&args.repo).unwrap_or(false),
                working_tree_clean_after: tracked_worktree_is_clean(&args.repo).unwrap_or(false),
            }),
            pull_request: None,
            proof_lanes: proof_lanes(&plan),
            notes,
        };
        write_run(&args, &run)?;
        return Ok(());
    }

    let preflight = preflight_git(&args.repo, &args);
    if let Err(error) = preflight {
        let reason = error.to_string();
        let run = RepairRun {
            schema_version: "1.0.0".to_string(),
            repo: args.repo.display().to_string(),
            plan: args.plan.clone(),
            generated_at: now_string(),
            status: "failed".to_string(),
            execution_mode: execution_mode(&args).to_string(),
            dry_run: false,
            auto_pr_requested: args.create_pr || args.auto_pr,
            auto_pr_status: if args.create_pr || args.auto_pr {
                "failed".to_string()
            } else {
                "not-requested".to_string()
            },
            max_risk: max_risk.as_str().to_string(),
            planned_packets: plan.packets.len(),
            risk_summary: gate.risk_summary,
            blocked_packets: Vec::new(),
            applied_edits: Vec::new(),
            skipped_edits: Vec::new(),
            files_written: Vec::new(),
            proof_evidence_index: None,
            auto_pr_draft: None,
            git_mutation: Some(GitMutationSummary {
                mutation_allowed: false,
                branch_name: String::new(),
                base_branch: String::new(),
                base_sha: String::new(),
                head_sha: None,
                commit_sha: None,
                remote: args.remote.clone(),
                created_branch: false,
                committed: false,
                pushed: false,
                rollback_performed: false,
                rollback_command: None,
                working_tree_clean_before: false,
                working_tree_clean_after: false,
            }),
            pull_request: None,
            proof_lanes: proof_lanes(&plan),
            notes: vec![
                "real repair apply failed git preflight before mutation".to_string(),
                reason,
            ],
        };
        write_run(&args, &run)?;
        bail!("real repair git preflight failed; see repair run receipt");
    }

    let original_branch = git_stdout(&args.repo, &["symbolic-ref", "--quiet", "--short", "HEAD"])
        .context("determine current git branch")?;
    let base_branch = args
        .base_branch
        .clone()
        .unwrap_or_else(|| original_branch.clone());
    let base_sha =
        git_stdout(&args.repo, &["rev-parse", base_branch.as_str()]).context("resolve base branch")?;
    let branch_name = repair_branch_name(&args.branch_prefix, &plan);

    if branch_exists(&args.repo, &branch_name)? {
        bail!("refusing to reuse existing repair branch `{branch_name}`");
    }

    let mut applied_edits = Vec::new();
    let mut skipped_edits = Vec::new();
    let mut files_written = Vec::new();
    let mut proof_evidence_index: Option<String> = None;
    let mut commit_sha: Option<String> = None;
    let mut pushed = false;
    let mut rollback_performed = false;
    let mut pull_request: Option<PullRequestSummary> = None;
    let mut terminal_error: Option<anyhow::Error> = None;

    git(&args.repo, &["checkout", "-b", branch_name.as_str()]).context("create repair branch")?;

    for edit in &plan.planned_edits {
        match apply_planned_edit_real(&args.repo, &catalog, &packets, edit, max_risk) {
            Ok(EditOutcome::Applied(applied)) => {
                push_unique(&mut files_written, applied.path.clone());
                applied_edits.push(applied);
            }
            Ok(EditOutcome::Skipped(skipped)) => skipped_edits.push(skipped),
            Err(error) => {
                terminal_error = Some(error);
                break;
            }
        }
    }

    if terminal_error.is_none() && (!skipped_edits.is_empty() || applied_edits.is_empty()) {
        terminal_error = Some(anyhow::anyhow!(
            "real repair apply did not apply every planned edit"
        ));
    }

    if terminal_error.is_none() {
        let evidence_path = args.repo.join("target/jankurai/p13-real-evidence-index.json");
        let proof_args = proof::ProveArgs {
            repo: args.repo.clone(),
            plan: None,
            changed: files_written.iter().map(PathBuf::from).collect(),
            changed_from: None,
            plan_out: args
                .repo
                .join("target/jankurai/p13-real-proof-plan.json")
                .display()
                .to_string(),
            plan_md: args
                .repo
                .join("target/jankurai/p13-real-proof-plan.md")
                .display()
                .to_string(),
            out_dir: args
                .repo
                .join("target/jankurai/p13-real-proof-receipts")
                .display()
                .to_string(),
            evidence_index: evidence_path.display().to_string(),
            continue_on_error: false,
            allow_unsigned_commands: false,
        };
        if let Err(error) = proof::run_prove(proof_args) {
            terminal_error = Some(error.context("real repair proof verification failed"));
        }
        proof_evidence_index = Some(relative_repo_path(&args.repo, &evidence_path));
    }

    if terminal_error.is_some() {
        rollback_performed = rollback_branch(&args.repo, &base_sha, &files_written).unwrap_or(false);
        let _ = git(&args.repo, &["checkout", original_branch.as_str()]);
    }

        if let Err(error) = commit_repair(&args.repo, &files_written, &plan, &applied_edits) {
            terminal_error = Some(error.context("commit real repair branch"));
        } else {
            commit_sha = git_stdout(&args.repo, &["rev-parse", "HEAD"]).ok();
        }
    }

        "blocked".to_string()
    } else {
        "not-requested".to_string()
    };

        let pr_body = render_pr_body(
            &args,
            &plan,
            &branch_name,
            &base_branch,
            commit_sha.as_deref(),
            proof_evidence_index.as_deref(),
            &applied_edits,
        );

        match git(
            &args.repo,
            &[
                "push",
                "--set-upstream",
                args.remote.as_str(),
                branch_name.as_str(),
            ],
        ) {

                            status: "created".to_string(),
                            draft: true,
                            url: Some(url),
                            remote: args.remote.clone(),
                            base_branch: base_branch.clone(),
                            head_branch: branch_name.clone(),
                        });
                    }
                    Err(error) => {
                        auto_pr_status = "failed".to_string();
                        pull_request = Some(PullRequestSummary {
                            status: "failed".to_string(),
                            draft: true,
                            url: None,
                            remote: args.remote.clone(),
                            base_branch: base_branch.clone(),
                            head_branch: branch_name.clone(),
                        });
                        terminal_error = Some(error.context("create draft GitHub PR"));
                    }
                }
            }
            Err(error) => {
                auto_pr_status = "failed".to_string();
                pull_request = Some(PullRequestSummary {
                    status: "failed".to_string(),
                    draft: true,
                    url: None,
                    remote: args.remote.clone(),
                    base_branch: base_branch.clone(),
                    head_branch: branch_name.clone(),
                });
                terminal_error = Some(error.context("push repair branch before PR creation"));
            }
        }
    } else if terminal_error.is_none() && (args.create_pr || args.auto_pr) {
        auto_pr_status = "created".to_string();
    }

    let status = if terminal_error.is_some() {
        "failed"
    } else if !skipped_edits.is_empty() || applied_edits.is_empty() {
        "blocked"
    } else {
        "complete"
    };

    let head_sha = git_stdout(&args.repo, &["rev-parse", "HEAD"]).ok();
    let working_tree_clean_after = tracked_worktree_is_clean(&args.repo).unwrap_or(false);
    let notes = real_apply_notes(
        &args,
        &applied_edits,
        &skipped_edits,
        proof_evidence_index.as_deref(),
        commit_sha.as_deref(),
        rollback_performed,
        terminal_error.as_ref().map(|error| error.to_string()),
    );

    let run = RepairRun {
        schema_version: "1.0.0".to_string(),
        repo: args.repo.display().to_string(),
        plan: args.plan.clone(),
        generated_at: now_string(),
        status: status.to_string(),
        execution_mode: execution_mode(&args).to_string(),
        dry_run: false,
        auto_pr_requested: args.create_pr || args.auto_pr,
        auto_pr_status,
        max_risk: max_risk.as_str().to_string(),
        planned_packets: plan.packets.len(),
        risk_summary: gate.risk_summary,
        blocked_packets: Vec::new(),
        applied_edits,
        skipped_edits,
        files_written,
        proof_evidence_index,
        auto_pr_draft: None,
        git_mutation: Some(GitMutationSummary {
            mutation_allowed: true,
            branch_name,
            base_branch,
            base_sha,
            head_sha,
            commit_sha,
            remote: args.remote.clone(),
            created_branch: true,
            committed: commit_sha.is_some(),
            pushed,
            rollback_performed,
            rollback_command: if rollback_performed {
                Some(format!("git reset --hard {}", base_sha))
            } else {
                None
            },
            working_tree_clean_before: true,
            working_tree_clean_after,
        }),
        pull_request,
        proof_lanes: proof_lanes(&plan),
        notes,
    };

    write_run(&args, &run)?;

    if let Some(error) = terminal_error {
        return Err(error);
    }

    Ok(())
}

fn evaluate_plan_gates(
    catalog: &RepoCatalog,
    packets: &HashMap<String, RepairPacket>,
    plan: &RepairPlan,
    max_risk: RepairRisk,
) -> GateResult {
    let edit_map = planned_edit_map(plan);
    let mut risk_summary = RiskSummary::default();
    let mut blocked_packets = Vec::new();
    let mut skipped_edits = Vec::new();

    for packet in &plan.packets {
        let risk = packet_risk(packet);
        let eligibility = packet_eligibility(packet);
        increment_risk_summary(&mut risk_summary, risk);

        let mut reasons = Vec::new();
        if !stable_fingerprint(&packet.finding_fingerprint) {
            reasons.push("finding fingerprint must be a stable sha256:<64 hex> value".to_string());
        }
        if !risk.is_allowed_by(max_risk) {
            reasons.push(format!(
                "risk {} exceeds max {}",
                risk.as_str(),
                max_risk.as_str()
            ));
        }
        if packet.human_review_required {
            reasons.push("packet requires human review".to_string());
        }
        if !matches!(
            eligibility,
            RepairEligibility::AutoSafe | RepairEligibility::AgentAssisted
        ) {
            reasons.push(format!("repair eligibility is {}", eligibility.as_str()));
        }
        if packet.allowed_paths.is_empty() {
            reasons.push("allowed_paths must be explicit".to_string());
        }
        if packet.forbidden_paths.is_empty() {
            reasons.push("forbidden_paths must be explicit".to_string());
        }
        if packet.required_proof.is_empty() && plan.planned_commands.is_empty() {
            reasons.push("proof commands must be explicit".to_string());
        }
        if packet.rollback_guidance.trim().is_empty() || plan.rollback_guidance.is_empty() {
            reasons.push("rollback guidance must be explicit".to_string());
        }

        let Some(edit) = edit_map.get(&packet.finding_fingerprint) else {
            reasons.push("no planned edit matches finding_fingerprint".to_string());
            blocked_packets.push(blocked_packet(packet, risk, eligibility, reasons.join("; ")));
            continue;
        };

        let path = match normalize_edit_path(&edit.path) {
            Ok(path) => path,
            Err(error) => {
                reasons.push(error.to_string());
                String::new()
            }
        };
        if !path.is_empty() {
            if normalize_edit_path(&packet.finding_path)
                .map(|packet_path| packet_path != path)
                .unwrap_or(true)
            {
                reasons.push("planned edit path does not match packet finding_path".to_string());
            }
            if !path_allowed(&path, &packet.allowed_paths) {
                reasons.push("path is outside packet allowed_paths".to_string());
            }
            if path_forbidden(&path, &packet.forbidden_paths) {
                reasons.push("path is in packet forbidden_paths".to_string());
            }
            if is_repo_forbidden_path(&path) {
                reasons.push("path is in a forbidden repository zone".to_string());
            }
            if catalog
                .generated_zones
                .iter()
                .any(|zone| path_matches(&path, &zone.path))
            {
                reasons.push("path is in a generated zone".to_string());
            }
        }
        if !matches!(
            edit.apply_strategy.as_str(),
            "append-text" | "replace-exact" | "create-file"
        ) {
            reasons.push(format!(
                "apply_strategy `{}` is not writable in real apply mode",
                edit.apply_strategy
            ));
        }

        if !reasons.is_empty() {
            blocked_packets.push(blocked_packet(packet, risk, eligibility, reasons.join("; ")));
        }
    }

    for edit in &plan.planned_edits {
        if !packets.contains_key(&edit.finding_fingerprint) {
            skipped_edits.push(SkippedEdit {
                finding_fingerprint: edit.finding_fingerprint.clone(),
                path: edit.path.clone(),
                reason: "planned edit has no matching packet".to_string(),
            });
        }
    }

    GateResult {
        risk_summary,
        blocked_packets,
        skipped_edits,
    }
}

fn blocked_packet(
    packet: &RepairPacket,
    risk: RepairRisk,
    eligibility: RepairEligibility,
    reason: String,
) -> BlockedPacket {
    BlockedPacket {
        finding_fingerprint: packet.finding_fingerprint.clone(),
        rule_id: packet.rule_id.clone(),
        risk_level: risk.as_str().to_string(),
        repair_eligibility: eligibility.as_str().to_string(),
        reason,
    }
}

fn apply_planned_edit_real(
    repo: &Path,
    catalog: &RepoCatalog,
    packets: &HashMap<String, RepairPacket>,
    edit: &PlannedEdit,
    max_risk: RepairRisk,
) -> Result<EditOutcome> {
    let path = normalize_edit_path(&edit.path)?;
    let packet = match packets.get(&edit.finding_fingerprint) {
        Some(packet) => packet,
        None => {
            return Ok(EditOutcome::Skipped(SkippedEdit {
                finding_fingerprint: edit.finding_fingerprint.clone(),
                path,
                reason: "no packet matches finding_fingerprint".to_string(),
            }));
        }
    };

    if normalize_edit_path(&packet.finding_path)? != path {
        return Ok(EditOutcome::Skipped(SkippedEdit {
            finding_fingerprint: edit.finding_fingerprint.clone(),
            path,
            reason: "planned edit path does not match packet finding_path".to_string(),
        }));
    }
    if !path_allowed(&path, &packet.allowed_paths) {
        return Ok(EditOutcome::Skipped(SkippedEdit {
            finding_fingerprint: edit.finding_fingerprint.clone(),
            path,
            reason: "path is outside packet allowed_paths".to_string(),
        }));
    }
    if path_forbidden(&path, &packet.forbidden_paths) || is_repo_forbidden_path(&path) {
        return Ok(EditOutcome::Skipped(SkippedEdit {
            finding_fingerprint: edit.finding_fingerprint.clone(),
            path,
            reason: "path is forbidden for real apply".to_string(),
        }));
    }
    if catalog
        .generated_zones
        .iter()
        .any(|zone| path_matches(&path, &zone.path))
    {
        return Ok(EditOutcome::Skipped(SkippedEdit {
            finding_fingerprint: edit.finding_fingerprint.clone(),
            path,
            reason: "path is in a generated zone".to_string(),
        }));
    }

    let risk = packet_risk(packet);
    let eligibility = packet_eligibility(packet);
    if !risk.is_allowed_by(max_risk)
        || !matches!(
            eligibility,
            RepairEligibility::AutoSafe | RepairEligibility::AgentAssisted
        )
        || packet.human_review_required
    {
        return Ok(EditOutcome::Skipped(SkippedEdit {
            finding_fingerprint: edit.finding_fingerprint.clone(),
            path,
            reason: "packet is not eligible for real apply".to_string(),
        }));
    }

    match edit.apply_strategy.as_str() {
        "append-text" => apply_append_text(repo, edit, &path),
        "replace-exact" => apply_replace_exact(repo, edit, &path),
        "create-file" => apply_create_file(repo, edit, &path),
        other => Ok(EditOutcome::Skipped(SkippedEdit {
            finding_fingerprint: edit.finding_fingerprint.clone(),
            path,
            reason: format!("apply_strategy `{other}` is not writable in real apply mode"),
        })),
    }
}

fn apply_append_text(repo: &Path, edit: &PlannedEdit, path: &str) -> Result<EditOutcome> {
    let append_text = match edit.append_text.as_deref() {
        Some(text) => text,
        None => {
            return Ok(EditOutcome::Skipped(SkippedEdit {
                finding_fingerprint: edit.finding_fingerprint.clone(),
                path: path.to_string(),
                reason: "append-text strategy requires append_text".to_string(),
            }));
        }
    };

    require_no_symlink_components(repo, path, false)?;
    require_tracked_file(repo, path)?;

    let file_path = repo.join(path);
    let mut content = read_utf8(&file_path, path)?;
    let before_sha256 = sha256_text(&content);
    content.push_str(append_text);
    write_text(&file_path, &content)?;
    let after_sha256 = sha256_text(&content);

    Ok(EditOutcome::Applied(AppliedEdit {
        finding_fingerprint: edit.finding_fingerprint.clone(),
        path: path.to_string(),
        apply_strategy: edit.apply_strategy.clone(),
        before_sha256,
        after_sha256,
        status: "applied".to_string(),
    }))
}

fn apply_replace_exact(repo: &Path, edit: &PlannedEdit, path: &str) -> Result<EditOutcome> {
    let match_text = match edit.match_text.as_deref() {
        Some(text) => text,
        None => {
            return Ok(EditOutcome::Skipped(SkippedEdit {
                finding_fingerprint: edit.finding_fingerprint.clone(),
                path: path.to_string(),
                reason: "replace-exact strategy requires match_text".to_string(),
            }));
        }
    };
    let replacement_text = match edit.replacement_text.as_deref() {
        Some(text) => text,
        None => {
            return Ok(EditOutcome::Skipped(SkippedEdit {
                finding_fingerprint: edit.finding_fingerprint.clone(),
                path: path.to_string(),
                reason: "replace-exact strategy requires replacement_text".to_string(),
            }));
        }
    };

    require_no_symlink_components(repo, path, false)?;
    require_tracked_file(repo, path)?;

    let file_path = repo.join(path);
    let content = read_utf8(&file_path, path)?;
    let before_sha256 = sha256_text(&content);
    let occurrences = content.matches(match_text).count();
    if occurrences != 1 {
        return Ok(EditOutcome::Skipped(SkippedEdit {
            finding_fingerprint: edit.finding_fingerprint.clone(),
            path: path.to_string(),
            reason: format!("replace-exact expected exactly one match, found {occurrences}"),
        }));
    }
    let updated = content.replacen(match_text, replacement_text, 1);
    write_text(&file_path, &updated)?;
    let after_sha256 = sha256_text(&updated);

    Ok(EditOutcome::Applied(AppliedEdit {
        finding_fingerprint: edit.finding_fingerprint.clone(),
        path: path.to_string(),
        apply_strategy: edit.apply_strategy.clone(),
        before_sha256,
        after_sha256,
        status: "applied".to_string(),
    }))
}

fn apply_create_file(repo: &Path, edit: &PlannedEdit, path: &str) -> Result<EditOutcome> {
    let create_text = match edit.create_text.as_deref() {
        Some(text) => text,
        None => {
            return Ok(EditOutcome::Skipped(SkippedEdit {
                finding_fingerprint: edit.finding_fingerprint.clone(),
                path: path.to_string(),
                reason: "create-file strategy requires create_text".to_string(),
            }));
        }
    };

    require_no_symlink_components(repo, path, true)?;

    let file_path = repo.join(path);
    if file_path.exists() {
        return Ok(EditOutcome::Skipped(SkippedEdit {
            finding_fingerprint: edit.finding_fingerprint.clone(),
            path: path.to_string(),
            reason: "target file already exists for create-file".to_string(),
        }));
    }
    if let Some(parent) = file_path.parent() {
        fs::create_dir_all(parent)?;
    }
    write_text(&file_path, create_text)?;

    Ok(EditOutcome::Applied(AppliedEdit {
        finding_fingerprint: edit.finding_fingerprint.clone(),
        path: path.to_string(),
        apply_strategy: edit.apply_strategy.clone(),
        before_sha256: "missing".to_string(),
        after_sha256: sha256_text(create_text),
        status: "applied".to_string(),
    }))
}

fn preflight_git(repo: &Path, args: &RepairArgs) -> Result<()> {
    git_stdout(repo, &["rev-parse", "--is-inside-work-tree"]).context("not a git repository")?;
    let current_branch =
        git_stdout(repo, &["symbolic-ref", "--quiet", "--short", "HEAD"]).context("detached HEAD")?;
    let base_branch = args
        .base_branch
        .clone()
        .unwrap_or_else(|| current_branch.clone());
    git_stdout(repo, &["rev-parse", base_branch.as_str()])
        .with_context(|| format!("base branch `{base_branch}` does not resolve"))?;
    if !tracked_worktree_is_clean(repo)? {
        bail!("working tree has tracked or non-target untracked changes; commit/stash them before real repair apply");
    }
    Ok(())
}

fn tracked_worktree_is_clean(repo: &Path) -> Result<bool> {
    let output = Command::new("git")
        .args(["status", "--porcelain", "--untracked-files=all"])
        .current_dir(repo)
        .output()
        .context("git status")?;
    if !output.status.success() {
        return Ok(false);
    }
    let status = String::from_utf8_lossy(&output.stdout);
    Ok(status
        .lines()
        .all(|line| line.trim().is_empty() || line.starts_with("?? target/")))
}

fn require_tracked_file(repo: &Path, path: &str) -> Result<()> {
    let output = Command::new("git")
        .args(["ls-files", "--error-unmatch", "--", path])
        .current_dir(repo)
        .output()
        .with_context(|| format!("git ls-files {path}"))?;
    if !output.status.success() {
        bail!("real apply may only modify tracked files for append-text/replace-exact: `{path}`");
    }
    Ok(())
}

fn commit_repair(
    repo: &Path,
    files_written: &[String],
    plan: &RepairPlan,
    applied_edits: &[AppliedEdit],
) -> Result<()> {
    let mut add_args = vec!["add".to_string(), "--".to_string()];
    add_args.extend(files_written.iter().cloned());
    git_owned(repo, add_args).context("stage repair files")?;

    let title = commit_title(applied_edits);
    let body = format!(
        "Jankurai Phase 13 real repair apply\n\nsource_plan: {}\napplied_edits: {}\nproof: target/jankurai/p13-real-evidence-index.json\n\nThis commit was created only after scoped proof completed. Auto-merge is not supported.",
        plan.source_report,
        applied_edits.len()
    );

    git(
        repo,
        &[
            "-c",
            "user.name=Jankurai Repair",
            "-c",
            "user.email=jankurai-repair@users.noreply.github.com",
            "commit",
            "-m",
            title.as_str(),
            "-m",
            body.as_str(),
        ],
    )?;
    Ok(())
}

fn rollback_branch(repo: &Path, base_sha: &str, files_written: &[String]) -> Result<bool> {
    git(repo, &["reset", "--hard", base_sha])?;
    for path in files_written {
        let _ = git(repo, &["clean", "-fd", "--", path.as_str()]);
    }
    Ok(true)
}

fn gh_pr_create(
    repo: &Path,
    base_branch: &str,
    branch_name: &str,
    pr_title: &str,
    pr_body: &str,
) -> Result<String> {
    let output = Command::new("gh")
        .args([
            "pr",
            "create",
            "--draft",
            "--base",
            base_branch,
            "--head",
            branch_name,
            "--title",
            pr_title,
            "--body",
            pr_body,
        ])
        .current_dir(repo)
        .output()
        .context("run gh pr create")?;
    if !output.status.success() {
        bail!(
            "gh pr create failed: {}{}",
            String::from_utf8_lossy(&output.stderr),
            String::from_utf8_lossy(&output.stdout)
        );
    }
    Ok(String::from_utf8_lossy(&output.stdout).trim().to_string())
}

fn write_run(args: &RepairArgs, run: &RepairRun) -> Result<()> {
    if let Some(path) = args.out.as_deref() {
        validation::write_json(&args.repo, ArtifactSchema::RepairRun, path, run)?;
    } else {
        validation::validate_serializable(&args.repo, ArtifactSchema::RepairRun, run)?;
        println!("{}", serde_json::to_string_pretty(run)?);
    }
    if let Some(path) = args.md.as_deref() {
        crate::render::write_markdown(path, &render_markdown(run))?;
    }
    Ok(())
}

fn real_apply_notes(
    args: &RepairArgs,
    applied_edits: &[AppliedEdit],

) -> Vec<String> {

        "patches were limited to explicit planned edit strategies".to_string(),
        "proof was executed before commit".to_string(),

    ];
    notes.push(format!("applied edits: {}", applied_edits.len()));
    notes.push(format!("skipped edits: {}", skipped_edits.len()));

    }

    }

}

fn packet_map(plan: &RepairPlan) -> HashMap<String, RepairPacket> {
    let mut map = HashMap::new();
    for packet in &plan.packets {
        map.insert(packet.finding_fingerprint.clone(), packet.clone());
    }
    map
}

fn planned_edit_map<'a>(plan: &'a RepairPlan) -> HashMap<String, &'a PlannedEdit> {
    let mut map = HashMap::new();
    for edit in &plan.planned_edits {
        map.insert(edit.finding_fingerprint.clone(), edit);
    }
    map
}

fn normalize_edit_path(path: &str) -> Result<String> {
    if path.trim().is_empty() {
        bail!("edit path must not be empty");
    }
    let candidate = Path::new(path);
    if candidate.is_absolute() {
        bail!("absolute edit paths are not allowed: `{path}`");
    }
    let mut parts = Vec::new();
    for component in candidate.components() {
        match component {
            Component::Normal(part) => parts.push(part.to_string_lossy().to_string()),
            Component::CurDir => {}
            Component::ParentDir | Component::RootDir | Component::Prefix(_) => {
                bail!("path traversal is not allowed: `{path}`");
            }
        }
    }
    let normalized = parts.join("/");
    if normalized.is_empty() {
        bail!("edit path must not be empty");
    }
    Ok(normalized)
}

fn path_allowed(path: &str, allowed_paths: &[String]) -> bool {
    allowed_paths
        .iter()
        .any(|allowed| path_matches(path, allowed))
}

fn path_forbidden(path: &str, forbidden_paths: &[String]) -> bool {
    forbidden_paths
        .iter()
        .any(|forbidden| path_matches(path, forbidden))
}

fn is_repo_forbidden_path(path: &str) -> bool {
    matches_prefix(path, "reference/")
        || matches_prefix(path, "paper/")
        || matches_prefix(path, "target/")
        || path == "reference"
        || path == "paper"
        || path == "target"
}

fn path_matches(path: &str, prefix: &str) -> bool {
    let prefix = prefix.trim_end_matches('/');
    path == prefix || path.starts_with(&format!("{prefix}/"))
}

fn matches_prefix(path: &str, prefix: &str) -> bool {
    let prefix = prefix.trim_end_matches('/');
    path == prefix || path.starts_with(&format!("{prefix}/"))
}

fn stable_fingerprint(fingerprint: &str) -> bool {
    let Some(rest) = fingerprint.strip_prefix("sha256:") else {
        return false;
    };
    rest.len() == 64 && rest.chars().all(|c| c.is_ascii_hexdigit())
}

fn require_no_symlink_components(repo: &Path, rel_path: &str, target_may_not_exist: bool) -> Result<()> {
    let mut current = repo.to_path_buf();
    let parts = Path::new(rel_path)
        .components()
        .filter_map(|component| match component {
            Component::Normal(part) => Some(part.to_os_string()),
            _ => None,
        })
        .collect::<Vec<_>>();

    for (index, part) in parts.iter().enumerate() {
        current.push(part);
        let is_last = index + 1 == parts.len();
        if is_last && target_may_not_exist && !current.exists() {
            break;
        }
        if let Ok(metadata) = fs::symlink_metadata(&current) {
            if metadata.file_type().is_symlink() {
                bail!("real apply refuses symlink path component `{}`", current.display());
            }
        }
    }

    let canonical_repo = fs::canonicalize(repo).context("canonicalize repo root")?;
    let guard_path = if target_may_not_exist {
        repo.join(rel_path)
            .parent()
            .map(Path::to_path_buf)
            .unwrap_or_else(|| repo.to_path_buf())
    } else {
        repo.join(rel_path)
    };
    let canonical_guard = fs::canonicalize(&guard_path)
        .with_context(|| format!("canonicalize repair path {}", guard_path.display()))?;
    if !canonical_guard.starts_with(&canonical_repo) {
        bail!("repair path escapes repository root: `{rel_path}`");
    }
    Ok(())
}

fn read_utf8(file_path: &Path, rel_path: &str) -> Result<String> {
    match fs::read_to_string(file_path) {
        Ok(content) => Ok(content),
        Err(error) if error.kind() == std::io::ErrorKind::NotFound => {
            bail!("target file does not exist: `{rel_path}`")
        }
        Err(error) if error.kind() == std::io::ErrorKind::InvalidData => {
            bail!("target file is not valid UTF-8: `{rel_path}`")
        }
        Err(error) => Err(error.into()),
    }
}

fn write_text(path: &Path, text: &str) -> Result<()> {
    if let Some(parent) = path.parent() {
        fs::create_dir_all(parent)?;
    }
    fs::write(path, text)?;
    Ok(())
}

fn repair_branch_name(prefix: &str, plan: &RepairPlan) -> String {
    let prefix = sanitize_branch_prefix(prefix);
    let mut seed = String::new();
    seed.push_str(&plan.source_report);
    seed.push('|');
    seed.push_str(&plan.target_stack_id);
    seed.push('|');
    for edit in &plan.planned_edits {
        seed.push_str(&edit.finding_fingerprint);
        seed.push('|');
        seed.push_str(&edit.path);
        seed.push('|');
        seed.push_str(&edit.apply_strategy);
        seed.push('|');
    }
    let digest = sha256_hex(&seed);
    format!("{prefix}/{}", &digest[..12])
}

fn sanitize_branch_prefix(prefix: &str) -> String {
    let mut out = prefix
        .chars()
        .map(|c| {
            if c.is_ascii_alphanumeric() || matches!(c, '/' | '-' | '_') {
                c
            } else {
                '-'
            }
        })
        .collect::<String>();
    while out.contains("//") {
        out = out.replace("//", "/");
    }
    while out.contains("..") {
        out = out.replace("..", "-");
    }
    let out = out.trim_matches('/').trim_matches('.').to_string();
    if out.is_empty() {
        "jankurai/repair".to_string()
    } else {
        out
    }
}

What this finishes:

The previous Phase 13 surface could plan repairs, dry-run PR eligibility, write draft PR evidence, apply patches only in fixture repos, optimize advisories, and expire exceptions. This patch adds the missing hardened mutation path: **`jankurai repair --apply`** and **`jankurai repair --apply --create-pr`**.

The important safety shape is intentional:

`--dry-run` stays the default review lane.  
`--fixture-apply` remains the test/sandbox mutation lane.  
`--apply` becomes the real mutation lane, but only after a clean git preflight, stable finding fingerprints, explicit allowed/forbidden path checks, generated-zone refusal, symlink refusal, proof execution, and rollback support.  
`--create-pr` is draft-only and requires the real apply path to pass proof and commit first.  
There is still no auto-merge path.

The main hardened behavior is:

```bash
# Real local branch + patch + proof + commit, no push or PR.
cargo run -p jankurai -- repair . \
  --plan target/jankurai/repair-plan.json \
  --apply \
  --max-risk medium \
  --out target/jankurai/repair-run.json \
  --md target/jankurai/repair-run.md

# Real branch + patch + proof + commit + push + draft PR through gh.
cargo run -p jankurai -- repair . \
  --plan target/jankurai/repair-plan.json \
  --apply \
  --create-pr \
  --max-risk medium \
  --remote origin \
  --out target/jankurai/repair-run.json \
  --md target/jankurai/repair-run.md
````

The PR path requires the GitHub CLI to be authenticated. It creates a **draft** PR, records the returned URL in `repair-run.json`, and does not merge anything.
