# Phase 13: Autonomous Repair And Optimization — hardened completion

I would treat **Phase 13** as the selected phase, and the hardening target is very clear: the current repo already has the conservative substrate, but it still stops short of the moonshot payoff. The moonshot says Jankurai’s operating loop is `intent -> bounded agents -> proof lanes -> evidence -> expired exceptions -> reusable primitives`, and its requirements include explicit intent, bounded authority, path ownership, proof coverage, generated-source discipline, durable evidence, expiring exceptions, and reusable primitives.  The public repo framing matches that: Jankurai is a control plane for agent-native repositories, not just a validator. ([GitHub][1])

The current Phase 13 doc says the implemented surface is deliberately conservative: dry-run repair planning, fixture-only bounded patch execution, dry-run PR evidence, optimization reports, and exception expiry; it explicitly defers real repo patch execution and real auto-PR creation.  The Phase 00 router also records Phase 13 as `complete-with-deferred-work` specifically because “real repo mutation and PR creation” are still deferred.  The current code confirms that: `repair.rs` rejects real execution unless `--fixture-apply` is used, `repair_apply.rs` only applies patches in fixture repos with `agent/repair-fixture.toml`, and `repair_pr.rs` only emits draft-only PR evidence with git/GitHub mutation flags set to false.   

So the right Phase 13 hardening is **not** “let the agent write freely.” It is a gated real mutation path that reuses the existing packet/risk/proof machinery, refuses unsafe work before touching files, runs proof after writes, only commits after proof, and only creates a **draft** PR behind explicit environment and CLI gates. That keeps it aligned with the standard’s prime directive: reject wrong code fast, localize failure, repair narrow scope, and leave evidence.

+            Ok(path) => proof_evidence_index = Some(path),
+            Err(error) => {
+                proof_evidence_index = Some("target/jankurai/p13-repair-evidence-index.json".to_string());
+                terminal_error = Some(error.context("real repair proof verification failed"));
+                auto_pr_status = if args.auto_pr {
+                    "failed".to_string()
+                } else {
+                    "not-requested".to_string()
+                };
+            }
+        }
+    }
+
+    if terminal_error.is_none() && args.git_commit && !files_written.is_empty() {
+        if let Some(receipt) = git_receipt.as_mut() {
+            let title = auto_pr_draft
+                .as_ref()
+                .map(|draft| draft.commit_title.clone())
+                .unwrap_or_else(|| "Jankurai repair apply".to_string());
+            match commit_paths(&args.repo, &files_written, &title) {
+                Ok(commit_sha) => {
+                    receipt.status = "committed".to_string();
+                    receipt.commit_sha = Some(commit_sha);
+                    auto_pr_status = "committed".to_string();
+                }
+                Err(error) => {
+                    terminal_error = Some(error.context("commit repair changes"));
+                    receipt.status = "failed".to_string();
+                    auto_pr_status = "failed".to_string();
+                }
+            }
+        }
+    }
+
+    if terminal_error.is_none() && args.github_pr {
+        if let Some(receipt) = git_receipt.as_mut() {
+            let title = auto_pr_draft
+                .as_ref()
+                .map(|draft| draft.pr_title.clone())
+                .unwrap_or_else(|| "Jankurai repair draft".to_string());
+            let body = auto_pr_draft
+                .as_ref()
+                .map(render_auto_pr_body)
+                .unwrap_or_else(|| "Jankurai repair draft PR".to_string());
+            if let Err(error) = push_branch(&args.repo, &args.remote, &receipt.branch_name) {
+                terminal_error = Some(error.context("push repair branch"));
+                receipt.status = "failed".to_string();
+                auto_pr_status = "failed".to_string();
+            } else {
+                receipt.pushed = true;
+                match create_github_pr(&args.repo, &args.base, &title, &body) {
+                    Ok(url) => {
+                        receipt.status = "draft-pr-created".to_string();
+                        receipt.pr_url = Some(url);
+                        receipt.draft_pr_created = true;
+                        auto_pr_status = "draft-pr-created".to_string();
+                    }
+                    Err(error) => {
+                        terminal_error = Some(error.context("create GitHub draft PR"));
+                        receipt.status = "failed".to_string();
+                        auto_pr_status = "failed".to_string();
+                    }
+                }
+            }
+        }
+    }
+
+    let status = if terminal_error.is_some() {
+        "failed"
+    } else if blocked_before_write || !skipped_edits.is_empty() || applied_edits.is_empty() {
+        "blocked"
+    } else {
+        "complete"
+    };
+    if let Some(receipt) = git_receipt.as_mut() {
+        if status == "blocked" && receipt.status == "planned" {
+            receipt.status = "blocked".to_string();
+        }
+        if status == "complete" && receipt.status == "planned" {
+            receipt.status = "not-requested".to_string();
+        }
+    }
+
+    let run = RepairRun {
+        schema_version: "1.0.0".to_string(),
+        repo: args.repo.display().to_string(),
+        plan: args.plan.clone(),
+        generated_at: now_string(),
+        status: status.to_string(),
+        execution_mode: "apply".to_string(),
+        dry_run: false,
+        auto_pr_requested: args.auto_pr,
+        auto_pr_status,
+        max_risk: max_risk.as_str().to_string(),
+        planned_packets: plan.packets.len(),
+        risk_summary,
+        blocked_packets,
+        applied_edits,
+        skipped_edits,
+        files_written,
+        proof_evidence_index,
+        auto_pr_draft,
+        git_receipt,
+        proof_lanes: proof_lanes(&plan),
+        notes: apply_notes(status, blocked_before_write, terminal_error.is_some()),
+    };
+
+    write_run(&args, &run)?;
+
+    if let Some(error) = terminal_error {
+        return Err(error);
+    }
+    Ok(())
+}

+fn preflight_planned_edit(
+    repo: &Path,
+    catalog: &RepoCatalog,
+    packets: &HashMap<String, RepairPacket>,
+    edit: &PlannedEdit,
+    max_risk: RepairRisk,
+) -> Result<Option<SkippedEdit>> {
+    let path = normalize_edit_path(&edit.path)?;
+    let Some(packet) = packets.get(&edit.finding_fingerprint) else {
+        return Ok(Some(SkippedEdit {
+            finding_fingerprint: edit.finding_fingerprint.clone(),
+            path,
+            reason: "no packet matches finding_fingerprint".to_string(),
+        }));
+    };
+    if let Some(skipped) = guard_packet_path_and_policy(repo, catalog, packet, edit, &path, max_risk)? {
+        return Ok(Some(skipped));
+    }
+    match edit.apply_strategy.as_str() {
+        "append-text" => {
+            let append_text = match edit.append_text.as_deref() {
+                Some(text) if !text.is_empty() => text,
+                _ => {
+                    return Ok(Some(SkippedEdit {
+                        finding_fingerprint: edit.finding_fingerprint.clone(),
+                        path,
+                        reason: "append-text strategy requires non-empty append_text".to_string(),
+                    }))
+                }
+            };
+            let file_path = repo.join(&path);
+            match fs::read_to_string(&file_path) {
+                Ok(_) => {}
+                Err(error) if error.kind() == std::io::ErrorKind::NotFound => {
+                    return Ok(Some(SkippedEdit {
+                        finding_fingerprint: edit.finding_fingerprint.clone(),
+                        path,
+                        reason: "target file does not exist for append-text".to_string(),
+                    }))
+                }
+                Err(error) if error.kind() == std::io::ErrorKind::InvalidData => {
+                    return Ok(Some(SkippedEdit {
+                        finding_fingerprint: edit.finding_fingerprint.clone(),
+                        path,
+                        reason: "target file is not valid UTF-8".to_string(),
+                    }))
+                }
+                Err(error) => return Err(error.into()),
+            }
+            if append_text.is_empty() {
+                return Ok(Some(SkippedEdit {
+                    finding_fingerprint: edit.finding_fingerprint.clone(),
+                    path,
+                    reason: "append_text would not change the file".to_string(),
+                }));
+            }
+        }
+        "replace-exact" => {
+            let match_text = match edit.match_text.as_deref() {
+                Some(text) if !text.is_empty() => text,
+                _ => {
+                    return Ok(Some(SkippedEdit {
+                        finding_fingerprint: edit.finding_fingerprint.clone(),
+                        path,
+                        reason: "replace-exact strategy requires non-empty match_text".to_string(),
+                    }))
+                }
+            };
+            let replacement_text = match edit.replacement_text.as_deref() {
+                Some(text) => text,
+                None => {
+                    return Ok(Some(SkippedEdit {
+                        finding_fingerprint: edit.finding_fingerprint.clone(),
+                        path,
+                        reason: "replace-exact strategy requires replacement_text".to_string(),
+                    }))
+                }
+            };
+            if match_text == replacement_text {
+                return Ok(Some(SkippedEdit {
+                    finding_fingerprint: edit.finding_fingerprint.clone(),
+                    path,
+                    reason: "replacement_text is identical to match_text".to_string(),
+                }));
+            }
+            let content = match fs::read_to_string(repo.join(&path)) {
+                Ok(content) => content,
+                Err(error) if error.kind() == std::io::ErrorKind::NotFound => {
+                    return Ok(Some(SkippedEdit {
+                        finding_fingerprint: edit.finding_fingerprint.clone(),
+                        path,
+                        reason: "target file does not exist for replace-exact".to_string(),
+                    }))
+                }
+                Err(error) if error.kind() == std::io::ErrorKind::InvalidData => {
+                    return Ok(Some(SkippedEdit {
+                        finding_fingerprint: edit.finding_fingerprint.clone(),
+                        path,
+                        reason: "target file is not valid UTF-8".to_string(),
+                    }))
+                }
+                Err(error) => return Err(error.into()),
+            };
+            let occurrences = content.matches(match_text).count();
+            if occurrences != 1 {
+                return Ok(Some(SkippedEdit {
+                    finding_fingerprint: edit.finding_fingerprint.clone(),
+                    path,
+                    reason: format!("replace-exact expected exactly one match, found {occurrences}"),
+                }));
+            }
+        }
+        "create-file" => {
+            if edit.create_text.is_none() {
+                return Ok(Some(SkippedEdit {
+                    finding_fingerprint: edit.finding_fingerprint.clone(),
+                    path,
+                    reason: "create-file strategy requires create_text".to_string(),
+                }));
+            }
+            if repo.join(&path).exists() {
+                return Ok(Some(SkippedEdit {
+                    finding_fingerprint: edit.finding_fingerprint.clone(),
+                    path,
+                    reason: "target file already exists for create-file".to_string(),
+                }));
+            }
+        }
+        "none" => {
+            return Ok(Some(SkippedEdit {
+                finding_fingerprint: edit.finding_fingerprint.clone(),
+                path,
+                reason: "apply_strategy is none".to_string(),
+            }))
+        }
+        "review-only" | "regenerate" => {
+            return Ok(Some(SkippedEdit {
+                finding_fingerprint: edit.finding_fingerprint.clone(),
+                path,
+                reason: format!("apply_strategy `{}` is not writable in real apply mode", edit.apply_strategy),
+            }))
+        }
+        other => {
+            return Ok(Some(SkippedEdit {
+                finding_fingerprint: edit.finding_fingerprint.clone(),
+                path,
+                reason: format!("unknown apply_strategy `{other}`"),
+            }))
+        }
+    }
+    Ok(None)
+}

+fn apply_planned_edit(
+    repo: &Path,
+    catalog: &RepoCatalog,
+    packets: &HashMap<String, RepairPacket>,
+    edit: &PlannedEdit,
+    max_risk: RepairRisk,
+) -> Result<EditOutcome> {
+    let path = normalize_edit_path(&edit.path)?;
+    let Some(packet) = packets.get(&edit.finding_fingerprint) else {
+        return Ok(EditOutcome::Skipped(SkippedEdit {
+            finding_fingerprint: edit.finding_fingerprint.clone(),
+            path,
+            reason: "no packet matches finding_fingerprint".to_string(),
+        }));
+    };
+    if let Some(skipped) = guard_packet_path_and_policy(repo, catalog, packet, edit, &path, max_risk)? {
+        return Ok(EditOutcome::Skipped(skipped));
+    }
+    match edit.apply_strategy.as_str() {
+        "append-text" => apply_append_text(repo, edit, &path),
+        "replace-exact" => apply_replace_exact(repo, edit, &path),
+        "create-file" => apply_create_file(repo, edit, &path),
+        other => Ok(EditOutcome::Skipped(SkippedEdit {
+            finding_fingerprint: edit.finding_fingerprint.clone(),
+            path,
+            reason: format!("apply_strategy `{other}` is not writable in real apply mode"),
+        })),
+    }
+}
+
+fn guard_packet_path_and_policy(
+    _repo: &Path,
+    catalog: &RepoCatalog,
+    packet: &RepairPacket,
+    edit: &PlannedEdit,
+    path: &str,
+    max_risk: RepairRisk,
+) -> Result<Option<SkippedEdit>> {
+    if normalize_edit_path(&packet.finding_path)? != path {
+        return Ok(Some(SkippedEdit {
+            finding_fingerprint: edit.finding_fingerprint.clone(),
+            path: path.to_string(),
+            reason: "planned edit path does not match packet finding_path".to_string(),
+        }));
+    }
+    if !path_allowed(path, &packet.allowed_paths) {
+        return Ok(Some(SkippedEdit {
+            finding_fingerprint: edit.finding_fingerprint.clone(),
+            path: path.to_string(),
+            reason: "path is outside packet allowed_paths".to_string(),
+        }));
+    }
+    if path_forbidden(path, &packet.forbidden_paths) {
+        return Ok(Some(SkippedEdit {
+            finding_fingerprint: edit.finding_fingerprint.clone(),
+            path: path.to_string(),
+            reason: "path is in packet forbidden_paths".to_string(),
+        }));
+    }
+    if is_repo_forbidden_path(path) {
+        return Ok(Some(SkippedEdit {
+            finding_fingerprint: edit.finding_fingerprint.clone(),
+            path: path.to_string(),
+            reason: "path is in a forbidden repository zone".to_string(),
+        }));
+    }
+    if catalog
+        .generated_zones
+        .iter()
+        .any(|zone| path_matches(path, &zone.path))
+    {
+        return Ok(Some(SkippedEdit {
+            finding_fingerprint: edit.finding_fingerprint.clone(),
+            path: path.to_string(),
+            reason: "path is in a generated zone".to_string(),
+        }));
+    }
+    let risk = packet_risk(packet);
+    let eligibility = packet_eligibility(packet);
+    if !risk.is_allowed_by(max_risk) {
+        return Ok(Some(SkippedEdit {
+            finding_fingerprint: edit.finding_fingerprint.clone(),
+            path: path.to_string(),
+            reason: "packet risk exceeds the configured max risk".to_string(),
+        }));
+    }
+    if !matches!(
+        eligibility,
+        RepairEligibility::AutoSafe | RepairEligibility::AgentAssisted
+    ) || packet.human_review_required
+    {
+        return Ok(Some(SkippedEdit {
+            finding_fingerprint: edit.finding_fingerprint.clone(),
+            path: path.to_string(),
+            reason: "packet is not eligible for real apply".to_string(),
+        }));
+    }
+    Ok(None)
+}

+fn apply_append_text(repo: &Path, edit: &PlannedEdit, path: &str) -> Result<EditOutcome> {
+    let append_text = match edit.append_text.as_deref() {
+        Some(text) if !text.is_empty() => text,
+        _ => {
+            return Ok(EditOutcome::Skipped(SkippedEdit {
+                finding_fingerprint: edit.finding_fingerprint.clone(),
+                path: path.to_string(),
+                reason: "append-text strategy requires non-empty append_text".to_string(),
+            }))
+        }
+    };
+    let file_path = repo.join(path);
+    let mut content = fs::read_to_string(&file_path)
+        .with_context(|| format!("read append-text target {}", file_path.display()))?;
+    let before_sha256 = sha256_text(&content);
+    content.push_str(append_text);
+    let after_sha256 = sha256_text(&content);
+    if before_sha256 == after_sha256 {
+        return Ok(EditOutcome::Skipped(SkippedEdit {
+            finding_fingerprint: edit.finding_fingerprint.clone(),
+            path: path.to_string(),
+            reason: "append-text produced no content change".to_string(),
+        }));
+    }
+    write_text(&file_path, &content)?;
+    Ok(EditOutcome::Applied(AppliedEdit {
+        finding_fingerprint: edit.finding_fingerprint.clone(),
+        path: path.to_string(),
+        apply_strategy: edit.apply_strategy.clone(),
+        before_sha256,
+        after_sha256,
+        status: "applied".to_string(),
+    }))
+}
+
+fn apply_replace_exact(repo: &Path, edit: &PlannedEdit, path: &str) -> Result<EditOutcome> {
+    let match_text = match edit.match_text.as_deref() {
+        Some(text) if !text.is_empty() => text,
+        _ => {
+            return Ok(EditOutcome::Skipped(SkippedEdit {
+                finding_fingerprint: edit.finding_fingerprint.clone(),
+                path: path.to_string(),
+                reason: "replace-exact strategy requires non-empty match_text".to_string(),
+            }))
+        }
+    };
+    let replacement_text = match edit.replacement_text.as_deref() {
+        Some(text) => text,
+        None => {
+            return Ok(EditOutcome::Skipped(SkippedEdit {
+                finding_fingerprint: edit.finding_fingerprint.clone(),
+                path: path.to_string(),
+                reason: "replace-exact strategy requires replacement_text".to_string(),
+            }))
+        }
+    };
+    if match_text == replacement_text {
+        return Ok(EditOutcome::Skipped(SkippedEdit {
+            finding_fingerprint: edit.finding_fingerprint.clone(),
+            path: path.to_string(),
+            reason: "replacement_text is identical to match_text".to_string(),
+        }));
+    }
+    let file_path = repo.join(path);
+    let content = fs::read_to_string(&file_path)
+        .with_context(|| format!("read replace-exact target {}", file_path.display()))?;
+    let before_sha256 = sha256_text(&content);
+    let occurrences = content.matches(match_text).count();
+    if occurrences != 1 {
+        return Ok(EditOutcome::Skipped(SkippedEdit {
+            finding_fingerprint: edit.finding_fingerprint.clone(),
+            path: path.to_string(),
+            reason: format!("replace-exact expected exactly one match, found {occurrences}"),
+        }));
+    }
+    let updated = content.replacen(match_text, replacement_text, 1);
+    let after_sha256 = sha256_text(&updated);
+    if before_sha256 == after_sha256 {
+        return Ok(EditOutcome::Skipped(SkippedEdit {
+            finding_fingerprint: edit.finding_fingerprint.clone(),
+            path: path.to_string(),
+            reason: "replace-exact produced no content change".to_string(),
+        }));
+    }
+    write_text(&file_path, &updated)?;
+    Ok(EditOutcome::Applied(AppliedEdit {
+        finding_fingerprint: edit.finding_fingerprint.clone(),
+        path: path.to_string(),
+        apply_strategy: edit.apply_strategy.clone(),
+        before_sha256,
+        after_sha256,
+        status: "applied".to_string(),
+    }))
+}

+fn apply_create_file(repo: &Path, edit: &PlannedEdit, path: &str) -> Result<EditOutcome> {
+    let create_text = match edit.create_text.as_deref() {
+        Some(text) => text,
+        None => {
+            return Ok(EditOutcome::Skipped(SkippedEdit {
+                finding_fingerprint: edit.finding_fingerprint.clone(),
+                path: path.to_string(),
+                reason: "create-file strategy requires create_text".to_string(),
+            }))
+        }
+    };
+    let file_path = repo.join(path);
+    if file_path.exists() {
+        return Ok(EditOutcome::Skipped(SkippedEdit {
+            finding_fingerprint: edit.finding_fingerprint.clone(),
+            path: path.to_string(),
+            reason: "target file already exists for create-file".to_string(),
+        }));
+    }
+    if let Some(parent) = file_path.parent() {
+        fs::create_dir_all(parent)?;
+    }
+    write_text(&file_path, create_text)?;
+    Ok(EditOutcome::Applied(AppliedEdit {
+        finding_fingerprint: edit.finding_fingerprint.clone(),
+        path: path.to_string(),
+        apply_strategy: edit.apply_strategy.clone(),
+        before_sha256: "missing".to_string(),
+        after_sha256: sha256_text(create_text),
+        status: "applied".to_string(),
+    }))
+}

+fn normalize_edit_path(path: &str) -> Result<String> {
+    if path.trim().is_empty() {
+        bail!("edit path must not be empty");
+    }
+    let candidate = Path::new(path);
+    if candidate.is_absolute() {
+        bail!("absolute edit paths are not allowed: `{path}`");
+    }
+    let mut parts = Vec::new();
+    for component in candidate.components() {
+        match component {
+            Component::Normal(part) => parts.push(part.to_string_lossy().to_string()),
+            Component::CurDir => {}
+            Component::ParentDir | Component::RootDir | Component::Prefix(_) => {
+                bail!("path traversal is not allowed: `{path}`");
+            }
+        }
+    }
+    let normalized = parts.join("/");
+    if normalized.is_empty() {
+        bail!("edit path must not be empty");
+    }
+    Ok(normalized)
+}

+fn path_allowed(path: &str, allowed_paths: &[String]) -> bool {
+    allowed_paths.iter().any(|allowed| path_matches(path, allowed))
+}
+
+fn path_forbidden(path: &str, forbidden_paths: &[String]) -> bool {
+    forbidden_paths
+        .iter()
+        .any(|forbidden| path_matches(path, forbidden))
+}

+fn is_repo_forbidden_path(path: &str) -> bool {
+    matches_prefix(path, ".git/")
+        || matches_prefix(path, "reference/")
+        || matches_prefix(path, "paper/")
+        || matches_prefix(path, "target/")
+        || path == ".git"
+        || path == "reference"
+        || path == "paper"
+        || path == "target"
+}
+
+fn path_matches(path: &str, prefix: &str) -> bool {
+    let prefix = prefix.trim_end_matches('/');
+    path == prefix || path.starts_with(&format!("{prefix}/"))
+}
+
+fn matches_prefix(path: &str, prefix: &str) -> bool {
+    let prefix = prefix.trim_end_matches('/');
+    path == prefix || path.starts_with(&format!("{prefix}/"))
+}

+fn write_text(path: &Path, text: &str) -> Result<()> {
+    if let Some(parent) = path.parent() {
+        fs::create_dir_all(parent)?;
+    }
+    fs::write(path, text)?;
+    Ok(())
+}
+
+fn sha256_text(text: &str) -> String {
+    let digest = Sha256::digest(text.as_bytes());
+    format!("sha256:{:x}", digest)
+}
+
+fn relative_repo_path(repo: &Path, path: &Path) -> String {
+    path.strip_prefix(repo)
+        .ok()
+        .unwrap_or(path)
+        .to_string_lossy()
+        .replace('\\', "/")
+}

+fn require_env_gate(name: &str, reason: &str) -> Result<()> {
+    let enabled = std::env::var(name)
+        .map(|value| value == "1" || value.eq_ignore_ascii_case("true"))
+        .unwrap_or(false);
+    if !enabled {
+        bail!("{reason}; set `{name}=1` to opt in");
+    }
+    Ok(())
+}

+fn ensure_git_worktree_clean(repo: &Path) -> Result<()> {
+    let status = git_output(repo, &["status", "--porcelain"])?;
+    if !status.trim().is_empty() {
+        bail!(
+            "refusing git mutation with a dirty worktree; commit, stash, or remove existing changes first:\n{}",
+            status.trim()
+        );
+    }
+    Ok(())
+}

+fn current_branch(repo: &Path) -> Result<Option<String>> {
+    let output = Command::new("git")
+        .arg("-C")
+        .arg(repo)
+        .arg("symbolic-ref")
+        .arg("--short")
+        .arg("HEAD")
+        .output()
+        .context("read current git branch")?;
+    if !output.status.success() {
+        return Ok(None);
+    }
+    let branch = String::from_utf8_lossy(&output.stdout).trim().to_string();
+    if branch.is_empty() {
+        Ok(None)
+    } else {
+        Ok(Some(branch))
+    }

+fn create_github_pr(repo: &Path, base: &str, title: &str, body: &str) -> Result<String> {
+    let output = Command::new("gh")
+        .arg("pr")
+        .arg("create")
+        .arg("--draft")
+        .arg("--base")
+        .arg(base)
+        .arg("--title")
+        .arg(title)
+        .arg("--body")
+        .arg(body)
+        .current_dir(repo)
+        .output()
+        .context("run gh pr create")?;
+    if !output.status.success() {
+        bail!(
+            "gh pr create failed with status {}\nstdout:\n{}\nstderr:\n{}",
+            output.status,
+            String::from_utf8_lossy(&output.stdout),
+            String::from_utf8_lossy(&output.stderr)
+        );
+    }
+    let stdout = String::from_utf8_lossy(&output.stdout);
+    let url = stdout
+        .lines()
+        .rev()
+        .find(|line| !line.trim().is_empty())
+        .map(str::trim)
+        .unwrap_or("gh pr create completed")
+        .to_string();
+    Ok(url)

+fn deterministic_branch_name(plan: &RepairPlan) -> String {
+    let mut seed = String::new();
+    seed.push_str(&plan.source_report);
+    seed.push('|');
+    for edit in &plan.planned_edits {
+        seed.push_str(&edit.finding_fingerprint);
+        seed.push('|');
+        seed.push_str(&edit.path);
+        seed.push('|');
+        seed.push_str(&edit.apply_strategy);
+        seed.push('|');
+    }
+    let digest = Sha256::digest(seed.as_bytes());
+    let hex = format!("{:x}", digest);
+    format!("jankurai/repair/{}", &hex[..12])
+}
