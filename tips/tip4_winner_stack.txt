Default winner stack:
Rust core + TypeScript/React/Vite product surface + PostgreSQL truth + generated contracts + bounded Python AI/data service.

Use it when correctness, security, performance, and auditability matter more than raw language familiarity.

Boundary rule:
- TypeScript owns UI, forms, and local validation
- Rust owns core logic, authz, idempotency, and orchestration
- PostgreSQL owns durable truth and constraints
- Python owns models, embeddings, and evals only

If a layer starts owning another layer's truth, the stack has already drifted.
